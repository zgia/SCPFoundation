物理部门威胁实体数据库摘要
设定中心 » 全球超自然联盟档案 » 物理部门威胁实体数据库摘要
  • 介绍
  • 常用代称
  • KTE: 已知威胁实体
  • UTE: 未知威胁实体
  • PTE: 潜在威胁个体
  • LTE:清除威胁个体
  • 其他

介绍

基于对该实体的了解程度，全球超自然联盟将超自然威胁实体分为五大类。威胁实体使用如下格式命名：

  • 一串三字符代号表明威胁类别。
  • 一个独一无二的文档代号。
  • 一或多个描述性代称。
  • 对该威胁实体的官方描述。

例子: KTE-7912-Spiral Blackchild: “康沃尔可变维度传送门。”
前面的三字符代号表明这是一个已知为威胁实体 (KTE), 文档代号 7912, 这是一个局部时空撕裂 (代称 Spiral), 还是一个半神级威胁创造的。(代称 Black [半神级], 修饰后缀 -child [被创造的超自然实体])。

本列表只是用来编级威胁的数百个代号的简单删节版。完整的代称列表，请查阅你自己的物理部门手册。

人形

人形威胁实体普遍采用颜色分类系统。
例子包括:

  • Type Black: 半神
  • Type Blue: 奇术师
  • Type Green: 现实扭曲者
  • Type Grey: 死后复苏者
  • Type Red: 再生者
  • Type Silver: 传染危机
  • Type White: 非典型天使
  • Type Yellow: 变形者，通灵者以及其他异常增强的人形实体
  • Type Beige: 被分割的身体部位

非人形

以下代称用于非人形活体。

  • Blackwood: 前人类
  • Corona: 来自地外的生命形式
  • Cetus: 大型水生生命形式.
  • Denver: 临时转换生命形式
  • Fiji: 神秘生物 (为了第二任务因素被压制有关其存在信息的一种本土生命形式)
  • Goodrick:奇术生命形式
  • Hogweed: 植物类
  • Moro: 人造生命形式
  • NACL: 非本土生命形式。本代号只用于那些确定不来自于本时空，但不能判定来源的生命形式
  • Plague: 传染性微生物
  • Ragweed: 来自一个平行宇宙的生命形式
  • Schopenhauer: 地灵（genius loci）。本代号指某个具现化了的或有生命的地理特征

物品

与人造威胁个体项目有关的。

  • Alexandria: 书籍类
  • Blaecca: 计算机类器械
  • Caliburn: 武器类项目
  • Flint: 货币类项目
  • Kapala: 仪式或宗教类项目
  • Knickknack: 可穿戴项目
  • Velveteen: 可动项目

修饰后缀

  • -buster: 试图摧毁其他威胁实体的威胁
  • -child: 由另一个有特定代号的威胁个体所创造
  • -morph: 威胁个体是在对另一个威胁个体的清除程序中被创造的
  • -father: 与一个组织代号一同使用；上述组织的奠基者或领导者。
  • -dianetic: 仅与一个特定威胁实体直接相关联的组织。

已知威胁个体是那些已经被完全确认和分配了应对等级的个体。

Access KTE-0706-Black
Hide KTE-0706-Black

威胁 ID: KTE-0706-Black "亚伯项目"

授权应对等级: 4 (严重威胁)

描述: 组件Able是个类人威胁个体，年龄不超过20岁，约有2米高，80公斤重。项目平时会佩戴魔怪意象的饰品，并偏好黑色皮革衣物。并另外拥有标准的全套超人能力（1级或更高的力量，速度，和再生能力）。项目展示出能从未知来源变化出刀剑类武器的能力。项目在近距离战斗中拥有很高的技术，并使用当前没有记录的未知战斗风格。

心理分析指出项目的主要动机来自于无聊：项目似乎只能通过杀戮来得到满足，特别是在对抗他认为“值得一战的家伙”时。项目经常与相当危险的威胁首先交战，并绕开那些威胁较小的：作业员应把这个因素列入行动计划中。

组件Baker是一块有雕刻的石制立方体，边长3.66米，用21把锁（最大的一把锁被其他20把较小的类似的锁所环绕）锁住。组件Baker-2是一块2.13米高的石棺，被收容在Baker中，并用未知材质的锁链固定。Baker-2有能力通过任何手段将组件Able进行再生，即使是在组件Able被彻底摧毁的情况下也是如此。在发生如上所述情况时，组件Baker将被认为是主要威胁并应尽快确保。

交战规则: KTE-0706 Black（称之为项目）由2个部分组成。与组件Able的交战应十分小心：该个体是一名十分致命的战士并至今已导致超过███名作业员死亡。绝不要与组件Able在近距离交战，应尽可能保持距离。由狙击手造成的严重头部创伤将作为消灭计划被执行。

在消灭后，组件Albe将在组件Baker内开始再生，并在数个星期后就能以完整的威胁效力出现。因此，为了清除项目必须寻找并摧毁组件Baker。任何发现能追查到组件Baker的信息的作业员应马上将该信息告知GOC最高指挥部。

历史: <由GOC最高指挥部下令编级为Q>

附录 001: 外勤作业员指出KTE-0706-Black当前在基金会手中。作业员在此被命令将工作集中于寻找该组织并定位和回收组件Baker。

附录 002: 根据来自作业员Crane的外勤报告，联盟作业员在此被告之基金会开始将组件Albe投入战斗中，并由一支6-12人的小队负责支援。

附录 003: 由于在最近三次与威胁部队Albe（Threat Force Able）的交战都导致了不成功的伤亡，所有外勤作业员将在除了自卫之外的任何情况下中止与组件Able的交战并撤离。作业员应重新将工作集中于寻找并摧毁组件Baker。

Access KTE-0937-Velveteen
Hide KTE-0937-Velveteen

威胁ID: KTE-0937-Velveteen "第六把椅子"

授权应对等级: 3 (中等威胁)

描述: 主要由██.█公斤的橡木碎片和衬垫物构成。之前是一把用橡木和皮革制作的椅子，形状像一个向后倾身的女性。Type III 空间存在个体。

交战个体: 只能由潜伏很深的作业员进行交战。项目对于联盟及其特工有很高的敌意。项目应在有机会时通过焚烧进行摧毁。

历史: 最早是一套六个类似物品，由KTE-7201-Pygmalion ("木匠") 打造并作为礼物送给[身份删除]。在████被编级为一个第二任务威胁。创造者，购买者，和木匠都已经被一个GOC外勤行动单元所清理。所有其他同类物品都被成功焚毁：KTE-0937-Velveteen由于焚化炉故障而没有被焚毁，转而进行了浸泡处理。该项目随后通过瞬间移动逃离，并在随后导致了多名特工的死亡。据信当前被基金会所持有。

附录:

KTE-0937-Velveteen案例是一个应遵守适当操作程序的严重教训。由于现场的特工缺乏足够的警惕，项目的威胁等级逐步上升，项目本身没有被成功处理，并在那之后落入了一个敌对机构手中。一个作业员犯下的一个错误导致了6人的死亡。在你下次打算敷衍了事时想想这个。

— 主任助理"Kipling"

Access KTE-1181-Plague Grey
Hide KTE-1181-Plague Grey

威胁ID: KTE-1181-Plague-Grey "感染性黏菌"

授权应对等级: 3 (中等威胁)

描述: 一种黄-绿 肌细胞（myocetezoan）原生生物。生命周期开始于一个单细胞变形虫，并普通的捕食细菌和其他原生生物。一旦遭遇其他同类物种，生物体将配对并变成一个原质团，并在之后延伸，吃掉所有遭遇到的食物物质。当所有可吃的食物物质都被吞噬后，生物体将团起来，变成一个孢子囊（子实体），并释放孢子，孢子将孵化成单细胞，重新开始生命周期。

该生物体被视为一个准威胁，这是因为其在遭遇到人类脑部组织后的行为。若变形虫穿过血脑屏障进入脑部，它将开始吞噬人脑胶质细胞并将人脑细胞替换成原质团化合物。第二原质团将跟随血管移动到皮肤毛孔，并在那之后将变成类似孢子囊的结构并开始产出一种气味类物质。

寄主的身体随后将开始强制移动到公共空间，其释放的气味将吸引其他人类到寄主当前的位置。一旦在特定的距离上侦测到足够的人类寄主，生物体开始以指数级的速度繁殖，在寄主的头骨内产生巨大的压力，直到其头颅爆炸，散布出的孢子和原质团将感染周围所有的人类寄主，并重复生命周期。

交战规则: 感染寄主应进行射杀处决，其尸体将用焚毁处理。避免爆头，这会导致寄主的头颅过早爆炸并导致感染散布。最完美的处决方法是由神枪手在远距离上对寄主的心脏进行精确射击：若需要进行近距离战斗，所有作业员应穿戴MOPP装具并在行动后进行不少于72小时的隔离。

若发生感染范围逐渐上升并散布出检疫区，授权上升到应对等级4。此时，整个隔离区应被封闭并进行焚毁消毒。所有试图离开隔离区的人员应用射杀处决。
