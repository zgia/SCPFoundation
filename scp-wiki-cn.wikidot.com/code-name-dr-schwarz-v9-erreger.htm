SCP-001-DE

项目编号： SCP-001-DE

项目等级： Keter

特殊收容措施： SCP-001-DE位于南极洲新斯瓦比亚南纬1██°█'██"、西经███°█'██"处，该地点同时也建有211号基地。自1946年██月██日起，SCP-001-DE便无法被完全收容且处于不断的扩散中。211号基地区位于一片半径300千米的封锁区中，对外宣称为在珍惜鸟类筑巢地建造的自然保护区。鉴于SCP-001-DE不会停止扩散且每年扩散1.45千米，封锁区也必须每年扩大以与之相称。考虑到SCP-001-DE除了在表面扩散外，还会向冰层深处传播，必须每两（2）周进行一次深钻，以确定当前情况。

于1990年██月██日在本区域最外层处建立移动研究设施研究设施Alpha-001-██，其中生活及工作有基金会研究者以及多支MTF单位。一旦发生D级人员暴动即允许MTF单位投入致命武力。基地中同样布设有防御系统的监控以及操作系统以监视整个地区。封锁区依靠运动检测器以及环程飞行来保卫。运动检测器需每两（2）周检测一次效率。

研究设施Alpha-001-██

211号基地工作及生活有多名D级人员。该设施因而得以在更新换代的同时将病原体侵害降到最低。设施内部设有一个组织完备的实验室，通称生命部门-C12，其中由D级人员为基金会科学家执行准备工作。当该设施因样品或试验被废弃时，必须按规定穿着防护服。这些防护服可提供对病原体的短时间隔断。供役于211号基地的D级人员必须经过生物相关基础训练（参见经训练的D级研究人员）。受感染人员需在视频监视以及自动射击装置下满足其所有要求。若不服从则对D级人员实施处决，而对参与其中的D级研究人员实施警告。对死亡D级人员的移除仅允许由未经教育的D级人员在八天的周期内完成。

所有关于211号基地的数据与报告均需清除或涂黑。该事需由基金会雇员在最高级别保密的情况下执行。为了保护Base 211以及研究设施Alpha-001-██，必须保持所有阴谋论不失控。

描述： SCP-001-DE是一种耐多种药物的外星病原体，于1938年██月██日随一次小行星撞击抵达地球（见事故001-█）。该超级细菌与多药耐药性大肠杆菌2有类似之处。鉴于该病原体无需培养基即可培育的独特性质，目前仍未完全知晓其如何增殖。该病原体可通过直接接触或体液接触发生传染。因而可确定，体液传播最为有效。当一名人员被感染后，其将在两天后具高度感染性。此时皮肤直接接触造成的体液接触即具有传染风险。目前SCP-001-DE不受所有已知治疗方法影响。同时在经过数十年的扩散后，菌株已不再有可能被乙醇、氯气或臭氧化学扑灭，除武器装备与实验室材料（若微量接触SCP-001-DE，可利用杀菌剂对其消毒）以外。目前SCP-001-DE占有直径101.5千米，至少深1.2千米的区域，该区域被称作污染区。由于该进程似乎是随机发展的，因而无法确定其到多深之下仍在运转。SCP-001-DE能够抵御极地地区0°C至-35°C的严寒，而一般而言此温度下细菌即会进入休眠状态。必须不惜一切代价阻止SCP-001-DE接触流水。SCP-001-DE与水进行接触不久后，该病原体即会无法阻挡。其可在流水中飞快增殖。一旦发生此类事故，整个海洋即会在数天内受到污染。目前还无法研制出能够成功收容在水中的SCP-001-DE的方法。据计算，当其进入海水██年后，即可灭绝地球上所有的生物。

SCP-001-DE可导致全身细胞的不可控分裂。在感染后最初3天，身体会开始出现大型肿瘤与肿块，此时体细胞会毫无规律地变异，且变异速度会随时间加快。在感染后第6天时，开始出现异常效应，感染对象的头部及躯干会强烈变形。此阶段受感染者会感觉到强烈的疼痛，但仍然拥有语言及活动能力。感染后第8天时，感染对象会生长出一些异常身体部件，例如畸形肢体，这些部件大多数位于头部。当出现这一变化之后，感染对象便仅能再存活两天。其尸体中的病原体在宿主死亡后仍具有高度传染性。除非实验目的需要，否则受感染的尸体均需实施焚化。在某些感染对象身上，感染进程会出现意外现象，即提前死亡。

附录： 即使是穿着防护服，也仅能在一定时间内提供对SCP-001-DE的防护。当与病原体接触时间过长时，防护服便会成功被病原体穿透。目前仍在研究SCP-001-DE是如何成功穿透防护服的。参见现场实验001-01-09[拒绝访问]。

以下事件记录包含1938年███月28日由研究设施Alpha-001-██及生命部门-C12经训练的D级研究人员回收的坠落小行星相关信息。请提交您的权限证明。

> 请提交您的权限证明以查看文件：“事故记录001-██”

生物部门-C12简介： 生物部门-C12为第三帝国当时在南极洲建立的首个研究机构，位于211号基地中。其在1939至1946年间由第三帝国建造并使用。1946年至1989年由基金会总部管理并用以存放异常物品。1989年该设施被移交给基金会德国分部，从此生物部门-C12便仅限经训练的D级研究人员进入。

事故记录001-██：

SCP-001-DE的来源

1938年██月██日，一枚R级小行星坠落在南极洲新斯瓦比亚地区。该枚小行星的颜色由黄棕色渐变至黑色，生命部门-C12的德国研究人员在对其进行首次分析时，发现其很大一部分都附着有未知的传染性病原体。该枚小行星着陆于位于211号基地域内南纬██°█'██"、西经███°█'██"处。在外星病原体落地后，由于使用D级人员在SCP-001-DE项目中工作可将危险程度降至很低，211号基地于1990年██月██日更新装备。1947年██月██日，该枚小行星被打上新C类危险物质标记3后被转移至生命部门-C12以供分析。1989年，211号基地被重新移交给基金会德国分部。

经训练的D级研究人员


本类D级人员必须接受过某一类生物职业领域的基础教育，以供被选来研究SCP-001-DE及其小行星。这些生物职业领域包括以下几类受教育者：

  • 生物学家
  • 生态学家
  • 保育生物学家
  • 农业生物学家
  • 生物信息学家
  • 生物医学家
  • 生物工艺学家
  • 化学家
  • 微生物学家
  • 海洋生物学家

研究设施Alpha-001-██

为了研究SCP-001-DE在水中的反应以及开发出可阻止该病原体于水中快速扩散的方式，本研究设施主区由海洋生物学部、生物工艺学部以及微生物学部组成。需不计一切代价持续该研究，然而目前该研究尚未得出实质性成果。唯一一种可暂时抑制SCP-001-DE的方法为尝试尽可能将该种病原体留存在极地地区。由于本研究设施流动，因而SCP-001-DE一年扩散1.45千米并未带来问题，因为感染区无法接近本研究设施。

注意：一旦发生D级研究人员损失，即说明研究设施需更换地址。

以下为1938年至1946年“跳高行动”事件及新斯瓦比亚的真实报告。所有历史记载均保持对SCP-001-DE存在的掩饰。仅限4级安保权限人员检阅。

1938年极地考察


事故前言： 1938年██月██日，第三帝国对南极冰盖展开考察期间，在一枚小行星于考察队附近着陆后，考察队发现SCP-001-DE，并立即认识到其异常性质。第三帝国将SCP-001-DE编号为V9（Vergeltungswaffe 9，9号报复武器）项目。当时第三帝国的迅速反应成功控制了SCP-001-DE。有人认识到其对大规模杀伤性武器的建造的巨大潜力，因而第三帝国建立了211号基地。由于对SCP-001-DE的开发与研究十分危险，直到1946年██月██日都未获得成果。


关于“跳高行动”事件的文件

日期：1946年██月██日

一级突击队大队长██████ █████████ 谨缄

自：新斯瓦比亚，211号极地基地

至：██████████ █████

████ ██████ 谨启

报告

那些间谍报告完全是错的！报告说敌人最早也得在██月██日才会进入我们的水域。


1946年██月██日██:██： 138号侦察机报告视见距我方岸边78千米的一支敌对联合舰队。

1946年██月██日██:██： 138号侦察机报告舰队规模：

  • 航空母舰一艘，水上飞机两架，驱逐舰两艘，油轮两艘，破冰船两艘，补给船两艘，通信船两艘以及潜艇一艘

1946年██月██日██:██： ██████-II飞行联队进入紧急状态。

1946年██月██日██:██： 观察到两架敌对侦察机飞向211号基地。下达击毁指令。两架██████-II开始截击敌对侦察机。

1946年██月██日██:██： 敌对侦察机被击毁。己方无损失。

1946年██月██日██:██： 再度视见侦察机队一支，由侦察机两艘与战斗机六艘组成。派出两架██████-II对其进行截击。

1946年██月██日██:██： 敌对机队被完全击毁。己方无损失。截获一条无线电讯。翻译如下：

  • 去他娘的，那是什么鬼东西？我们被不明飞行物攻击！他们一个一个地被击落了！它们太快了，打不过。等等……那是啥？其中一个东西上面有个钩子……

1946年██月██日██:██： 由此出现我们伪装失败的猜想。下达反攻指令。派出四架██████-II。

1946年██月██日██:██： 在我方██████-II对敌方舰队造成严重伤害时，一支未经觉察的战斗机飞行联队靠近211号基地东北方。该地区此时仅受到基本防护。仍未知敌方如何得知薄弱之处。

1946年██月██日██:██： 我方无法足够快地反应。由于该大楼仅粗浅考虑了对V9项目的封存，其受到了敌机的轰炸。随后击退并击毁敌机时，整座大楼已完全被炸毁。

1946年██月██日██:██： 敌方舰队后撤。可断定敌方损失惨重。己方无损失。

1946年██月██日██:██： 对V9项目大楼的调查显示小行星仍完好。然而大楼被炸毁导致V9项目直接扩散。无法将其遏制。

1946年██月██日██:██： 兹结束本事故报告。此致敬礼，一级突击队大队长██████ █████████。

报告结束

附录： 本事故导致德国人无法再保证对其进行遏制。由于数十年的扩散，后来基金会也无法对SCP-001-DE进行成功收容。
