SCP-1513-JP

Site-81KI-A的设备之一。该Site表面上被视作[已编辑]变电站处理。

项目编号: SCP-1513-JP

威胁等级: 橙 ●

项目等级: Keter

特殊收容措施: 由于事件-1513-JP仅在以ISTL1为准的东部(日本东部)发生，因此基金会将在此地区积极进行活动。

沿ISTL设置的数个变电站在确认项目后由基金会进行管理。为便于管理，各变电站视为Site-81KI群处理，这些Site的设备仍被伪装为“变电站”。

机动特遣队つ-5("电灯处理者")负责确认事件-1513-JP的发生，确认事件后立即配备机动特遣队つ-5，处理SCP-1513-JP-1并回收SCP-1513-JP-2。SCP-1513-JP-2将在记忆处理后被释放。

SCP-1513-JP之一。

描述: SCP-1513-JP是日本国内东部电线线路的总称。在使用SCP-1513-JP的供电电流进行照明时，被指定为事件-1513-JP的现象不定时地发生。

目前已断定事件-1513-JP仅在日本东部发生的原因为该地区的商用电源频率为50Hz。商用电源频率为60Hz的日本西部没有确认到事件-1513-JP的发生，由于上述设施进行了频率变换，因此推测发生地区只限于日本东部。此特性现被利用于收容措施，基金会正在计划将日本的国家频率统一为60Hz。

虽然某些例外情况存在，但大多数照明设备都安装在民宅内部2。发生事件-1513-JP的照明设备被指定为SCP-1513-JP-1，目击SCP-1513-JP-1的人被指定为SCP-1513-JP-2。

SCP-1513-JP-1具有影响SCP-1513-JP-2精神的异常性质。SCP-1513-JP-2在目击之后，会开始对德意志民主共和国 抱有爱国情绪，并同时抱有反日情绪。虽然SCP-1513-JP-2在日常生活中很少表现出此种思想，但在保持自己的匿名性的情况下，或者周围只存在与自己具有相同思想的人的情况下，SCP-1513-JP-2会显示出此种思想。SCP-1513-JP-2通过目击得到的思想可根据需要使用记忆处理药剂进行消除。SCP-1513-JP-1的异常性在无电力供给时消失。

2003/██/██，日本匿名论坛中亲德思想和反日思想的投稿急剧增加，与SCP-1513-JP相关的异常引起了基金会的兴趣。之后，由于匿名论坛上的投稿者大都住在日本东部，因此怀疑其与商用电源频率有所关联，确认到了投稿者终端附近的照明设施具有精神影响，加上后来进行的试验，明确了SCP-1513-JP的特性。

目前SCP-1513-JP被分类为Keter级，因为SCP-1513-JP相关对象难以收容，当前尚未确认这些对象对平民人员造成严重损害。

补充1:日本西部和日本东部的商用电源频率分别为60Hz和50Hz的原因是，明治时代在关东进行交流输电转换时，从德国引进了50Hz规格的AEG发电机。日本引进的这个发电机可能有某种异常性，基金会正在搜索该发电机的下落。 请参阅补充2。

补充2: 从引进AEG发电机到现在，对包括小规模在内的亲德、反日团体是否存在进行了调查，但这些团体大多存在于1959年到1989年之间，因此判明异常原因不是引入的AEG发电机。对发电机的搜索已中断，已将人员重新部署到其他部门。

在本次调查中，发现这些团体的总数自1961年开始逐渐减少。但是，由于团体成员的身份难以确定，以及调查结果获利甚少，所以没有实施调查，减少的原因仍未确定。

补充3: 2018/05/19，在日本山梨县富士河口湖町鸣泽村的青木原树海，进行巡逻的地方政府职员发现了相当于大约3名成人量的人骨。之后警察调查时，在发现地点的地下发现了相当于约1██人量的人骨，此事件被得到消息的基金会所隐藏，遗体全部被移送到附近的基金会设施中。

根据之后的调查结果推测这些人骨的身份大部分是在1961年~1989年间失踪的人物。死因未定，基金会推测这些人物属于上述亲德·反日团体。基金会现在计划积极进行调查。

补充4:在基金会德国支部实施的德意志民主共和国国家安全部25号部门3相关文件的研究过程中，发现了以下文件。本文件与SCP-1513-JP相关，根据文件内容判明与25号部门的异常制作有所关联。

文件 01#: 补充1513-JP-1-01

柏林、1957年10月07日

国家安全部

25号部门

我有一个关于25号部门目的的建议。

我们25号部门的目的是，使用异常进行谍报行为和怠工以及从其他国家获利的工作。比起活人来做，如果使用异常的话更有效率，也更有效果，并且能将牺牲者控制在最小限度。

但是，即使使用了异常，也会出现为达成目的而造成的牺牲。遗憾的是，由于异常的暴走，甚至会出现本不该有的牺牲。这些牺牲会对组织造成重大损失，并可能导致士气下降。

只要使用我们所有的异常，就能在一定程度上隐藏事实。如果被利用的别国人的数量是“某种程度”的话，我这个尝试不会像25号部门的存在一样公开。

文件 01#: 补充1513-JP-1-02

柏林、1957年10月10日

国家安全部

25号部门

批准。此提案为「Project・Freimut」，提案者Hugbald Freimut为项目领导人。

但项目要稳妥进行。同时，使用的其他国民的人数要在能隐蔽事实的范围内。

署名：Constantine Murder军官

文件 01#: 补充1513-JP-1-03

柏林、1959年03月02日

国家安全部

25号部门

好久不见，长官。报告Project・Freimut的进展。

前几天，我成功地制作了异常。这是使用由50Hz的输电线输电的电灯――使用电灯的理由是，只要是发达国家的市中心·近郊，无论在哪个民家都有电灯――所以才将它改成异常。目击了电灯的人的异常变化是成为对东德有很强爱国心的人，与此同时，也失去了对那个人本国的诚意。

然后，我选择了日本作为Project・Freimut的实施国。根据从西德得到的情报，由于日本引进了AEG发电机，从大深沟的西面开始，东部和西部的频率不同，被引进的东侧与东德相同，为50Hz。标准频率采用50Hz的国家，还有中华人民共和国等国，不过，国家面积狭窄，人口众多，而且输电网很充实，在这一点上，我们选择了日本，而不是中华人民共和国等国。

日后，将在日本东部实施Project・Freimut。如有报告我会随后发送信息。

文件 01#: 补充1513-JP-1-04

柏林、1959年03月03日

国家安全部

25号部门

明白了。

署名：Constantine Murder军官

文件 01#: 补充1513-JP-1-05

东京、1959年05月07日

国家安全部

25号部门

好久不见，长官。报告Project・Freimut的进展。

在日本东部的25号部门特工的增产，目前保持成功。虽然绝不会表露出来，但他们内藏着对东德的深厚诚意。而我已与他们结成的亲德、反日团体成功接触了――只要我们有指示，他们就会随时行动。

有几个人已经出海到东德了。对于我来说，几周后那些人员和25号部门职员会合。另外，其他人员的出国目前也在计划之中，并尝试着继承基础战术学。

我们随时准备完成任务。如有未完成的任务，请务必通知我。

文件 01#: 补充1513-JP-1-064

柏林、1961年05月16日

国家安全部

25号部门

你好，长官。百忙之中给您添麻烦了，我将就Project・Freimut提出一个建议。

目前，日本东部特工的增产进度本身未见异常。但是增产的人员，以及形成的团体的数量已经不能视而不见了，另外，Project・Freimut的领导人Hugbald Freimut的权力也与团体的增加成正比。

正如调查资料上所说，日本并不是不存在超常组织"――JOICL、东弊重工、甚至还有基金会和世界超自然联盟等高危险性组织。如果是他们的情报网的话，那么用不了多久，Project·Freimut的存在就会被人们所知吧。而这同时也意味着25号部门的存在。

我建议冻结Project・Freimut，并抹杀其领导者Hugbald Freimut。他现在不是我们的同志，而是阻挡在我们面前的障碍。不仅有泄露情报的可能性，如果以这个军事力量造反的话，受害也会非常大吧。

认为「战胜即美德」的人，在东德最早是不需要的。

文件 01#: 补充1513-JP-1-07

柏林、1961年05月17日

国家安全部

25号部门

批准。为抹杀Hugbald Freimut，配备专用人员。

署名：Constantine Murder军官

文件 01#: 补充1513-JP-1-08

柏林、1961年05月18日

国家安全部

25号部门

感谢长官的爽快承诺。配备的人员马上前往日本。

另外，我建议抹杀Freimut过去接触过的团体成员。虽然对东德的忠诚是必须的，但是盲信反而会起到反作用，而且从对Freimut的忠诚来看，可以预测Freimut遇害后的叛乱。

因此，我希望追加杀害团体成员的人员前往日本。关于杀害，为了避免上述团体的察觉，应该定期、长时间地进行吧。关于杀害手段和遗体遗弃的地点我再提建议。

文件 01#: 补充1513-JP-1-09

柏林、1961年05月20日

国家安全部

25号部门

批准。为抹杀团体成员，配备专用人员，作战要稳妥进行。

署名：Constantine Murder军官
