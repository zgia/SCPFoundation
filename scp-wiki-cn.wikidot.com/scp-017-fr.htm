SCP-017-FR

项目编号：SCP-017-FR

威胁等级：绿 ●

项目等级：Safe

特殊收容措施：目前，在多变的情况下，SCP-017-FR-A收容于Site-Aleph的安全储藏室内。 房间温度应保持在18°C，湿度则为50%。

一个墨水瓶将会放在一个单独的架子上的木箱中。书本的内容随时可供查阅。

描述：SCP-017-FR是一套由9本书组成的系列丛书（如果书本一并存放则编号为SCP-017-FR-A，如果各自独立放置则编号为SCP-017-FR-01至SCP-017-FR-09）。用一系列古代语言用红墨水书写。项目分别在处在不同的大陆的三个寺庙中被发现（见附录）。所有目击过项目的人都将被基金会带走进行讯问。

构成这些书页的纸张是由许多未知的材料组成的。 他们的页面上的墨水（编号为SCP-017-FR-B）被分析，并被发现是一种由智人、番红花幼苗、茜草、伊波加生物碱和圣佩德罗仙人掌混合物配制而成的墨水。该种墨水的复制品已用在对SCP-017-FR-A的实验上。

当任何人与书中书写所用的墨水直接接触并在书本上进行书写时，SCP-017-FR的异常性质将会展现。页面上的文字将在他的视野中不断浮现，并在视野内出现密度极大的红色色块。颜色将以极快的速度消失，随后根据文字内容显示相应情景。受影响者通常会宣称出现在他们拿着这本书的地点相同的位置，这个位置总是在他们的“原地”。尽管那些受到SCP-017-FR影响的人报告曾进入这些“空间”内部，但是外部的观察无法检测到任何尽管是细微至极的肌肉运动。离开“空间”的唯一可靠的方式是执行与以前相同的过程来写入这些内容。在离开空间后需要立刻进行情况简报并描述“空间”内部的内容。

一系列的实验显示，墨水中含有的许多神经性药物，当其和构成书本的未知物质混合并与人类皮肤同时接触时，即产生化学反应，药物将渗透至身体中，将虹膜和瞳孔染成红色。一旦该过程完成，这种物质将穿过玻璃体到达大脑并蔓延到额叶，顶叶和岛叶皮质。除了部分区域脑活动会增加（特别是阿尔法和伽玛脑电波），其他重要功能都将受到阻滞。

当任何人与SCP-017-FR-B进行直接接触但不目击书中的内容，那么药物就不会进入眼球，受影响者视野内也不会出现书中的内容，也不会影响到大脑。一旦史莱姆管清除了所有残留物，受影响者的视力将在几分钟内恢复正常。

当完成一次正确的SCP-017-FR-A使用流程时，SCP-017-FR-B会沿着输入路径逐渐去除身体内的所有油墨痕迹。然而，这个流程对大脑的伤害是不可逆转的。如果与SCP-017-FR的接触受到外部干预，受影响者将会陷入脑死亡的状态——由于其内部精神异常的存在，他的大脑机能也会受到严重损害。

附录：

书本编号 外观/ 回收地点 书中所写的环境的描述
SCP-017-FR-01 蓝色封面，共175页/ 亚马逊 周边环境是由沙岛群岛组成。没有可观察到的动物群，但植物群显然非常茂盛。外貌特征最为显著的植物是：树干螺旋上升的螺旋越来越宽（较大的样本估计尺寸为15米，直径为2米的螺旋） 在其顶部可以发现一个像螺旋和棕榈树一样宽的蛋，其上生长着类似蓝色木瓜的水果。食用这些果实将使食用者进入植物人状态。
SCP-017-FR-02 蓝色封面，共368页/冰岛 一个动植物丰富，并被认定有本地人部落存在的大型岛屿。其技术水平约相当于公元前2500年。由于该些部落的敌对行为，进行进一步的研究是困难的。与之进行沟通的尝试导致了受试者的植物人状态。随后数日的观察使用了一个有民族学历史资历的D级人员，并定期在现实世界与书中的“世界”来回奔波。这个岛上的社区有近二百人，其他部族似乎生活在邻近的岛屿上。对该“世界”的观察现在暂停，等待项目主管的进一步指示（见事件-017-FR-02）
SCP-017-FR-03 红色封面，共252页/ 冰岛 能看见一块被烧毁的土地，被灰色的土壤和灰烬覆盖。地平线上几个活火山。观察到稀疏植物群的存在，以小的红色浆果生长的小的棘手的灌木丛为主。同时也啮齿动物，以及食肉类、飞行类、爬行类动物。
SCP-017-FR-04 绿色封面，共285页/印度尼西亚 一个非常密集的古老森林。由于森林内部存在极大危险性，比如存在各类有意识的利用藤蔓捕捉猎物的藤蔓植物，探索非常短暂。有受试者在三次从该书出逃离的尝试中幸存了下来，但是在逃离该危急的情况之前，绝不能在原地停留两分钟以上。
SCP-017-FR-05 棕色封面，共183页/印度尼西亚 一个没有任何生物群落痕迹的环境。土壤上覆盖着直径约五十厘米的鳞片。地面摸起来温暖柔软，有时会波澜起伏。其中一块鳞片的拉扯导致了一系列的强烈地震，以及听到强大的威胁性的低吼。粘稠的绿色液体开始从鳞片的开口流出，液体具有强酸性。受试者受伤，并声称即使在他回到现实之后，疼痛依然存在，但无伤痕痕迹。当返回这个世界时，绿色的液体已经凝固并形成一个坚实的硬壳。
SCP-017-FR-06 棕色封面，共219页/亚马逊 丛林中有丰富的动植物。一个智能竞赛住在小屋里，形成一个可容纳三十人的小村庄。发现有兔形生物，高度约1m40。这些生物通常会躲避受试者，故无法建立联系。谨慎的观察使得发现一个新的有组织的社会制度成为可能，每个社区成员都有自己的角色。然而，这个观察难度很高，因为这个世界的居民越来越警觉，因为他们早已看见了负责观察它们的受试者。
SCP-017-FR-07 白色的金丝镶边封面，共564面 / 亚马逊、印度尼西亚、冰岛 无数据，所有受试者“进入”本书后即立刻死亡。
+ 访谈记录
- 访谈记录

当基金会特工到达发现的第一座寺庙时，一名年龄为六十岁左右的亚洲人进入SCP-017-FR-07，但在接触之前被拦截。当特工抵达基金会发现的另外两座寺庙时，每个都有一个亚洲人在其中双脚盘起坐下，手中拿着一个SCP-017-FR-07的复制品。所有目击过项目者都被带回基金会。其所说的语言是根据目击者对基金会所表述出的侮辱性语言所知晓的。

访谈对象： 平（从冰岛捕获），被戴上手铐拘束于椅子上。
访谈者： Dr █████
前言： 获取关于项目的一切信息。
Dr █████ : 早上好，先生。那么，让我们早点开始吧。你叫什么名字？
平： 我叫平，我不认为你有资格知道我的全名。
Dr █████ : 随你便。那么……你当时在那个庙里做什么？
平 : 保护这些书，以免你们这群暴徒把它们抢走。
Dr █████ : 这些书，它们究竟是什么？
平 : 你不懂。我也不会向你解释。
Dr █████ : 在我们到达这里时，你为什么要从这本书中逃跑？要知道你可跑不掉，我们牢牢控制着这副肉体。
平 : 我的身体只不过是一个信封！真正重要的是我的头脑。
Dr █████ : 为什么这本书有什么特别之处吗？
平 : 没有什么特别的。这只是其中的一本书。
Dr █████ : (从盒子里拿出SCP-017-FR-07的复制品) 真的……没有吗？
平 : (烦躁不安，欲起身) 把那个给我，混球！你没有权利进入白城！你会玷污它！
Dr █████ : 为什么？这只是其中一本书而已嘛，你自己说的。莫非它里面确实有什么？ 我们有三个这种东西，而且可以毫不犹豫地摧毁其中的一两个。
平 : (进入强烈不安的状态)。我…我屁都不会告诉你！
Dr █████ : 你已经露太多马脚了，现在现在已经没有回头路了！
平 : 我……我…… (咬断自身舌头，呈极度痛苦状)
Dr █████ : 妈的！把医疗队叫过来！

分析 : 男子最终死亡，喉咙被从舌头涌出的血液窒息。我们没有试图深入研究SCP-017-FR-07，甚至让基金会的一名员工使用了其中的一本。虽然在使用过程中没有人脑死亡，但相应地，项目也没能得到更进一步的探索。

+ 事件报告-017-FR-02
- 事件报告-017-FR-02

指定D级人员D-3251，观察并试图与SCP-017-FR-02中的土著居民接触，并指示每4小时报告一次情况。但是，一个星期后，D-3251没有回复。脑波指示没有变化，现假设其被当地人抓获，书被回收。当SCP-017-FR-02从其手中移开时，D-3251脑死亡。D-3254被选中取代之并继续实验。与以前的探索相比，D-3254的报告与之前有显著的差异。

访谈对象： D-3254
访谈者： Dr █████
前言： 更新关于SCP-017-FR-02的信息
Dr █████ : D-3254，请开始描述你那边的情况。
D-3254 : 这里很热，博士。我看见三个村民正在巡逻。
Dr █████ : 还有呢？他们的动向正常否？
D-3254 : 我不觉得正常，博士，他们武装到了牙齿，显然正在找什么。
Dr █████ : 他们在打猎吗？
D-3254 : 不像。他们有时会径直穿过动物群。他们有时候会跳起来，但是似乎当他们看到了什么之后，他们就退回原位了。
Dr █████ : 你的任务是观察，并与他们建立对话。你尝试过了吗？
D-3254 : 我试过了，我跟他们打过招呼，那些混蛋想用矛把我刺死，我不得不拼命跑！
Dr █████ : 学会巡逻对他们来说确实是一个长进。但是那个世界的人们也在观察着我们是否在他们的领土上，也许这对我们的存在不过是一个简单的反应而已。
D-3254 : 当然了，但这还不是全部！我看到他们的头儿！他的衣服和穿着，和我完全一样！ 一样！他正在对他们大喊！他当时坐在他的椅子上，其他人就跑过来把他带回了村子里去！
Dr █████ : 这确实是一个重要的新发现，非常重要。这次简报结束了，你可以回到你的牢房去了。

分析 : 在接受采访之后，伪装专精特工Sting被派去对新一任部落酋长进行更有针对性的观察。看起来，尽管D-3251的肉体已经死亡，但他的精神实体依然存在于书中，他利用自己的知识成为他们的领袖，从而更好地利用了当地人。部落的不寻常的行为与D-3251希望保护自己免受基金会对他的报复的行为相符合。
