Usage

To import this theme to your page, put the following anywhere inside it:

[[include component:creepypasta-theme]]

The standard CSS @import method won't work. This is intentional (thanks to stormbreath for providing the CSS code!). Importing the theme using [[include]] adds your page to a list of Backlinks. This is useful for seeing what pages would be affected when this theme is updated, and allows any errors caused by that to be noticed and corrected much faster.

You can view any page's Backlinks by clicking "+ Options" at the bottom of the page, then "Backlinks".

AnagallisarvensisL. 07/16/12 (Thur) 23:11:56 #17381048

诡秘的地下世界与我们的生活之间，就隔了一小块羸弱的金属板。

从人类智慧的星火蔓延开来，我们意识到自身的脆弱，转而铸造城墙抵御外敌开始，麻烦也渐渐多了起来。壁垒自然可以保护我们，但同时也将其他的东西禁锢其中。如果不加以疏导，仅仅是一场暴雨，便可以轻松将高墙内的建筑冲垮。

所以排水系统便应运而生。据说最早的排水系统起源于早已沉没的大西国，不过至今为止，这神话中的文明真实存在与否尚无法定论。但从同样古老神秘的埃及文明遗留下来的排水系统来看，当时的人们对于地下水道的建设早已经深谙其道。而经过几千年的发展和进步，如今的地下水道更是纵横交错、四通八达，酷似文化作品中的地牢或是迷宫。而对于这样深埋于脚下的另一个世界，人们经常会抱有恐惧——是否这迷宫中也有着啖人的米诺陶诺斯。

这并不算是无端的臆想。为了积水能够顺利流下，下水井的井盖处常有孔洞。粗心的路人掉落的物品或许会进入其中，而稍大一些或是密封得不够严密的井口，更是会吞噬不少流浪的动物。它们的绝大部分都摔死或淹死了，但也有少部分会因为各种各样的原因而存活下来，并在那里定居、繁衍，然后在没有天敌的地方肆意成长和进化。有报道说，在地下水道作业的工人们曾见到无比巨大的鼠类骨骼或是其他的什么顺水流下，但都因为没有证据而被当做笑谈。

AnagallisarvensisL. 07/16/12 (Thur) 23:53:09 #17382609

在当时，也的确曾经盛行过一段地下怪物的热潮。阴谋论者、猎奇者、各种各样的爱好者们都参与进来，讨论我们无法见到的、下水道内的种种。不得不说，有些人很精明，趁着人们对于地下世界探索的狂热，拍摄了不少类似《地底追猎》或者《怪兽密码》这样的伪纪录片。跟随拍摄者的镜头，普通人们也有机会进入那片黑暗恶臭的境地，去追捕怖人怪物的残影来满足自己的好奇心。

而事实上这些影片毫无例外都是假的，要么是除了一点人造的踪迹，根本就没有怪物的影子；要么那些所谓的怪物，只是一张丑陋的皮套。影片拍摄地点也大都是在临时搭建的摄影棚，用灯光和特效制作出类似下水道的效果，那些“探秘者”根本不会真正跑到漆黑的地下水道中，他们只是想沾热度，好趁机大捞一笔。

但有一部影片其真实性却仍值得怀疑。《暗影探秘者》这部接近1小时40分钟的录影相较于它的同类，在画质和拍摄手法上都显得更为拙劣和粗糙，而不存在演员表这一点更是让它看起来像是私人录制的视频。

《暗影探秘者》中的一幕。我费了好大劲才搜罗到它。


影片从四人潜入地下水道开始，慢慢向内探索并在墙壁上设置了信标。在大约1小时5分钟的地方，这些人已经进入了相当远的地方，深处似乎传来了某种……声音。其中一名男性决定不顾劝阻打头阵向前探路，但很快，连惊叫都没有，他便失踪在一片淤泥翻滚声中。其他人发现不对劲向前追赶，却只发现一堆巨大的、不知道是什么的东西潜藏在前方的阴影中——而它最终转向了他们。

后面的部分几乎无法观看，因为摄影机根本无法在疯狂的奔跑中好好地工作。我们能听到的只有飞奔的脚步声、尖叫，以及某种粘稠的低吼。影片到他们爬上扶梯便结束了。最后的一幕里，原本四人中只剩下了一位男性和摄影师瘫坐在地上。

影片似乎仅仅发行了几百张，而这四人最终也没了音讯。

AnagallisarvensisL. 07/17/12 (Fri) 00:10:36 #17384850

而另一方面，有一些杀人事件似乎也迷雾重重。其中最著名的应该要数福岛便池藏尸案。

在这件悬案中，男人以一种奇怪的姿势死在女厕的便池下，而事实上这个便池口根本容不下这样体型的人类进入。最终警员们认定男人是为偷窥而潜入便池，最终无法逃脱，死于其中。这种说法显然不合情理，其中仍然有无数的疑点。

如果男人是被从另一边的排水通道拖入的呢？似乎没人这样想过。凶手是没办法将男人以半蜷缩的姿势拖入便池的下水口，然后全身而退的。但为何凶手一定要是人类呢？这看起来像无稽之谈，但又不无道理。如果那东西能改变自己的外形，这样似乎就并非难事了。

事实上，除了这一案件，还有更多离奇的密室失踪或是杀人事件，只要你肯寻找，就会发现许许多多的无头悬案。

目光转回《暗影探秘者》，在这部影片发行过后不到一周，当地发生了一起有记录的失踪案件。Burner Wilhelm，他在失踪前似乎患上了某种精神疾病而一直待在四层的公寓中，监控并未观察到他外出。在他的公寓厕所地上发现了大量的淤泥。而他的相貌与影片中幸存的男人出奇得相似。

由于年代久远，我查不到更多的细节。但我想，如果这个男人真的是影片中的男性，那他的失踪便并非毫无根据的。

我们不清楚，或许也永不清楚，在家中听到的管道流水声，究竟是什么发出的。
