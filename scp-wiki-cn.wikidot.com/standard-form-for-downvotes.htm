为了对导致downvote的原因进行归纳，并且为了使这些原因看起来不是那么的针对个人，于是我就写了：

SCP基金会标准1downvote守则1040-EZ

我会downvote你的SCP，那是因为：（请自我检查一下）

[ ]你忘记检查拼写了。
[ ]你没有做校对。
[ ]你没有读过，或者直接忽视掉了“如何撰写一篇SCP（How to Write an SCP）”。
[ ]你在主站上发布的是一个未完工的SCP。
[ ]你的格式太糟糕。
[ ]你不知道its/it's; your/you're; there/their/they're的区别。
[ ]你好像不大会用英语写作。（译注：这里该改成中文？）
[ ]严重缺乏客观冷静的描述手法。
[ ]你可能不小心发布文章的时候搞成打鲨鱼中心（Shark Punching Center，SPC）了
[ ]给自己upvote是错误的行为，别这么做。
[ ]如果在论坛上晒出了你的草稿，有20个人告诉你别这样写的文章。
[ ]如果在沙盒上晒出了你的草稿，有20个人告诉你别这样写的文章。
[ ]你的太像SCP-__了，而且还没有能力完成它。
[ ]你的SCP直接剽窃了__。
[ ]你的SCP题材已经泛滥了，而你的写作水平又不足以让它脱颖而出。
[ ]请把你的D&D妖怪/人造/魔法物品/诅咒魔法物品送回那被遗忘的国度吧。
[ ]强奸不是一个好的SCP主题/效应。
[ ]恋尸癖什么的绝不可以。你的SCP应该要把我吓得睡不着觉，但绝不是通过让我奇怪你为什么还没进局子。
[ ]没有吸引力。
[ ]无聊。
[ ]没能吸引我：抱歉。
[ ]你的J条目不好玩。
[ ]你的J条目只是一个很差的主系列条目。
[ ]你的“Keter”级SCP仅仅只是危险，而不有趣。
[ ]你的描述并没有在描述你的SCP。
[ ]如果你的SCP“看上去”像是个啥（搬运注：大概是指玩梗）的话，它绝不应该是那个啥。
[ ]我得读到一半以后才能意识得到你写的是什么。
[ ]我读完了都不知道你的SCP究竟是什么。
[ ]你[已编辑]了收容措施部分。
[ ]你[已编辑]了整个事件报告。SCP-087可以不代表你就可以。
[ ]你[已编辑]了主要效应。SCP-231（或多或少地）可以不代表你就可以。
[ ]你[已编辑]了有吸引力的部分。
[ ]你[已编辑]了一些不该删除的东西
[ ]你不停地用着“模因”这个词，不过看起来你对这个词的词义的理解跟我有着很大的出入。
[ ]你的SCP毫无理由便强迫别人去使用它。
[ ]你的SCP不过是个能杀人的__。
[ ]你的SCP不过是一个能让人发疯，或者疯掉之后杀死他们的__ 。
[ ]你的“真实向”SCP还不如它在现实中的本体更为有趣。
[ ] 你的SCP徒劳地去解释: [ ] 卡特里娜飓风 [ ] 福岛地震 [ ] 印度尼西亚海啸 [ ] （其它真实发生的灾害）
[ ]你的采访日志中的对话写得仿佛全人类看起来都是彻头彻尾的傻瓜。
[ ]没常识到我想一把火点了我的电脑。
[ ]你的SCP进入了某种“狂暴状态”，但是你却没写它是怎么制造破坏的（visiting New Jersey）。
[ ]你的SCP发现故事太离谱了。
[ ]你的一些文段部分是从古老的语言翻译过了的……可为什么还会出现英语的韵脚
[ ]请移除：
— [ ]不恰当的标题
— [ ]枯燥的实验日志
— [ ]枯燥的补充页面
— [ ]非度量的测量
— [ ]搞笑基金会出版的附录
— [ ]自己添加的“标签”
— [ ] Telekill般的乱入。（telekill，参见148）
— [ ]与SCP-173/500/682/914进行交互实验
— [ ]随便/过度加入Clef/Bright/Gears/Rights/Kondraki/(以及其他的被用滥了的“经典”角色)
— [ ]关于鲨鱼和/或蜜蜂的蹩脚笑话。
[ ]你的SCP让我心生疑窦。特别是：
— [ ] 错误的科学引用
— [ ] 模糊科学
— [ ] 过时的
— [ ] 打斗情节得用英式摇滚来配音的
— [ ] 某个特工在他家的台阶上/一个奇怪的小店里/度假时在沙滩上/（插入一些滑稽的故事）发现你的SCP的。
— [ ] 其它什么的:
[ ] 你的SCP被收容于：
— [ ]一个无限大的收容室
— [ ]二维空间
— [ ]六维空间
— [ ]钛金属收容间（或其它不合适的材料）
— [ ]没有：你的SCP可以在Site里随处闲逛。
[ ]你的收容措施让人完全摸不着头脑，或者和收容对象毫无关系。
[ ]你的SCP“未被收容”仅仅是因为你懒得去想怎么收容。
[ ]你的人形SCP属于以下这几种：
— [ ]一个“X战警”或者超级反派。
— [ ]另一个龙与地下城的怪物。
— [ ]很差的自我代入/“玛丽苏”/“龙傲天”
— [ ]使用SCP格式描写明显出自其他作品的东西。对Slenderman就是说你。
— [ ]把基金会当作酒店一样使用。
— [ ]给它它想要的东西仅仅是因为我们不想它生气。
[ ]你的配图：
— [ ]你自己手绘的“印象派”作品，
— [ ]27MB，站内弄不上来。
— [ ]其它SCP用过的。
— [ ]完全跟你的SCP不搭调的
— [ ]电影剧照，或是广为人知的图片。
— [ ]有水印。
— [ ]带有诋毁性质的。
— [ ]明显是计算机合成的。
— [ ]PS太明显。
— [ ]没好好PS。
— [ ]即使没PS过，但看上去怎么都像是PS的。

提示：请记住，在实际讨论中不要使用这种格式，对此完全零容忍。
