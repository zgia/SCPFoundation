SCP-379-FR

项目编号：SCP-379-FR

威胁级别：黄●

项目等级：Safe

特殊收容措施：SCP-379-FR的两个个体必须收容于Site-██L122的标准收容库内。由于启用SCP-379-FR-2的长期影响，未经道德伦理委员会的书面许可，禁止任何SCP-379-FR有关的实验。

必须对任何新发生的疑似SCP-379-FR-A事件进行全面调查，并将其完整记录于存档档案中。调查须在严密掩护下进行，而不提及敏感或异常因素，因为无法对受影响者进行记忆删除。如确定SCP-379-FR-A的行为不构成对行动人员身体完整性的障碍，则可以提供心理治疗。

任何已知的SCP-379-FR-A事件都必须以中等优先级的跟进，以防止事件再次发生以及修复其造成的损伤。

描述：SCP-379-FR由两个特定的要素组成：

  • SCP-379-FR-1是一块胫骨，据称属于St.███████的遗体。该实体处于标准保存状态，并且在与球面关节接合处呈现一些多孔区域。遗骨被放置在一个一个25 x 10 x 10厘米的胸型小金色展示架内，其脊部采用纯金装饰，并雕刻有圣经人物（圣母玛利亚与3个天使）。面部为略微着色的玻璃，可以透过玻璃观察。
  • SCP-379-FR-2是一个长方体形状的空心瓮，有一个略微倾斜的面和用于捐赠钱款的槽。其质量略低于SCP-379-FR-1上的纯金，但装饰相似。应该在捐赠者面前呈现的面孔已经模糊不清，但只有字样“Deus dat et Accipit Deus”（“上帝给予，上帝再来”）1仍然清晰可辨。

SCP-379-FR的异常只有在SCP-379-FR-1靠近SCP-379-FR-2，并且在SCP-379-FR-1面前当面捐赠给Ste███████时才会激活。
当至少一枚硬币被丢入SCP-379-FR-2时，异常具有一定的被激活概率2，并且对St.███████的捐赠与祈祷者生效。异常激活的概率等于沉积在沉积件数上的碎片数加1，也就是说根据公式 :

P激活 = n投币数 / n投币数 + 1

如果异常被激活，那么存放硬币的人将接收他们个人祈祷似乎有效的“迹象”。可能包括寒意，与祈祷内容相符的琐碎事件，温暖的感觉等。然而，在捐赠后的二十四小时内，异常没有其他直接影响可察觉。

任何成功激活异常的人似乎都被一个无定型的实体跟随，以下称为SCP-379-FR-A。SCP-379-FR-A的存在尚未得到证实，但目前对于异常的现象是最合理的假设。

SCP-379-FR-A仅在捐赠时异常激活后二十四小时被激活，并且与当时祈祷内容有关。因此，在可能的随机时间内，如祈祷后的7天延伸到42天并且通常被称为“合作阶段”，该实体似乎能够直接或间接地影响周边环境。受影响者将会帮助“执行”其祷告（参见例子表）。

被确定为受SCP-379-FR异常影响者的证词表明有事物在此期间特意但定期地“存在”于周遭，并伴随着被认为是几率性的事件以绝对的可能性发生。尽管对SCP-379-FR-A所谓的物理外观的描述并不匹配，但所有描述都有一个共同点，即实体通常只会出现在人眼视野的边缘，并会跟随人的视场移动。

在“合作阶段”结束后大约一到两天（即祈祷已经完成），受影响个体的环境急剧瞬间变化，其将触发一个时间异常，虽然时间正常流逝，但受影响者将会发现自己处于祈祷之前的原始状态。所有其他未受影响者的记忆都会被修改，而最初祷告的受影响者则会成为唯一对异常留存有记忆的人。

基金会Dr Shizue通过受SCP-379-FR影响者的证词，以及使用一套工具检测时空扰动，了解了这种时间上的异常现象。

最后，第二个时期通常被称为“反制阶段”，并且异常影响持续时间为无限期（直到受影响者最终死亡）。在此过程中，SCP-379-FR-A也通过直接或间接影响环境来适应最初的祷告词内容。另一方面，与“合作阶段”不同，“反制阶段”的项目似乎能够直接修改与消灭可能导致受影响者祷告词无法实现的一切外部事件或个体。因此，如果受影响者在时间异常期间没有与祷告相关的改善，上述阶段即马上开始。

现已证明，对受影响者进行记忆清除是无用的：实体并不会消失，而且受影响者对自身情况和实体的存在只能徒增其困惑。

在St.███████执行的祷告与SCP-379-FR-A的反应。

受影响者 祷告内容 "合作阶段" "反制阶段"
Romain M.，53岁 希望能付清他母亲的手术费。 Roman M.赢得了彩票，并且能够支付他母亲的手术费用，后者康复出院 =回溯之后，Romain M.收到了一位远房亲属的遗产，但在行政程序上和对他母亲的深度治疗中使用了它。
Elizabeth S.，23岁 希望怀上一个孩子 Elizabeth S. 虽然生育系统天生有问题，但仍然成功怀孕 回溯后，Elizabeth S. 连续怀孕8次，2次宣布胎儿死亡，6次流产。
████ █.，██岁 希望保证他的新公司的建立 其公司在短时间内成功建立 =回溯后，其公司被富有的投资者以适中的价格收购，但该公司的股价迅速下跌并被宣告破产。
Michelle D.，87岁 治愈他的病 Michelle D. 肠息肉被清除，病症消失。 回溯后，Michelle D.先后经历了3次癌症，但从未死亡。
特工Michaël V.，42岁 未知 未知 （假设的）回溯后，2017年7月12日，SCP-███-FR收容失效并在Site-██的东翼引起了事故。特工V.向他的同事透露，他“预感所有这些[“都是他的错”]，他“立即[理解]这件事[是无法解决]的”，为SCP-███-FR重新收容牺牲了自己。在军官的储物柜中发现的遗嘱将基金会收容SCP-379-FR的工作提上了日程。

特工MichaëlV的遗嘱告知基金会SCP-379-FR-A的存在。2017年11月27日，收容工作遭遇了来自SAPHIR的轻微抵抗。基金会最终成功收容该项目，SAPHIR的一名成员被目击正给SCP-379-FR投币。目前该名成员被捕，数日后进行了访谈。

 SCP-379-FR收容后对Solomon D.的采访
收起

日期：03/02/2018
访谈人：Solomon D██████，SAPHIR成员，27岁。
访谈者：特工Hodges

特工Hodges : 你好，先生。我是特工Hodges，我在这里向你询问SCP-379-FR的事情。那么，首先，你在[删除]教堂做些什么？

Solomon D. : 我们3个人，都是同事。我需要钱，很大一比的钱，Dav和另外一个人有一个计划：他最近跟一个团体接触，他称他们为SAPHIR，想加入就必须抛弃“所有信仰”。还有另外一条路，那就是一位可以帮助我们的老太太。

Agent Hodges : 老太太？那与赚钱有什么关系？

Solomon D. : 起初，我也不知道。但我跟着他们，我们去了老妇人的房子……我以为她会魔法，就像那样。然后Dav，他……他晚上进了她家，然后袭击了她。你看见树干中的小腿骨了吗……？ 基本……就是这样。我们在教堂里清楚地说明了这一点，我们买通了守卫，然后就开始筹钱。

Agent Hodged : 知道了。那你是否清楚，这块小腿骨并不正常呢？

Solomon D. : 之后大卫向我解释过。我们对它的影响知之甚少，但他确信这会“让他们远离他们的假神”。他之前不是那样的，你知道，他之前很正常。但是，我必须承认他的演讲准备充分。 我们会相信，一些东西，之类的。

Agent Hodges : 当你被捕时，你正在祈祷。你祈祷的是什么内容？生效了吗？

Solomon D. 看向后方，似乎忧虑重重。

Solomon D. : 我……我不会告诉你的。你知道，“誓言如果被揭露就会被取消”，就像那样。 我们无法确定这件事是如何运作的。另一方面，我可以告诉你，你没法长时间困住我。

Agent Hodges : 你怎么能确定？你的同行会来救你吗？

Solomon D. : SAPHIR？不会的。我只是为这个St.███████祈祷而已，真的相信它。你知道它对那些人意味着什么。他们不会来，他们甚至懒得懒得来救我。

随后，Solomon D. 不再回答任何问题。

收起

附录SCP-379-FR-1：在2014年6月2日，在涉及SCP-███-FR收容失效的事件中，一个未知突击小组（不隶属于SAPHIR）的攻击和几个人为的错误导致Solomon D.成功越过食堂警卫的看守，逃离基金会的站点。怀疑这一系列的巧合是由SCP-379-FR-A在他被捕后祈祷后分配给所罗门D.的行为引起的；另一方面，当受影响者返回其原始位置时，推测其祷告内容并没有实现，抑或者此次收容失效导致的越狱是由“反制阶段”所引起的。

附录SCP-379-FR-2：在[删除]区域搜索并与参加[删除]教堂的工作人员名单交叉对比后，找到了Solomon D.提到的“老太太”。这是94岁的露西███████。她正式承认Solomon D.是袭击他的3人团体的一部分，但声称他记得他是下达命令并发起袭击的人。
正在进行审讯的特工Hodges也注意到了这位女子双腿的存在，以及她无忧无虑地行走的能力。

脚注
1. 应该注意的是，句子不正确，似乎来自于在线翻译软件，正确的措辞应该是“Deus dat et Deus recuperat”。
2. 不论何种货币。
