研究员Rachud正展示他从不离身的挂饰

姓名：Rachud K███

安保等级：3

专长：流体力学，认知危害，逆模因，布阵型奇术（经F级记忆删除后推测暂不可利用）

目前职位：研究员

人事履历：

  • 199█/█~1998/█ 从蛇之手组织脱离，并[权限不足]。
  • 1998/█~2001/7 作为水色型个体，于Area-CN-07-β[权限不足]。其掌握的布阵型奇术技术被吸纳至基金会基础奇术理论储备。
  • 2001/8~2005/7 因[权限不足]，接受F级记忆删除。依本人在记忆删除前志愿前往基金会旗下的██大学研究院深造，专攻认知危害与模因学，副修英语与日语。
  • 2005/8~2008/4 协议期满，其未取得学位。本人作为文书人员调入Site-CN-██的申请被通过，授予1级权限，负责部分别站文档的翻译工作。
  • 2008/5~2013/12 升迁为助理研究员，授予2级权限。跟随上官蒹萍高级研究员（详见附录五）进行项目研究的申请被通过。
  • 2014/1~2016/8 升迁为研究员，授予3级权限。[权限不足]
  • [权限不足]

附录一：
于200█年的员工体检中确诊轻度语言障碍，主诊医师███判断保持适量的日常阅读与文书工作有助于缓解病症。[权限不足]特许其在测试中仅条理性一项不合格时通过其进行文书工作的申请。为此将其编入处理低优先度文档人员的列表中，但其经手的部分文书仍会偶尔被部分员工投诉，已经研究员Sraosha引荐参加Dr.Rear的文档格式讲座。尽管其文档书写条理性在长期的工作锻炼中得到改善，仍不允许其独自完成文档的审核确认工作。

可确认讲座的成效不高，其仅是机械地搬运被认为是优秀的文档格式，但仍缺乏对整体美观的理解。——feitag

附录二：
其于进行F级记忆删除前定制了一直径为6厘米的青白色透明球状饰物，其内嵌入雕成气旋状的绿水晶。其要求在F级记忆删除后被劝说将之随身携带。除绿水晶形状与其最擅长的布阵形状框架相似外，该饰物无其他值得注意之处，亦不存在任何的奇术作用。其要求被通过并在该饰物内植入了[权限不足]。

附录三：
自接触SCP-CN-301后似乎对心理学产生了过度的关注，并多次提交了查看更完整版本的自身测试结果的申请。申请均被拒绝并给予其警告处分。

值得一提的是，助理研究员Rachud的数值明显地偏离于基金会研究人员的常模，其数据61的过低数值与其基金会忠诚度测试的结果有所矛盾，已对此进行调查调查由█████下令终止，不得将SCP-CN-301应用于非研究目的，下令调查的人事部门员工已被处分。以研究员Rachud为模板进行的系列写入测试显示，除非将数据1下调[权限不足]，D级人员均会表现出精神分裂症的特征，推测其背景的特殊性是研究员Rachud的测试数据异常的原因，已交由[权限不足]研究F级记忆删除剂的[权限不足]。

附录四：
在进行文书工作期间，Rachud对部分涉及奇术的文档表现出了过度的关注，为维持F级记忆删除的效果，已额外增加了其经手文档的审核要求其对SCP-CN-048-J的文书工作产生了高度兴趣，尽管其明确知道此为内部的玩笑性质文档。可确定F级记忆删除已将其对奇术领域的亲近感扭曲为常态认知下的小众爱好。为此，不影响其本职工作的相关兴趣活动是被鼓励允许的。

附录五：

高级研究员上官蒹萍的人员档案

20██年接触艺术家Bread时由后者所绘。其本人似乎对之非常满意，有数次将其用于各种正式场合充当证件照的不当尝试

姓名：上官蒹萍

安保等级：4

专长：模因与异常信息，语言学，[权限不足]

目前职位：高级研究员

人事履历：

  • [权限不足]
  • 199█/██~1998/█ 于Site-9062供职，期间从助理研究员升任至研究员，并取得3级安保权限等级。自1997█/█因[权限不足]，经O5议会命令不得接触[权限不足]相关异常的情报。
  • 1998/█~201█/█ 调往[权限不足]，主要负责中、高危级的模因异常与信息危害异常。享有双休与[权限不足]天年假的福利，前提是必须每年对6名以上文书人员进行外国文档文书的翻译训练，训练时间不算入正常工作时间。
  • [权限不足]

相关情报：
被研究员Rachud以老师及带路人的身份敬重。上官蒹萍曾声称自己能轻松地支使研究员Rachud，“起码当年为了他吃的苦没白受”。据调查两人的确关系融洽。

逆向自布阵型奇术技术的奇术设备：
  • E-R式气体奇术操纵仪：
    1. 功能：以仪器为中心的球型区域内气体的整体流向操纵与（或）作用范围内气体的喷射控制
    2. 功能参数：
      1. 作用半径：2米（常规、基准人类)，37.4米（极限，Ⅱ级以上奇术师）
      2. 延展/压缩范围：0.4~7 atm（常规、基准人类），0.03~26.4 atm（极限，Ⅳ级以上奇术师）
      3. 最大动能：2.4kJ/cm3（常规、基准人类），>1×101█kJ（作用区域内整体，Ⅳ级以上奇术师）
      4. 备注：极限值不能同时取得，有效动能因动量守恒及角动量守恒限制，实测平均利用率为32%
    3. 警告：该设备作用边界为混沌式，不建议针对3级或以上的危害应用临时气密处理。
    4. 推荐配备设备：小口径子弹（需Ⅱ级以上奇术师，以操纵形成子弹的加速用空气管道区域）、压缩空气罐（为操作者安全起见慎选其他气体）
    5. 特殊保养须知：因布阵型奇术的特色，回冲补正结构在空间上与法阵主体重合，表现为勾勒出布阵大体结构的光源，在待机状态下于黑暗于可见，如光强减弱需及时更换EVE源（推荐奇术师遵从此条，以免设备的自维持机制失效导致法阵溃散）。中央空洞上下两侧及设备内部的法阵空腔不可有杂物堵塞，回冲补正结构发光不完整时应停止使用该设备，掌握布阵式奇术技术的奇术师在紧急情况下可尝试利用设备外置法阵）
  • SCP-CN-854 履历相关，展露基金会与奇术师互动的一角。
