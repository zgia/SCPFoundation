# SiteCN 21简介

SiteCN21是SCP基金会的聊天频道。(测试)
SiteCN21是模仿主站Site19在线聊天室的模式(然而并不一样)

  • 谈论SCP。
  • 获得有关您的文章草稿的建议。
  • 与来自世界各地的网站成员进行社交，包括新人和网站资深人士。

聊天对所有人开放。您不必是已加入的网站成员。

聊天指南

  • 年龄：您必须年满15岁。如果您的行为与小学生一样，您可能会被禁止。
  • 主题：关于SCP想法和草案的讨论优先。不允许下面列出的聊天内容。期间不允许非法活动。
  • 报告：如果您有任何问题/建议，您可以通过在页面下评论来获得帮助。

禁止的行为(搬运自主站)

每个聊天用户都有责任知道什么是可接受的行为或不可接受的行为。

以下是一些要避免的具体事项。这并不包括所有可能的冒犯。

  • 拖钓：故意煽动性。如果你这样做，你知道你在做。如果操作员认为你是在拖钓并且你不这么认为，那么你需要重新思考你的行为。这包括根据性别，性别，种族等对一群人或一群人进行定位。
  • 骚扰：询问图片，强迫性生活讨论，等待或者反复与要求你停止与他们交谈的用户交谈。
  • 几乎任何涉及强奸的事。
  • 垃圾刷屏。
  • 不要使用＃SiteCN21打扰别人。没人在乎。不要使用＃SiteCN 21来进行拉票

关于程式

你可以在这里找到一些关于聊天室以及帮助排查故障的帮助。

说明

  • 目前服务器仍在测试中，聊天数据将频繁清空。
  • 服务器近期出现过载，访问出现“当前在线访问人数较多…”请刷新页面再试。正在抢修升级服务器，带来的不便敬请谅解!

故障排查

如果下方没有出现小框，请耐心等待
如果出现小框但报错，你可以尝试以下步骤

  • 刷新页面
  • 不要使用移动设备(手机/ipad)浏览
  • 不要使用IE浏览器浏览(可以，但有时会出问题)
  • 在其他人的设备上试试

如果尝试了以上所有步骤仍未解决问题，你可以在评论区留言，我将尽快解决


如果你还有疑问/建议，你可以在页面下方发表评论。
一定要在通读以上内容的情况下进入聊天！

服务器选择

  • PHP不记名聊天服务器
  • Webscoket在线群聊

特色

  • ★可以发送图片
  • ★保留聊天记录(在关闭页面后别人仍然可见你的话)
  • ★匿名聊天
  • ☆无法私聊
  • ☆IE必须手动刷新(对IE极端不友好）
  • ☆没有管理制度
  • ☆无法复制内容

适合

本服务器适合纯文字的简单聊天。你可以在这里体验到非常反大麻的聊天方式。

如果你不是IE，却看不见信息，请下翻页面(鼠标滚轮下滑）
如果出现故障，请刷新页面，或者阅读上方的[关于程式]
请静等聊天界面加载，如果是第一次访问大概需要等半分钟！

  • 在线缩图
  • 下载基金会专用浏览器(单文件)

特色

  • ★可以媲美QQ的聊天功能
  • ★可以发送图片
  • ★保留聊天记录(在关闭页面后别人仍然可见你的话)
  • ★支持私聊
  • ★支持自定义表情包
  • ★支持撤回消息！
  • ★完善的管理制度(私聊我申请管理员)
  • ★可以加好友
  • ★友好的用户界面
  • ☆对IE不太友好

注意事项

如需退出聊天，请直接关掉窗口。请不要点击左上角的退出按钮，那样会清空你的聊天记录。
本聊天室可以使用IE但不推荐。进入聊天室后点击群聊标志即可愉快的聊天。

管理可以在此根据比赛需要创建群聊，如需聊天室管理权限私聊我
