事件记录 Delta-3

眼前的15人并不只是普普通通的士兵而已。这些小伙子很自豪成为机动特遣队Delta-3“恶龙”，SCP基金会欧洲部门的主要武装力量的一员。如果基金会的座右铭是安全、收容、保护，那么Delta-3的座右铭则是“恶龙快递，死亡必达”。这是一支多才多艺的特遣队，他们能够用于捕获异常，进行空地支援或快速空中部署。在场三支队伍仅仅是广泛散布在欧洲大陆上众多机动分队的一小部分而已。综合他们以往的任务，他们注意到，他们即将完成了一次壮举——不计一切代价阻止一个不知名组织的壮大（鉴于此次行动敏感度与热度极高，很难知道谁真是真正的幕后黑手）。
他们被分成三支队伍，每一支都拥有自己的数架西科斯基UH-60“黑鹰”运输直升机。

其中第一队由Stefan Harcrow领导，绰号特洛伊大嘴汉。他是一个三十岁的英国男子，除了长了一个国字脸，没有什么特别之处，正如他的绰号所示，他总是拧着一副苦瓜脸，但这通常与他的真实情绪毫无关系。飞行员则是一位法国籍的短金发女郎，因为她开飞机的方式而被戏称为Heidy Sabine，绰号飞蛾。飞行技师则是来自日本的松·凯恩，因为他卓越的修理机能获称金手指。第一名狙击手是一个叫Georgi Vicherna的俄罗斯壮汉，绰号 破片 ——因为他总是习惯于在腰间别着两枚手雷。队伍里的2号狙击手叫Neremsa，因为他出名的悲观主义而被命名为Rabat-joie。至于直升机，则被称为“Kai-jiu”。

第二队的是飞行员一个亚洲人，他是从哪儿来的非常可疑（因为他从未提及他的过去），因为他午睡时间特别长而被命名为Chieu Sy-Quo，并且被称为 轮胎底儿。飞乘官是一个名叫阿西德·拉海（Asid Al-Rahai）的摩洛哥人，经常在一个喝醉的扑克之夜之后不断大叫，他的绰号便为死骗子。德国的机械师被命名为Joshua Gersimm，绰号为螺栓切割者。狙击手，或更确切地说是1号女狙击手，是一个叫Viviane Sherman的红头发脾气暴躁的女士，因为她能够发射任何材料而绰号赌徒。2号狙击手是Atyarya Rostrova-Vicherna，绰号 冰心 。不知为什么，这位一头棕色头发的俄罗斯人嫁给了一个叫Georgi的人。他们的直升机被命名为“Toyour”，意思是 在摩洛哥的鸟。

最后的第三队，飞乘官是一个叫做Damien Allessandro的弗朗哥 - 西班牙人，昵称老姜不是因为他的栗色头发，而是因为它有很多污渍和斑点。这位飞行员是挪威人，名叫Garret Paulsen，有着一双灰色眼睛。他的绰号叫做Jacasseur。机械师有着法国-埃及血统，名字叫Nesenna Semnu，Nefertiti。两名狙击手是一对兄弟，分别名为Beno和Chezzo Armelba，绰号为Mario和 Luigi。他们的直升机绰号为“乌鸦”。

+++日期 : 03.08.10 – 08.47+++
+++计算时区：T+00/00/00+++

+++标识 :+++

»队伍 : 机动特遣队Delta-3
»呼号 : Les Dragons
»注册人 : Neremsa Vandrake
»队号 : 09919051
»认证级别 : II
»项目等级 : C
»级别 : Opérateur de F.I.M.
»密码 : ████████████

+++准许接入：+++
+++开始通讯：+++
+++连接加密通道：+++
+++任务简报：+++
>UTM 61.910125,32.141386（俄罗斯卡累利阿共和国皮日茨基以北13.12公里）拦截飞行异常
>未知的目标特征，被描述为“翼龙（Hatzegopteryx thambema）由金属碎片组成，翼展13米，长度8米，重量未知”。潜在危险，授权使用武力捕获。无其他异常记录。
>项目等级：估测为Safe
>威胁等级 : 估测为绿
>行动舆论掩盖：军事演习
>投入力量 : 分队1, 2 与 3
>空中捕获标准程序 - 见Delta3-58b文件
>任务开始 : 03.08.10 – 1230小时

+++通讯结束+++
+++连接中断+++

Neremsa关闭了手腕上的通讯装置，并打了一个哈欠。他把手机放在了身上众多口袋的其中一个里，捋了捋刚蓄了没多久的胡子，走向兵营的食堂。熬夜修仙是痛苦的，尽管基地位于芬兰和俄罗斯之间的边界的北部，但天气依然出奇地炎热。最重要的是，空调已经两天没吱一声了。Joshua，Matsue和Nesenna还在徒劳地修理它。讽刺的是，对于维修直升机来说，这三人可以称得上是华佗在世，妙手回春，但此时却连个风扇冷却管盘都搞不定。当天晚上，在这种只属于特种部队典型的嘲讽气氛中，激烈的言论总是能够震撼每一个人的内心。

食堂看起来几乎没有一个人。 Neremsa看到Armelba兄弟正坐在一起激烈得讨论着一些什么，Asid和Garret则一起悄悄地吃饭。当Viviane抬起左脚在他面前经过时，口中念念有词，他正准备跟他们搭上几句话。Neremsa和他的同事侃了几句，然后在咖啡机倒满杯子之前跟那个红头发女士搭上了话，虽说她那满口粗鄙之语很难让人高兴起来。

- “把咖啡给我加满，他妈的！”

- “社会文明你我他，如果你语气好点，也许她会真的帮你加满。”Neremsa笑道。

Viviane快速转过身来，瞪着他，把手中的叉子指向他的鼻子。

- 你等着，到我体内咖啡因在血液浓度的一半水平之后，我特么把你射爆。妈妈的，我听到你在笑，我向你发誓，如果你是我的目标，我会把你大卸八块！

有趣的是，在咖啡机后面，就放着两个充满清澈绿色液体的玻璃瓶。Viviane的目光转了回来，她开始为自己辩解 :

- “嘿！听着，我没有，我压根没碰这两玩意！”

离开红发暴躁老姐，喝着咖啡，Neremsa和Matsu走到他们的同事的桌子上。比利时人注意到了玻璃杯：

- “这是茶吗？”

- “比这更好点，凉茶！”

- “放你娘的屁。”

Neremsa一口气把杯中的水喝光，然后交给Matsu，Matsu没有为他和桌子的其他人服务，非常感谢。在放弃喝咖啡和热茶的想法之后，Viviane坐到了Matsu的左边。其余的队员在他们离开的时候的时候渐渐让食堂的人多了起来。难得一见的是，Chieu并不是最后一个坐在桌子前的人，最后进入食堂的反而是Vicherna夫妇。餐具的噪音里夹杂着一些关于任务的谈话，讨论的结果是目标是一般的翼龙。当所有人就餐完毕后，Steve启动了广播:

- “都来简报室！快！”

简报室是基地里最大的房间。可以轻易地容纳五十名士兵，所以在场的十五人与这个房间的大小几乎不成比例。斯特凡正站在一块白板前面，在白板上画出了任务区的战术地图。他用激光笔指示了他们将用来拦截异常的航线。像往常一样，他们会使用三角形编队。他们抽出了一些人以领导这个编队，第3分队很幸运地当选。为了捕捉异常，他们还获得了测试新型榴弹的发射许可。如果发生特殊情况，每个人都会带上狙击步枪。如果异常被证明不可能捕获，即用反物质子弹将其直接击落。这是一个不会使那些古板的研究人员感到高兴的解决方案，但这对于团队的生存是必要的。

基金会所有部门中，机动特遣队除了有最高的死亡率（除了D级以及其他情况），有时还会不小心将一个异常摧毁，而不是捕获。干员们很难达到这样的目的，这令一些研究人员感到沮丧。Delta-3从未示弱，又一次他们摧毁了四件文物，包括三个在站点内的。当时，他们迫不得已这么干，以免落入GOC的手中。但是这个行为仍然引起了数名研究员的强烈不满。
小组会根据他们的目标的行动轨迹来商议决定使用什么样的作战方式。他们的偏好都是经典的狩猎技术，用最先进的直升机无情地追击目标，另外两架则在一旁监视，随时准备应对最糟糕的情况。这一战术的优点便是可以两队直升机交叉支援。

三架黑鹰直升机正在停机坪等候着他们，太阳光照在驾驶舱玻璃上四处反射。每个队伍都带着箱子或装备火速前往。所有人都佩戴了一只耳机，一个麦克风和一个平视显示器（HUD），当它与数据库相连时，就能够显示诸如高度，速度等有用的数据。

在完成最后的准备之后，装有捕网弹的MGL榴弹发射器安全地固定在位于直升机两侧的发射耳上，队员们随即等级。黑鹰直升机的引擎在数秒后之后呼啸起来，把地面上的灰尘吹散开去。其他三机接二连三获得升空许可，飞机一个接一个地拉升。Delta-3的队员们一开始就训练于在没有额外地面支援的情况下行动，他们的任务往往都是这种情况。

渡鸦 飞在了编队的最前方，引导着后方飞机航行至他们的目的地。
每名队员都切身感受到了那股危险的气息，但他们坚信他们是无敌的。
