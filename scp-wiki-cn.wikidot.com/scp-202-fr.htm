SCP-202-FR

SCP-202-FR的照片 可以在图像的顶部观察到面部。

项目编号： SCP-202-FR

威胁等级： 绿 ●

项目等级： Safe

特殊收容措施： 由于没有威胁，SCP-202-FR不需要任何特殊的收容措施，但是，在获得该地点之前存在的围栏已经被加固和电气化围栏所取代。

4个监视摄像机安装在围栏的4个角落，由安保人员进行持续控制。人力巡逻每周一次。

与SCP-202-FR相关的亲属成员在接受采访后接受了Ω级记忆删除1.

描述： SCP-202-FR是一种属于柏树属的树，其高度为90米。植物具有很强的新陈代谢：其能在20秒内生长出20cm的分枝。
项目位于希腊的克里特岛，克诺索斯，即希腊神话中的人物Cyparisse的死亡地点。项目之前下划于█████████家族的财产中，于██/████由基金会收容。

SCP-202-FR似乎是居住在希腊█████████的家族脸谱树。

整棵树的树皮（树枝和树干）呈现出类似于可识别的人类个体面孔的雕塑。 从根部三（3）米开始，所有代表的人大多是老人和所有死者。由于树枝没有树叶，只有树的顶端3米形成了树冠2 其中每一米树干就包含有两代家族人物的脸谱。

不同类型的植物会在面孔表面上生长。研究人员推测，这些植物代表了每个家族成员的个性。例如，玫瑰通常在女性个体的脸部附近开花，橡树橡子在对哲学感兴趣的家庭成员旁边生长，松树与苹果在从事体力劳动与军事业务的人旁边生长。尽管存在所有这些异常现象，树的主干组成部分仍然是柏树。

在完成对面部的分析之后，现场特工在树上找到了最高的面孔：Cyparisse的面孔，据传说，此柏树由太阳神阿波罗所造。3

还观察到了有剪裁修改迹象，例如凸起，裂缝或[数据删除]。当第20米处的三个分支的纠缠于一起时，研究人员发现该面部对应于██████侯爵和夫人███████公爵的儿子██，被父母杀害。他们推断出2个分枝代表父母，小枝是████，其儿子。他们的结论是，此类变化代表了家庭成员的非自然死亡。

+ 需要2级授权
- [允许接入]

科学人员随后编制了这份清单，其中列出了每项变化的已知含义。:

► 自然死亡原因:

  • 撞击痕迹 → 窒息致死（溺水……）
  • 变黑 → 葬身火场
  • 长出霉菌 → 中毒死亡
  • 青蓝色青苔 → 被击打致死

► 死于武器 :

  • 缺口 → 锋利的武器（刀，匕首……）致死
  • 凹槽 → 钝器（棍棒……）致死
  • 孔洞 → 枪击致死

(面孔上的洞对应当时将其击毙的武器的口径)

  • 红色青苔→ 电击致死

► 其他：

  • 分枝纠缠 → 家族内谋杀

请注意，如果死因是谋杀，相应的变化伴随着固定形状的** Y **分枝。该符号可能代表了哈迪斯（地狱之神）的权杖。
可能同时出现另外数种变化：纠缠并隐藏意味着受害者被其家庭成员窒息。

然而，一些分枝仍然未能被解读，包括一些树上的原始部分。这些人很可能是失踪者，但这些信息仍有待确认。

如果分枝断裂，断裂部分所属成员的面孔将会表现出患上严重的精神疾病的样貌，包括发展为急性形式的偏执狂。

在采访中，家庭成员表示，他们一直知道这棵树存在，他们认为它是一个家族墓地，放置鲜花和“瞻仰祖先的面孔”。

附录： 以下文件是19 /██/20██进行的访谈报告。

访谈对象： █████████与其夫人
访谈人： Dr ███████

前言： SCP-202-FR旁所属家族的成员接受了███████博士的采访，了解SCP-202-FR的一些谜团，秘密或故事。
需要口译员，因为该家庭只讲希腊语。

Dr ██ : 你好，女士，你好，先生。 别担心，这次采访会很简短。 你能告诉我什么关于SCP……这棵树的吗？

M. ███ : 我们的树一直在那里，无论是我父母的一代，还是祖父，曾祖父……等等。

Dr ██ : 能具体点吗？

Mme. ███ : 好……（互相看着对方）……没关系。当我们小的时候，我们的父母告诉我们，这棵树是我们祖先生活的地方。后来，在青年期时，我们再也无法相信这一点，所以他们告诉我们，我们最古老的祖先是塞浦路斯人……是的，是的，由阿波罗改造的那个！

Dr ██ : 你还相信这个故事吗？

Mme. ███ : 我当然相信！是塞浦路斯人以分支的形式代表他的后代，我们也是！所以，我们每周去一次那里。这东西甚至比墓地更好，因为我们所有的祖先都被铭刻在一个地方，没有人打扰我们！

Dr ██ : 但你从没想过要打电话给专家或媒体？

M. ███ : 不！为了什么？我们非常好。无论如何，是你的话你会怎么解释？唯一真实的解释就是这一个！哪怕就是一次，能不能放弃你的科学信仰，接受神圣的存在！

Dr ██ : 冷静下来，你没有别的话要对我说吗？

Mme. ███ : 如果…（犹豫不决）说实话，我们有时会在这些地方听到一些声音……非常清楚，它们不是幻觉，我们都知道……我，我的丈夫，孩子们！它们很少出现，但我的印象是听到了我母亲和父亲的声音……

Dr ██ : 你还能告诉我别的什么……吗？

M. ███ : 不，你看，这对我的妻子来说很艰难……

Dr ██ : 在这种情况下，我认为我们可以结束这次讨论，我掌握到了一些有趣的东西，我们将继续我们的研究。 女士和█████████先生，谢谢。

结语： 随后，工作人员试图核实█████████先生和夫人的言论，但未获得任何结果。 他们认为所有家族成员是产生有幻觉的。除了其家庭之外，研究人员也没有找到任何解释。这个现象需要进行广泛的研究，以找到可能的解释。

2. 树冠即对项目起到了叶子的作用
3. Cyparisse，神话人物。奥维德在阿波罗在杀死他的雄鹿后变成了柏树。(Lien Wikipédia)
