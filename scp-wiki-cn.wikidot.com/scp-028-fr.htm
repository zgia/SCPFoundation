SCP-028-FR

于医院走廊发现的照片，背景不明。

项目编号： SCP-028-FR

威胁等级： 橙 ●

项目等级： Safe

特殊收容措施： SCP-028-FR位于█████东北郊的一家废弃医院旁。由于SCP-028-FR内部有大量人员，所以须在医院入口处将设置一个可以容纳15名安保人员的安全站。尽管目前为止未有其中的表现出逃离的尝试，但必须安装滚动带以防万一。此外安保人员应尽量避免在SCP-028-FR-4离开时与SCP-028-FR-4发生任何潜在的接触。还必须安装一个圆形围栏以拦截潜在的入侵者。在实验过程中，安全人员和研究人员必须配备特殊的防化服，以防止意外感染。1并且必须通过一系列测试以确定SCP-028-FR-4传播的疾病。如发现未知疾病，应采集样本并尽快消灭受害者。

描述： SCP-028-FR是一个由6排组成的圆形小剧院，可通过位于SCP-028-FR中心和两侧的楼梯进入。实验受害者只能从中央楼梯下方的唯一的出口离开。1██名身份不明的人永久存在于SCP-028-FR中（不包括SCP-028-FR-1，SCP-028-2，SCP-028-3和SCP-028-4），他们似乎不需要进食与休息。存在于SCP-028-FR中的所有实体均拥有与50-60岁相对应的着装风格。医疗设备通常放在圆形剧院下方，但直到现今才开始使用。当受害者进入SCP-028-FR时，SCP-028-FR-1邀请他/她坐在讲堂中心的凳子上，同时SCP-028-FR-2开始针对受害者的过去的罪行开始审判，对这个问题加以辩论。一旦辩论结束，圆形剧场的1██名成员通过举手投票选择施加在受害者体内的疾病。 SCP-028-FR-1，-2，-3和-4与其他SCP-028-FR成员有不同的角色：

  • SCP-028-FR-1是一个约25岁的年轻人，戴着眼镜，着装随意。其似乎相当于辩护律师的角色，通过要求轻微的疾病作为惩罚来捍卫实验受害者。
  • SCP-028-FR-2是一个年纪稍大的男人，35岁左右，其态度较为冷淡，经常双臂交叉。似乎相当于检方的律师，通过要求以一个严重且经常致命的疾病惩罚该实验受害者。
  • SCP-028-FR-3是一名约35岁的女护士，其经常抱着中立态度。通过不为人知的方式，其能够充分了受害者的医疗和司法历史，并回答SCP-028-FR-1和SCP-028-FR-2的问题。
  • SCP-028-FR-4是一个约20岁的年轻人，穿着黑衣服，不像SCP-028-FR-1，-2和-3。其经常报以极度冷漠的态度，总是站在出口前，以防止受害者逃脱。 只有在他离开SCP-028-FR之前触摸受试者才能传播决定的疾病。

实验-01

实验对象： D-7842
备注 : 自被监禁以来十分配合
犯罪记录： 谋杀██人。
设备 : 视频和音频记录设备
实验目的： 观察一场典型的项目对话。
会议进程： 进入SCP-028-FR的D-7842被邀请坐在剧场中心的凳子上。SCP-028-FR-4站立于出口前。
SCP-028-FR-1 : 好吧，会议开始。
SCP-028-FR-2 : Huguette，被告的犯罪记录，请。
SCP-028-FR-3 : 起立说话。他是谋杀了██人的凶手，他犯了轻罪。
SCP-028-FR-1 : 足够了，Huguette，谢谢你。
SCP-028-FR-2 : 嗯，这对我来说还不够，请继续。
SCP-028-FR-3 : 还有两次交通盗窃，八次偷窃，四次商业欺诈。
SCP-028-FR-2 : 谢谢，你可以坐下了。
SCP-028-FR-1 : 我建议施以甲型肝炎。
SCP-028-FR-2 : 你在开玩笑吗，我建议？他杀了多少人你没看见吗？这名男子至少应该患上肺癌。
SCP-028-FR-1 : 说了多少次了，Rouget，你就不能尊重我一次吗？
SCP-028-FR-2 : 你的意见不仅值得怀疑，而且你就像那名罪人一样邪恶！
SCP-028-FR-1 : 击打桌子。 Rouget，你再这么撒野我就尽我所能把你给炒了！
SCP-028-FR-2 : 小声说话。 我们进行投票吧。
SCP-028-FR-1 : 态度转变挺快，我说服你了吗？
SCP-028-FR-2 : 不，没有，倒是看看你说服下面多少人吧。
SCP-028-FR-1 : Rouget.
██ 人举手
SCP-028-FR-2 : Dupont.
██ 人举手。
SCP-028-FR-1 : 那么， Pasteur，请下决定吧。
SCP-028-FR-4前进到D-7842并与其接触。D-7842起身离开SCP-028-FR。
病原体获取： 经分析D-7842患上了肺癌。
Dr ████的结论：根据我们的观察，SCP-028-FR-1和SCP-028-FR-2存在不和。 需要进一步实验以更好地了解SCP-028-FR-3和SCP-028-FR-4的能力。

实验-03
实验-03

实验对象 D-7850
备注： 已患上肝癌。
犯罪记录： 4起强奸，其中█名受害者轻伤。
携带设备： 视频和音频记录设备。
实验目的： 观察已患有严重疾病的实验对象的结果。
会议进程： 进入SCP-028-FR的D-7850被邀请坐在圆形剧场中心的凳子上。SCP-028-FR-4如常。
SCP-028-FR-1 : 好的，会议开始。
SCP-028-FR-2 : Huguette，被告的犯罪记录，请。
SCP-028-FR-1 : Huguette ? 这可不妙 ?
SCP-028-FR-3看起来十分犹豫，并用忧郁的眼神看向其脚部。
SCP-028-FR-2 : Huguette，你可别说……
SCP-028-FR-3 : 他娈童……是的。
SCP-028-FR-2 : 叹息并对SCP-027-FR-1说。 M.Dupont，你想……
SCP-028-FR-1 : 以鄙视的眼神看向D-7850。 不不不……这没关系。
SCP-028-FR-2 : 你的提议是……？
SCP-028-FR-1 : 看起来十分生气。 随意，你想咋样就咋样。
SCP-028-FR-2 : 我提议，一种癌症，最好是大脑，心脏或者肝脏。
SCP-028-FR-3 : 他已经患上肝癌了。
SCP-028-FR-2 : 这没有大碍，Pasteur先生，让肝癌加重就好。
SCP-028-FR-1 : 为什么不加多一个？
SCP-028-FR-2 : 癌症吗？好吧，心脏如何？
SCP-028-FR-1 : 我觉得睾丸不错。
SCP-028-FR-2 : 如果这能平息你心中的怒火。转向观众席 先生们，这次我们不举行投票。

SCP-028-FR-4前进至D-7850，太阳穴处接触后并让其离开SCP-028-FR。

Dr ████的结论：在运送D-7850时我们把他麻醉了，有些人告诉我他到最后才会保持安静。

实验-04

实验对象 D-7891
备注： 前医生
犯罪记录： 弑子
携带设备： 视频和音频记录设备。
实验目的：当被告是医生时，试探是否存在差异。

D-7891 : 这不是真的！这是一个可怕的错误……
SCP-028-FR-4 : 正义永远不会错。
D-7891立即沉默，被SCP-028-FR-4的声音吓住。
SCP-028-FR-1 : 我建议施与他永久的失聪与失明。
SCP-028-FR-2 : 从常识来看，这就够了，但我觉得还是太便宜你了。
SCP-028-FR-1 : 你有什么建议能不让他一直错下去？
SCP-028-FR-2 : 心脏癌。
SCP-028-FR-1 : 一般的癌症？你就只知道用癌症了吗？
SCP-028-FR-2 : 行吧，如果你想要更多样化，我们可以施与他永远的便秘。
D-7891 : 啥？
SCP-028-FR-2 : 我不记得允许你发言了。
D-7891 : 我…我发誓我什么也没做，我可以发誓……
SCP-028-FR-2 : Pasteur说得还不够清楚吗？
D-7891开始哭泣。
SCP-028-FR-1 : Rouget.
██ 人举手。

██ 人举手。

实验-05

SCP-028-FR-2 : 注意礼仪，Pasteur先生。
SCP-028-FR-4 : 站起来站在出口前。 小的失礼。

SCP-028-FR-2 : Huguette，被告的犯罪记录，请。

SCP-028-FR-2 : Huguette，谢谢，请坐。
SCP-028-FR-1 : 我建议赋予其支气管炎两个月。
SCP-028-FR-2 : 四个月
SCP-028-FR-1 : 两个月
SCP-028-FR-2 : 四。
SCP-028-FR-1 : 两。
SCP-028-FR-2 : 没完了是吧？
SCP-028-FR-1 : 没完。我们还是投票吧。
SCP-028-FR-2 : Dupont先生
██ 人举手。
SCP-028-FR-1 : Rouget先生
██ 人举手。
D-7912的狗开始骚动，其主人拉着皮带使其沉默。
SCP-028-FR-2 : 让那狗闭嘴！
D-7912 : 好……
SCP-028-FR-2 : 我没有问你。
SCP-028-FR-1 : Pasteur先生，
SCP-028-FR-4前进到D-7912，接触并让其离开。D-7912拉起他的狗，但狗更为暴躁。

Dr ████的结论： 显然，它们与动物没有太大关系。
我更清楚为什么这些白痴在人群中像鲸鱼一样笑。

实验对象 Dr.████
备注： 已知没有犯罪记录
携带设备： 视频和音频记录设备。
实验目的： 了解哪种类型的疾病将传染给犯罪记录很少或没有犯罪记录的个人。
会议进程： 进入SCP-028-FR的████博士被邀请坐在圆形剧场中心的凳子上。 SCP-028-FR-4站在出口前。
SCP-028-FR-1 : 好的，开始会议。
SCP-028-FR-2 : Huguette，被告的犯罪记录，请。
SCP-028-FR-3 : 沉默。
SCP-028-FR-2 : 等待许久。 Huguette ?
SCP-028-FR-3 : …
