SCP-016-IT

光盘盒中的SCP-016-IT。

项目编号： SCP-016-IT

项目等级： Safe

特殊收容措施： 非实验状态下的SCP-016-IT需被收容在Site Vulcano的42号储藏单元中的一个玻璃匣中。需请求Dr. Montorsi的授权后方可利用其进行进一步实验。

描述： SCP-016-IT外观类似一张待录入的空白DVD，没有任何可辨识的品牌或型号信息。在发现时，其被装在一个透明硬质塑料制光盘盒中，在光盘盒背面用胶带贴有一张纸条，其上写有“写下来再玩:)”字样。

如果将SCP-016-IT塞入一台可以读取DVD的仪器（例如一台电脑），该仪器将无法探测到DVD的存在，从而无法识别项目。多次实验后，研究人员发现如果在将SCP-016-IT插入仪器之前在其表面上写下文字，其便可被仪器识别并能以类似正常DVD的方式被播放。

现在算是知道了碟盒背后那张条子说的是啥意思了。——Dr. Montorsi

导出的项目（被编号为SCP-016-IT-01）会随SCP-016-IT上书写内容发生变化，但是总是以一个题目与DVD上书写文字一致的电子游戏的形式呈现（更多信息参见附录02）。若将SCP-016-IT移出读取仪器，SCP-016-IT-01即会中止，而读取仪器会回到初始界面。此外，SCP-016-IT上所书写的文字也会消失。

每当尝试物理复制SCP-016-IT时，无论有无书写文字，最后都总会得到一张无异常性质的空白DVD。当尝试数字拷贝SCP-016-IT-01时，情况类似，这时会得到一类可执行文件，但当启动时，该类文件不会产生任何结果。

此外，研究人员还试图通过外部透明的DVD读取设备来确定SCP-016-IT上文字消失的途经。首次观测中发现这些文字在DVD刚开始旋转时似乎即开始消失。在观察慢动作录像时，可发现DVD每转一圈，文字即褪色一些，并在这一进程开始后几秒内即呈现为与DVD背景色一致的颜色。尝试使用不褪色墨水后，发现无法得到值得注意的结果。

附录01： SCP-016-IT回收记录
Addendum 01: Appunti sul ritrovamento di SCP-016-IT.

SCP-016-IT由MTF-I“金色名流”在对████████的总部进行搜查时发现，后者是一家小型独立游戏公司，在成立几个月后即消失。

该公司在消失前不久发布了一则关于一款即将面世的革命性电子游戏的广告，而在宣布这一消息后，其旋即就宣告消失，这引起了潜伏在当地的基金会特工的怀疑，并使得他们建议对该公司进行一次搜查，因此基金会才决定进行这一搜查任务。

对████████办公室的搜查显示所有雇员均神秘失踪。在董事长办公室发现了一个极其精致的保险箱，其箱型通常被用于高安全度金库中。专家们花了数小时成功将其撬开，仅在其中发现了SCP-016-IT，并出于科研目的将其回收。

在搜查完成后，基金会的技术人员利用该公司的通讯渠道，宣称该游戏公司因财务原因破产。进一步研究尚在进行中。

附录02： 下列实验中的“标题”指SCP-016-IT上书写的词句
+ 部分实验记录
- 部分实验记录

Test 016-01 - ██/██/2016

标题： 实验
读取设备： 电脑1
结果： 得到的游戏主要内容为一系列基于常识的多选问题。
分析： SCP-016-IT会启动一个电子项目，严格来说会启动一个电子游戏项目，其内容与DVD上书写文字相一致。

Test 016-05 - ██/██/2016

标题： Pong
读取设备： 电脑
结果： 得到的游戏与1972年的游戏“Pong”完全一致。
分析： SCP-016-IT似乎包含有某种包含已存在游戏的数据库。

目前除了SCP-016-IT-01的启动程序外，我们还无法打开任何其他SCP-016-IT内部的文件。—— Dr. Montorsi

Test 016-12 - ██/██/2016

标题： 超级马里奥
读取设备： 电脑
结果： 得到一款主角是一名叫“超级马里奥”的超级英雄的第三人称动作游戏。
分析： SCP-016-IT-01并不总是已存在游戏的复制版本，据信所给题目的解读方式更可能会更直白一些。

Test 016-23 - ██/██/2016

标题： 波音747
读取设备： 电脑
结果： 得到一款波音747型飞机驾驶模拟器
分析： 打算书写相同文字并重复实验，以查证SCP-016-IT-01是否呈现出不同形式。

Test 016-24 - ██/██/2016

标题： 波音747
读取设备： 电脑
结果： 得到一款与前一次实验完全相同的游戏。
分析： 多次插入时，若书写文字相同，SCP-016-IT-01并不会呈现出不同游戏。

不如试试电脑以外的东西？ —— Dr. Rivoli

Test 016-35 - ██/██/2016

标题： 战争
读取设备： Xbox家用游戏主机
结果： 得到一款时间地点均架空的第一人称射击游戏。
分析： SCP-016-IT似乎可以在任何能读取DVD的设备上运行。

Test 016-36 - ██/██/2016

标题： 战争
读取设备： Xbox360家用游戏主机
结果： 得到一款与前一次试验一致的游戏，但值得注意的是，其画质与游戏体验都有微小提升。
分析： 显然SCP-016-IT可以随其插入的设备对自己进行调整以提升其表现。

Test 016-43 - ██/██/2017

标题： Tsgevrjsocueh
读取设备： 电脑
结果： 初始桌面上出现黑屏，中间有一行白色的“Error”。
分析： SCP-016-IT-01首次报错。

项目解读范围可能存在限度。我们可以进一步进行类似实验来确定这一假说。 —— Dr. Montorsi

Test 016-64 - ██/██/2017

标题： SCP-682
读取设备： PlayStation3家用游戏主机
结果： 得到一款玩家扮演一只巨大的蜥蜴且游戏目标为尽可能破坏的游戏。
分析： 尽管是以一种模糊的方式，但似乎SCP-016-IT还是了解一些机密信息。禁止在无必要授权时提供机密文件相关的标题。

Test 016-117 - ██/██/2017

标题： SCP基金会
读取设备： 电脑
结果： 得到一款管理游戏，其控制一个架空的跨国SCP基金会，该基金会与现实基金会毫无关系，且游戏目标是尽可能多赚钱。
分析： 旨在确定SCP-016-IT内关于基金会信息限度的受控实验。结果令人欣慰。

Test 016-117 - ██/██/2017 （原始文档）

标题： SCP基金会
读取设备： 电脑
结果： 得到一款设定在近未来的管理游戏，游戏内容为控制SCP基金会在全世界的零星站点，游戏目标为收容多种异常。
分析： 该款SCP-016-IT中出现的大部分异常都与基金会各站点真实收容的异常极其相似。此外还存在多个未知异常，其中之一，一个Keter级人形SCP项目，为该款游戏的主要主人公，且其收容措施当前仍未发现。SCP-016-IT-01内的基金会与现实中的基金会在有几点上并不相同：O5议会被唯一的一个监督者所取代，且除了Safe级以外的所有SCP项目均可以考虑对其进行处决。此外，在游戏过程中，该款游戏还多次暗示，若不对SCP项目进行清理，那么就不可避免地会导致最后的失败。

SCP基金会意大利分部主管下达了进一步研究的通知，需要对这个SCP-016-IT-01版本进一步研究以确认是否真的存在或有人创造了某种未知异常。此外，我还建议我们可以利用我们本次实验中获得的这款游戏作为收容措施研究的“试验田”，没准能提供一些有趣的线索呢。而实验日志的话，姑且还是插入一段假消息吧。 —— Dr. Montorsi

Footnotes
1. 被使用的电脑是3号实验室的14号工作站。技术细节参见实验室文件。
