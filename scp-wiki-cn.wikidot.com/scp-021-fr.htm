SCP-021-FR

项目编号： SCP-021-FR

威胁等级： 绿 ●

项目等级： Euclid

特殊收容措施：由于SCP-021-FR并不遵循常规物理规律，将其收容在短期内不是可行的。因此，收容程序的重点是减少SCP-021-FR个体的出现，并对公众隐藏其存在。

机动特遣队Phi-61“悬崖灯笼”各部，必须每周检查一次以下所有地点的悬崖:

  • 拉特拉波耶格悬崖（爱尔兰）。
  • 莫赫悬崖（爱尔兰）。
  • 埃特勒塔悬崖（法国）。
  • 北角崖（挪威）。

任何突出超过30厘米的空隙，位于其中一个悬崖的边缘并且可从其顶部进入的石块，必须进行编目并定期检查，以记录与SCP-021-FR相关的任何事件。 她会成为震中。

任何此类可由公众触及的石块必须从地面移除并送往Site-61进行分析。

必须尽快确定任何与SCP-021-FR直接相关的事件。这些事件的任何异常因素都必须向公众隐瞒。

描述： SCP-021-FR指的是在一些欧洲悬崖顶部以不规则的间隔时间发生的现象，位于其震中的凸起的岩石上，悬崖边缘的平衡处，可以从山顶步行到达。但请注意，只有符合标准的部分岩石可能成为SCP-021-FR现象的中心。

当满足此类标准的岩石将会影响在夜间或黄昏时位于悬崖顶部附近的人类（以下称为“受影响者”）。在其视野范围内时，SCP-021-FR的异常性质将被触发，受影响者感知到的区域与其实际所处位置不一致（尤其是悬崖的位置）。1 受影响者还将感觉到存在可变外观的实体，携带光源对其进行引导（以下称为SCP-021-FR-1）。该实体以及SCP-021-FR的影响过程对于未受影响者来说是不可见的，但可以出现在受影响者制作的任何记录，照片或视频上。

悬崖边缘的虚假外观与对受影响的对象的实体的存在的组合会驱使其靠近，导致其越过悬崖的真实边缘时跌落。因坠崖而死亡的受影响者没有异常。

现象 021-41 du ██/██/██ 21h31 (埃特勒塔悬崖)

前言： 这是SCP-021-FR的第一个记录到的现象，其影响了可能成为现象震中的一名石控基金会工作人员。

<开始报告>

(删除无关对话)

Dr. Ryan : 打包上路，今晚这件事搞定了，孩子们。

研究员 Strauss : 我以为我们已经完成了传感器的移除，等等，为什么石头上有人？

Dr. Ryan : 这边没人啊。

研究员 Koch : 等一下，Strauss，你看到了吗？

研究员 Strauss : 肯定有个人在那里，正在用手电筒在地上找东西。可能是凯恩斯。 （大喊）凯恩斯，你在做什么？收拾收拾就走了。

研究员 Koch : 施特劳斯，凯恩斯和我们在一起。悬崖附近没有灯光。您已经受到了SCP-021-FR的影响。你有相机吗？

研究员 Strauss : 我把它放在了我的车里。这是我的习惯。

Dr. Ryan : 红外探测仪没有检测到任何东西。不要再动了，准确描述你所看到的东西。

研究员 Strauss : 有一个高大的人影，身高约190，离我约30米远。他手里拿着一些明亮的东西，像手电筒的光束一样扫过地面。不知道他穿的是什么，但它看起来像湿k路，它在灯的光线下发光。 （暂停）他绕着悬崖在走动，用灯照着地面。我想他在寻找一些东西。

Dr. Ryan : 绕着悬崖在走？在空中？

研究员 Strauss : 应该是吧，周边地质不同是不同的。还有更多的石头……我看不太清楚，但看起来悬崖的边缘已经消失了。我凑近点看看。

Dr. Ryan : Strauss，我告诉你别动。

研究员 Strauss : 只要我远离那块石头，我就不会有任何风险。（他点亮自身的手电筒来准备接近）

Dr. Ryan : 你看到的人的位置可能与真实的位置不完全相同。不要冒险。

研究员 Strauss : （停下来）你是对的。（暂停）对不起

Dr. Ryan : 我啥也没说。

研究员 Strauss : 不是你 （他显然正在与该实体对话）你在说什么？ （长时间停顿）对不起，我得走了，你得自己解决了。

(在相当长一段时间内，施特劳斯似乎多次犹豫不决。其最终在8分钟后归队，但似乎对他的选择感到后悔。)

研究员 Strauss : 他说他丢了他给他未婚妻在草地上的戒指，他希望我用手电筒帮忙找到它。

<报告结束>

现象 021-55 ██/██/██ 23h04 (北角悬崖)

前言： 所有在场的队员都配备了录音机，以检查是否可以录制SCP-021-FR-1的声音。 SCP-021-FR于四昼夜的监视之后出现。这份报告出自Nilsen博士录音机的录音，该录音机受到了项目的影响。

<开始报告>

Dr. Nilsen : 我觉得出事了。

研究员 Strand : 红外无异常。

Dr. Nilsen : 别傻了。如果我受到影响，那么它就不会干扰那些设备。如果我离悬崖边缘太近，请立刻警告我。

(Dr. Nilsen下车，并朝异常方向前进)

Dr. Nilsen : 有人正穿着一件非常明亮的衣服，看起来像一个旧的蜡烛灯笼。那个实体很小，正在转过身来，从左到右缓慢地挥动灯笼。

研究员 Strand : 根据红外探测，你距离真正的悬崖边缘有四十多米。

Dr. Nilsen : 该实体位于稍远的位置。我越走越近了。

(无线电静默数秒)

Dr. Nilsen : 那里的地面似乎覆盖着浮木。我好奇知道他是如何到达那里的。

(无线电沉默约二十秒，然后听到一个模糊不清的孩子的声音。 这种声音仅由录音机录制。 根据尼尔森博士的说法，这是一个“美好的夜晚”。)

Dr. Nilsen : 晚上好。 你叫什么名字？

SCP-021-FR-1 : Mayken Jørgensen，夫人。

Dr. Nilsen : Mayken，你一个人在这做什么？

SCP-021-FR-1 : 我在对我的船说晚安，夫人。

Dr. Nilsen : 真的吗？哪艘船？

SCP-021-FR-1 : 夜晚的船只。我用我的灯灯给他们说晚安。你想和他们说晚安吗？

Dr. Nilsen : 我宁愿待在原地，Mayken.

SCP-021-FR-1 : 如果你现在告诉他们晚安，他们将看不到你的灯。

Dr. Nilsen : （与Strand交谈）这个实体看起来像一个大约八岁的小女孩。她穿着破旧的衣服，无法准确估计，但似乎可以追溯到十三世纪或十五世纪。她的头上系着一条白色围巾。

SCP-021-FR-1 : 如果你在离开之前对晚上的船只说晚安，那就太好了，夫人。那会很好。 我们不时向他们说晚安，当他们消失时，他们会在下面的海滩上留下一些小木头，还有各种漂亮的东西可以拿起来。爸爸说你要对夜晚的船只礼貌。

Dr. Nilsen : 他没有告诉你大晚上停留在悬崖的边缘是危险的吗？他让你做到了吗？

SCP-021-FR-1 : 我不害怕，夫人，因为您与我同在。

<报告结束>

现象 021-62 du ██/██/██ à 04h50 (莫赫悬崖)

前言： 从技术角度来看，这是SCP-021-FR最清晰的记录案例。研究员Asling制作的视频录像和团队制作的视频录像可在Site-61的档案中找到。
作为SCP-021-FR现象震中的石头不再被研究员Asling所察觉。另一方面，此实体似乎居住在悬崖边上的一个小房子，用粗糙的石头建造。已经将安全线确定至能够足够接近实体而杜绝坠崖的危险。

<开始报告>

研究员 Asling : 这个地方看起来非常不同。我看到几棵早先时候没有的树。悬崖边上还有一个石头屋。我看到门下有微弱的移动光源。真心希望所有这些都会出现在视频记录中。

研究员 Teagan : 场地安全，你可以前进。

研究员 Asling : 天气突然变了，正在下雨，你那边怎么样？

研究员 Teagan : 什么事也没有。

研究员 Asling : 有趣的。 我会尝试——（突然停止）实体出现了。

研究员 Teagan : 请描述。

研究员 Asling : 这是一个驼着背的男人，身高约六英尺，戴着宽边帽和一种破旧的斗篷。 他在向我招手。

(作为SCP-021-FR现象震中的石头不再被研究员Asling所察觉。另一方面，该实体居住于悬崖边上的一幢用粗糙的石头建造的小屋。已经将安全线确定至能够足够接近实体而杜绝坠崖的危险。)

SCP-021-FR-1 : 在这样的天气里你在外面做什么？你湿透了，伙计。进屋暖暖身子吧。

研究员 Asling : 五分钟前，这所房子还不在那里。表明你的身份。

SCP-021-FR-1 : 我是Meallán，这个房子是我的小家，我祖父的房子。这外面一直在下雨，我的孩子。11月的降雨是最糟糕的。我去暖个炉子，你进去暖暖身子吧。

(研究员Asling进入房子并立即从团队拍摄的红外视频中消失。)

研究员 Teagan : Asling, 你还在吗？我看不见你了。

研究员 Asling : 我进了他的房子

SCP-021-FR-1 : 当然，你终于进来了。哦，看哪，你开始发烧了。坐在那里的席子上。火将对你有好处。

(研究员Asling的视频记录下证明其处于一个非常破旧的房子内，只有一个房间。一层稻草似乎充当了西墙的床。这个房间有两个大致雕刻出来的石凳，位于中央火炉的两侧。研究员Asling坐在靠近门口的长凳上。一堆木头在对面的长凳后面。研究员Asling将摄像机拍向房间的天花板；其由各类钩子以及粗糙织物所。这些挂钩上挂着小物品，如洋葱，稻草，铁锅，各种动物爪子，干浆果和各种磨砂玻璃瓶，似乎内装液体。SCP-021-FR-1在火中放置了干柴，坐在研究员Asling对面并仔细观察。)

SCP-021-FR-1 : 你不是本地人，对吧？

研究员 Asling : 显然不是。

(SCP-021-FR-1研究员Asling交谈，记录仪器静默40分钟，亮度开始逐渐下降。)

SCP-021-FR-1 : 你应该向篝火中投入干柴。

研究员 Asling : 我不喜欢这样。

SCP-021-FR-1 : 它会消失的。

研究员 Asling : 无论如何，这简直度日如年。

(亮度继续下降。SCP-021-FR-1继续与研究员Asling交谈，持续几个小时。亮点终于消失。无线电静默三十分钟。)

研究员 Teagan : Asling ? 一切还好吗？我们仍然没有在屏幕上看到你。你应该回来。

研究员 Asling : 还没有。我宁愿等到现象结束。这样我们就可以准确地知道我与当前地形拓扑的关系。

(太阳在7:50左右升起并导致现象结束。研究员Asling重新出现在屏幕上，坐在作为SCP-021-FR的现象震中的石头上。)

<报告结束>
