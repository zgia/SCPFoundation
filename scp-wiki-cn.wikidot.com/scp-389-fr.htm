SCP-389-FR

由D级工作人员拍摄的SCP-389-FR图像。

项目编号：SCP-389-FR

威胁等级：黄 ●

项目等级：Euclid

特殊收容措施：SCP-389-FR必须收容在配备不透明锁的传统收容间内，以便完全无法从外部可见SCP-389-FR。

SCP-389-FR必须放置于一个配备有显示器，可以远程开关的不透明顶盖内。

如果顶盖发生故障或后者堵塞，在对收容间进行排查期间，必须向工作人员提供移动不透明板材。SCP-389-FR上的实验必须通过安装于房间内的相机进行，为确保研究人员的安全，绝不能在镜子前进行实验。如果需要对SCP-389-FR的收容间进行维护，则相关工具必须带夹具的移动机器人进入房间内，远程控制进行顶盖维护。可应要求提供由专家Arthur O'Lerry专门为此目的而设计的排爆机器人。

描述：SCP-389-FR是一个33cm高的塑料小雕像，没有铭牌与标记。项目外貌为死亡寓言的漫画形象，通常被称为“死神”，其中不成比例的头与弹簧连接以便在触摸时可摆动。雕像有一个可拆卸的左臂，肩部有一个可移动的铰接点。左手为模仿手枪的手势。物体放置在设有滚球的底座上，允许360°旋转。在雕像的背面，有2个AAA（1.5V）电池的插槽，与一个小型电子卡和一个压电高音扬声器相连接。当小雕像的头部离开其连接点时，其内部将可以播放骨折声音。

SCP-389-FR的效果是随机触发的，前提是至少有1个人存在于项目视觉范围内。项目将在不定的时刻毫无征兆地在顺时针方向上在其基部上旋转（驱动源迄今未知），以朝向在场的者。然后SCP-389-FR的右臂将向后急剧摆动以击中头部，使后者摆动。

然后，项目所指向者将立即死亡。目前为止，SCP-389-FR致死机理尚未确定。尸检未发现生物异常，除了死后（从几小时到几天）身体腐烂过程略有延迟。

SCP-389-FR的两个异常效应发动间隔并不固定，似乎没有遵循一个特定模型：发动间隔可以很长，也可以短至2,3秒。

该效果的出发似乎与手臂的折叠的机构的活动有关，而不是驱赶。头部的移动或电池的存在与否不会影响异常效果。

SCP-389-FR被发现隐藏于圣-███████医院206室的一个病房内，这是一个复苏室，不寻常的死亡人数引起了基金会关注。医院的监控摄像无法确定是谁将其安置在此。调查仍在进行中。

事件报告389-FR-01-T██

██/██/2017，12时21分, 在对在SCP-389-FR实验中丧生的D-2224进行尸检后，负责医师Dr Brenan被紧急送往在重症监护室，多处受伤。这些包括深刺伤和多处瘀伤，但手术期间没有人可以解释此类伤痕的来源。
18h12 : 监视摄像机显示Brennan博士在Fakih Najib，Eva Curset以及HervéDr Armot的陪同下进入尸检室。
18h45 : 特工Etienne Bonsaint回到太平间，然后前往12-A去的冷室。
19h11 : Dr Brenan独自离开了手术室。全身覆满血迹，最后倒在地上。他似乎试图通过缓慢的动作来阻止大出血。
21h03 : 特工Etienne Bonsaint回到原地，发现了Dr Brenan的血液。他在几分钟后带领救援队员前往营救。
为了确定受伤的原因，决定在事件期间分别隔离和审讯在停尸房中出现的所有人员。Dr Brenan因需要康复而一直处于人工昏迷状态，尚未接受采访。

+ 助理Fakih Najib的审讯
- 折叠审讯

审讯对象：助理Fakih Najib
审讯者：Dr Valentin Mazuret

前言：助理F. Najib由负责验尸的Brenan博士传唤，协助他完成尸检。

Dr Mazuret : 你好Najib先生，请坐，我们可以开始了。这次访谈的目的是了解D-2224尸检期间发生的事情。正如您可以想象的那样，它将被记录下来进行汇报。在开始之前，您有任何疑问吗？

F. Najib : 没有。

Dr Mazuret : 很好，那就让我们回到你们转运尸体的那时候，我希望你能给我们提供有用的信息。

F. Najib : 好。我和Eva Curset在休息室待着。Brenan博士冲进来，并要求我们陪他进行尸检，在错误的时间在错误的地方。

Dr Mazuret : 你没有被安排进行尸检吗？

F. Najib : 不，这活一般是他和Armot博士干的，我们两个只是凑巧在那天值班。

Dr Mazuret : 为什么有四个人进行尸检？这不是太多人了吗？

F. Najib : 我也是这么想的，但这是Brenan博士的命令….一个心神不宁的人。像往常一样在SCP-389-FR上进行实验——当然这是废话——我不知道他们是如何摆弄这个异常的，但每一次，我们都会发现自己的工作时间会比预计时间多。转运设备来迟了点，这对Brenan博士是一件坏事。

Dr Mazuret : 还有别的岔子？你们解决了吗？

F. Najib : 还好。这更像是周边环境，而不是其他任何事情导致的。实验发生在晚上，老Brenan博士非常疲惫，由他带领尸检。值得注意的是，那天，他开始有些粗暴地斥责我们而不是让我们使用转运设备。这些东西在这，他却又不敢亲自动手。除了这种见了鬼的现象外，尸体还在别处。目前为止还好，没必要那么急……但博士更像是着了魔。

Dr Mazuret : 很好，然后呢？

F. Najib : 我们去搬运尸体了。

Dr Mazuret : 去找尸体？那不是一开始就应该放在转运设备上的吗？

F. Najib : 是的，他们把尸体留在了太平间的门外。

Dr Mazuret : 这正常吗？

F. Najib : 不，但我们已经完成了搬运工作。通常，转运设备会将我们运送到相关的冷藏室

Dr Mazuret : 然后呢？

F. Najib : 我们去了尸检。Brenan博士正在对身体进行外部检查，Armot博士负责正在协助和做笔记。我们稳住尸体并传递工具。

Dr Mazuret : “稳住尸体”？

F. Najib : 是的。D-2224是个很大的胖子，像一个冰柜，尸检台有点小。继续吗？

Dr Mazuret : 继续。

F. Najib : 所以我们做了外部检查。在进行到一个时刻时，需要把尸体翻身，我们没接住，尸体从桌子上掉了下来。我们在Armot博士的帮助下把它抬了回去，然后继续。通常，房间被消毒了，所以没什么问题，但我们采取了一个猜测。简而言之，我们将身体放回原位，我们完成了外部检查并进行了穿刺。

Dr Mazuret : 穿刺？

F. Najib : 就是那种扎进体内的软式内窥镜，鉴于您不太了解那个术语，我觉得这可以略过了。我们继续吗？

Dr Mazuret : 我大概知道怎么回事，继续吧。

F. Najib : 我们从手臂开始。Brenan想用点力，他走过了桌子。随着家伙的重量，Brenan这边的受力越来越大，尸体再次翻转并滑落。正如俗语所云，祸不单行。他和其他的一些器械一起掉在地上了，我不知道她怎么做到的，但他几乎在房间地板上滑行了整个房间宽度的一半距离。Brenan，Eva，Armot博士和我将尸体放回尸检台上。这并不容易，因为那简直是个大胖子

Dr Mazuret : 两次，是不是已经够多了？

F. Najib : 尸体挺大的。已经有过一次时，我们已经觉得非常愚蠢了，这是非常罕见的，所以那里……最糟糕的是，当我们想要剖开胸腔时，它几乎再次滑落。但这一次，我们做得很好。在尸检中三次把尸体掉在地上，这简直可以印就辞职了！

Dr Mazuret : 然后？

F. Najib : Brenan博士表现出疲劳迹象，并要求外出。他脸色苍白，我们知道他过度劳累了，我们与Armot一并完成了尸检。一切顺利，当我们出去的时候，我们发现Brenan博士靠在一堵墙上流血，旁边是特工Bonsaint，他显然是在寻求帮助。

Dr Mazuret : 还有吗？

F. Najib : 没了

Dr Mazuret : 很好，感谢帮助。

+ 助理Eva Curset的审讯
- 折叠审讯

审讯对象: Assistante Eva Curset
审讯人： Dr Valentin Mazuret

前言：助理Eva Curset被负责验尸的Brenan博士传唤，以协助其完成事件。

Dr Mazuret : 你好，Curset夫人。拜托，坐下我们将要求您尝试了解D-2224尸检期间发生的情况。当然，这次采访将被记录下来，以便进行汇报。在开始之前，您有疑问吗？

E. Curset : emmmmm……没有。

Dr Mazuret : 我们开始吧。让我们回到Brenan博士来到休息室来接你的时候。你能告诉我更多吗？

E. Curset : 是的，当然。我们去找太平间以外的尸体。转运设备把他落了外面。本来是我，直到尸体进入太平间之前，我会拒绝签署收据，而Brenan博士却因此而训斥我和Najib。

Dr Mazuret : 为什么？

E. Curset : 因为他得靠某种方式发泄他的紧张感。他就是这样。

Dr Mazuret : 之后呢？

E. Curset : 我们开始尸检。一些琐事接二连三地发生。在尸体从台上掉下来之前我们还没有完成外部检查……这名D级人员很重，Brenan博士失手了。Brenan当时至少得和我们一样负这个责任。我们尽可能地把它放回去。幸运的是，Najib和我都是前运动员，否则我们永远都没法把他抬起来。然后我们用左臂开始进行穿刺，Brenan博士尽可能地在手臂施力，尸体再次翻倒。这简直糟透了。

Dr Mazuret : 然后？

E. Curset : Brenan全身心地投入到身体上，他担心尸体损坏了，因为它掉下来的时候撞上了桌子。然后我和Armot帮他把他放回桌子上。Fakih加入我们对尸体进行了拍摄。然后继续尸检。

Dr Mazuret : 继续。

E. Curset : 我…怎么说……“注意到”，D-2224手臂上的穿刺痕迹，这简直就是杀猪。然后Brenan又暴力地打开了胸腔，我以为我们会再次让尸体滑落。我们看的很清楚，这简直就是在杀猪。Brenan博士觉得累了要出去歇歇，剩下的就交给了Armot博士。我们都松了一口气。

Dr Mazuret : 然后呢？

E. Curset : 尸检时再也没有滑落。一切顺利。Armot甚至设法修复了一些尸体损伤。只有当我们离开手术室时，我们看到Brenan医生大量出血，靠在墙上。还好有那个负责维护太平间的军官……我不太记得他的名字了，接下来的事情就众所周知了。

Dr Mazuret : 很好，我认为这就足够了，还有什么要说的吗？

E. Curset : 我有个问题。

Dr Mazuret : 问吧。

E. Curset : 你难道不怀疑是我们其中的一个杀了他吗？

Dr Mazuret : 现阶段我不胡乱臆测任何事情，除非你有什么要说。

E. Curset : 三个人独自与受害者关在一起，他的出血没有任何人知道……但我发誓，我们什么也没做！Bren …… ——Brenan博士——与团队关系很紧张，但我们绝对不会那样做！我们不会谋杀同事，我们不是杀人狂！我不知道是什么造成的，但我发誓，我们什么也没做。一切都在照常进行！

Dr Mazuret : 无需担心，所有蛛丝马迹都会被发掘，真相迟早会浮出水面。明白？

E. Curset : 我不太……

Dr Mazuret : 而很不幸的是，我也不能告诉你更多了，有疑问吗？

E. Curset : 不。

Dr Mazuret : 非常好。但我还有最后一个问题：我注意到你进来时脚一瘸一拐的……

E. Curset : 哦，是吗？也许吧……可能之前没太注意。

Dr Mazuret : 什么时候？

E. Curset : 不知道。可能是睡姿不端正，也可能是抬着D-2224扭到的，我不太清楚。

Dr Mazuret : 谢谢你，Curset女士。

+ Hervé Armot博士的审讯
- 关闭审讯

审讯对象： Docteur Hervé Armot
审讯者： Dr Mazuret

前言： Hervé Armot博士在Brenan博士的监督下，对D-2224进行尸检，以了解SCP-389-FR的异常机理

Dr Mazuret : Armot博士。

Dr Armot : Mazuret博士，很高兴见到你。

Dr Mazuret : 你可以想象，我们D-2224的尸检中一起回来。当然，这次审讯将被记录在案。

Dr Armot : 当然，你想让我讲述哪个部分？

Dr Mazuret : 太平间那里。

Dr Armot : 好的，就是尸体没到达指定位置。

Dr Mazuret : 详细说说？

Dr Armot : 这是我们第一次对SCP-389-FR病人进行快速尸检。通常，在尸体最后几周准备转入太平间永久性储存前几天，我们会进行尸检。这些尸体存放在一个冷室里，一个是异常——在这种情况下是12-A——我们等待时间。在这里，负责389-FR实验的医生█████████明确要求进行快速尸检，看看我们是否还能看到更多东西。
当我们采取行动的那天，转运设备迟到了！嗯，事实上，这是根本无害的。但Brenan对每个人都尖叫着，就像着了魔一般。

Dr Mazuret : 那尸检呢？

Dr Armot : 我于是来到了太平间。我看到了一些不好的，非常值得怀疑的东西，但好在是个绒球！很高兴不用启动紧急反应措施。那么，谈到反应，我们有什么新消息吗？

Dr Mazuret : 已经压制了，没有进一步进化。

Dr Armot : 天啊……

Dr Mazuret : 谈谈尸检吧

Dr Armot : 是的，抱歉。我们得到了两个年轻人的协助。Eva Curset和Fakih Najib。那天遭遇不幸的两名助手。然而，四点钟，我们设法两次搬动尸体。那真的是个大胖子。Fakih比较壮硕与宽大。我知道我们已经在SCP-389-FR实体上发生了一些“逆转”的情况，可能是由于尸检时间较晚……在目前的匆忙中，我们被迫推迟几个小时……毫无疑问，Brenan希望加强这两个方面的收容措施。我不知道他是否害怕专业上的问题，但我认为他的神经太大条了。他像一个粗野的人一样摆弄着尸体，在打开手臂的那一刻……像屠夫在切肉。我非常理解为什么这两个助手无法把身体扶住。当他跌倒时，Brenan跑去检查实体的状态。Eva和我去帮助她，但没有Fakih的帮助，我们就完成不了接下来的工作。

Dr Mazuret : 然后？

Dr Armot : 耻辱的典范：在给胸腔开口的那一刻，他的双手颤抖着，在切割的那一刻……把胸腔组织搅得稀烂。幸好在这个职业中，每个人有一个强壮的胃。他的脸像毛巾一样白。他试图继续，但他发现我们看见的手再次颤抖。他谎称疲劳，离开了房间。在尸体解剖结束时，我们终于找到了Brenan博士，还有特工Etienne Bonsaint，他给予了帮助。在太平间，我们一无所获。我不知道他在那里待了多久，我完成尸检的时间。也许两个小时。他失去了很多血。我们已经看到他仍然设法阻止出血，他还保有着作为医生的素养。当时，我想到了可能是收容失效，但我们没有听到警报。

Dr Mazuret : 你真的什么也没看见没听见吗？

Dr Armot : 不。我们专注于尸检，这些门隔音效果很好。但是，如果确实有的话，我想我们会听到一声尖叫，或类似的声音。

Dr Mazuret : 那尸检本身呢？

Dr Armot : 没有异常。在Brenan离开之后，我拿起了我的手术刀，毫无疑问是在尸检期间掉在地上的。我们取出血液，消毒，然后继续工作了。我有一些轻微的瘀伤，那时还挺希望跟守卫借一些肌肉过来，如果可以的话……

Dr Mazuret : 非常好。还有什么要补充的？

Dr Armot : 不。我们没有看到任何东西，尸检一塌糊涂。没什么好说的。也许在手术前他就已经受伤了。这可以解释很多事情。如果是你，你应该会在这之前看看他在做些什么。

Dr Mazuret : 谢谢你博士。最后一个问题：你脸上的伤疤是什么？

[Armot博士摩擦他的脸，看起来很惊讶，感觉像是切割伤]

Dr Armot : 不知道……也许是不小心刮到的

Dr Mazuret : 谢谢。

+ 特工Etienne Bonsaint的审讯
- 关闭审讯

审讯对象： 特工Etienne Bonsaint，技术员
审讯人： Dr Mazuret

前言： 特工Etienne Bonsaint不得不在太平间进行维护操作，发现Brenan博士正在失血。

Dr Mazuret : 你好，Bosaint先生，请坐。

E. Bonsaint : 谢谢。

Dr Mazuret : 我们在这里讨论太平间事件并了解发生的事情。根据协议，此次审讯将记录在案。你有什么问题吗？

E. Bonsaint : 没有。

Dr Mazuret : 好吧，那就让我们开始吧。你能告诉我们你是如何找到Brenan博士的吗？

Etienne Bonsaint : 当然。我去了站点的太平间进行维护操作。室内状况不佳。损坏的室内装潢，生锈的门……我到了，我开始了工作。确实有很多工作……拆卸和更换的整个门，滚珠轴承也坏了……并且坏了不止一个。我花了至少两个小时。然后我出来看到Brenan博士靠在墙上，流血不止。他仍然有意识，但他的呼吸不规则，不稳定。

Dr Mazuret : 然后你做了什么？

E. Bonsaint : 我以为太平间是空的，所以马上联系了急诊室。当我看到Armot、Curset和Najib博士离开尸检室时，我正在等待他们的到来。他们看到了Brenan博士，前来帮助他。

Dr Mazuret : 还有别的东西吗？

E. Bonsaint : 别无其他。正如他们所说，我是在维护结束后到达的。我离得不是很远，但也没有听到任何声音

Dr Mazuret : 非常好。没有其他的了？

E. Bonsaint : 没有。

Dr Mazuret : 好吧，这时候，我记得……

E. Bonsaint : 即便如此。

Dr Mazuret : 还有什么要说的吗？

E. Bonsaint : 我不知道……这可能并不重要，但这个想法已经在我的脑海里待了一段时间……

Dr Mazuret : 是什么？

E. Bonsaint : 嗯……我正在检修的箱式门，这会对尸检工作不利吗？

Dr Mazuret : 不知道，为什么？

E. Bonsaint : 一个障眼法，很多逻辑可以解释得通，坏掉的手推车……之类的。这种箱式门是封闭起来的。

Dr Mazuret : 知道，然后呢？

E. Bonsaint : 里面可能有东西。
