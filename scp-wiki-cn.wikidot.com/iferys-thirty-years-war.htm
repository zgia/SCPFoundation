圣克里斯汀娜 书院宗教史实践课程

圣克里斯汀娜星间书院宗教史实践课程

评分: +28+–x

说明：20██年11月12日，Site-CN-91超子物理实验室观测到了一次小规模空间震动，与此同时，空间监测卫星“羲和二号”传来的图像表明一颗球体出现在月球轨道上与月球相对的位置。该球体随即坠向地球并在北纬█°█′█″，东经█°█′█″的太平洋洋面被回收。在球体内部发现了数个带有自译解系统的电磁波发生器。电磁波发生器已被收容并归档为SCP-CN-████，对其内信息的破译工作正在进行中。

备注：在球壳内部发现了一与圣克里斯汀娜书院校徽高度相似的图案以及“CCC ██级毕业生 林██特工”字样。但据由圣克里斯汀娜书院毕业的基金会工作人员表示，圣克里斯汀娜书院从未有过名为林██的学生。

影子监护人 - 目镜部 - 机密档案

仅限内部参阅

项目编号：SE - 163 - J

危险度：海兽级

监禁手段：SE - 163 - J被监禁在位于Iferysia行星的目镜部本部特别监控中心内的异种有机生物用监禁间内，拥有三级或以上权限的目镜部审查员可以在得到批准后与SE - 163 -J交流，对于SE - 163 - J提出的阅读申请需要经过高阶审查员cur'Hussel批准。

SE - 163 - J -1被监禁在本部特别监控中心内的机械监禁间内，对SE - 163 - J -1的研究申请已上报议会。

项目记录：SE - 163 - J为一智能哺乳生物个体，SE - 163 - J -1为发现SE - 163 - J时其所乘坐的宇航飞行器。其于2198.██.██被发现突然出现在Sadolic星系第二行星附近轨道并随即失速坠落，在进入大气层后被附近巡航的长剑部巡逻护卫艇捕捉，随后被移交至目镜部。值得注意的是，被发现时，周围各舰艇、站点的空间跃迁雷达均无反应。

SE - 163 - J拥有与Iferys类似的循环结构，拥有五个运动附肢。与Iferys相似，其神经中枢、感官器官、交换器官均位于最短的运动附肢上。由于未能采集到SE - 163 - J的活体生殖细胞，也未观测到SE - 163 - J进行自我分裂，目前尚不清楚SE - 163 - J是如何繁殖后代的。

SE - 163 - J拥有相当高的智能。似乎通过从捕食口发出低频振动波来传递信息，同时也习惯于使用图像进行交流。高阶审查员cur'Hussel已教会了SE - 163 - J使用Iferys文字。

根据SE - 163 - J描述，其所属文明已具备了在本恒星系中建造大型居住用建筑的能力，但SE - 163 - J拒绝提供其所属文明的坐标信息。

SE - 163 - J对混合舰队的历史——尤其是至高神信仰的历史——表现出了浓厚兴趣并反复申请阅读相关书籍，作为对其配合调查的奖励，该请求已被通过。

对SE - 163 - J - 1的初步调查表明，其技术水平符合SE - 163 - J的描述。但值得注意的是，其动力系统不同于已知的任何一种超空间引擎技术，且含有因果律构造-反构造装置。对SE - 163 - J - 1的进一步调查已被冻结直至得到议会批准。

据SE - 163 - J声称，SE - 163 - J - 1的动力系统已损坏而无法运作，SE - 163 - J提出了在目镜部协助下修复SE - 163 -J - 1的申请但未被通过。

目镜部提案2199-4-1：

提案内容：申请批准与SE - 163 - J的进一步沟通，以试图加深对其所属文明的技术与文化的了解。

SE - 163 - J本身也表现出了此意愿。

评议结果： 3-1-13 否决

议会回应： 未知的技术与思维方式太过危险，目镜部的主要职责是对未知的监控而非探究。

目镜部提案2199-4-2：

提案内容：申请对SE - 163 - J - 1进行拆解，尝试进行技术反溯。

SE - 163 - J - 1的动力系统含有因果律构造-反构造装置，量产此技术对混合舰队益处颇多。

评议结果：8-6-3 可复议

议会回应：对因果律的研究乃对至高神之不敬，可以考虑对动力系统之外部分进行研究。

圣咏部提案2199-4-1：

提案内容：申请对现有至高教会体制进行改革。使其能承担更多职能，应对高不安定度个体不断增多之局面。

根据SE - 163 - J所述，其所属文明中存在名为“圣座”、“教团”的多个与至高教会职能相似之组织。但与职能单一的至高教会不同，“圣座”、“教团”同时具备行政职能，并能以教诲人民之名义执行司法职能与军事职能。

作为改革的参考，申请允许圣咏部教法家在得到批准后与SE - 163 - J进行直接交流。

评议结果：10-0-7 批准

议会回应：必要时可向目镜部与长剑部借调人手。

长剑部提案2199-4-1：

提案内容：申请扩建已有宇宙舰队，以新式战列舰取代超期服役的旧旗舰Seper'Sagredioth号。

旧有的长剑部舰队已不足以应对高不安定度个体的不断增加，此外，舰队中配置的旧型战舰在战斗中能否发挥与其所需维护相匹配的战斗力也值得怀疑。

评议结果：5-2-10 否决

议会回应：长剑部的职能为对不安定个体进行秘密压制，不是与敌对势力正面作战，舰队只需满足机动护航功能即可。

另外，圣咏部已提出了更好的应对方案。

影子监护人提案2199-4-1：

提案内容：考虑对圣咏、目镜、长剑三部门的职能进行变更。

在现有的体制下，圣咏部负责以至高教会为掩护进行情报搜集与舆情监控、长剑部负责对不稳定的“已知”因素进行处理、目镜部对“未知”进行监禁。

但是，随着混合舰队的领土扩张与殖民星系增加，高不安定度个体的数量不断增长，这一体系中由于各部门间沟通不畅导致的效率低下问题变得愈发严重，此外，长剑部的处理手段也逐渐引起了公民的怀疑与不满。

以圣咏部的提案为契机，建议对三部门的职能进行调整，使圣咏部可依据所得情报直接处理部分不安定因素。

评议结果：8-4-5 可复议

奉至圣至上的我主至高神之名，我等以至高教会之名义，于2199年辉1月1日于Safalon召开公会议。以正本溯源，翦除异端，端正吾等侍奉至高神之仪礼。

我等于此立誓，至高神所赐之权柄所鉴，至高神之言辞即我之言辞，至高神之意志即我之意志，至高神之律法即我之律法，至高神之断罪即我之断罪。

秉至高神之言辞、意志、律法、断罪，我等于此发表《Safalon信经》。

此信经乃至高神之言辞。当刊印混合舰队各星域。

此信经乃至高神之意志，唯我等至高神之代言人有权解读订修。

此信经乃至高神之律法，违背律法者，皆为无可饶恕之异端。

此信经乃至高神之断罪，为断异端之罪，特设红衣法庭。

Safalon公会议期间，22位咏者秉持异端而不愿归信正教，已剥夺其咏者身份并移交红衣法庭。

在此吟诵至高神之名。

+ 以下内容仅限四级以上权限
- 权限已确认

《Safalon信经》已经影子监护人审阅，并以10：7通过。

红衣法庭全体成员均由圣咏部各部门抽调人员构成。

对不安定程度为黄级或以上者，必要时可交由红衣法庭处理。

2199年辉5月42日

这是我人生最高的巅峰，也是我——还有整个Iferys文明——最耻辱的一日。

我，cur'Hussel，一天之前还是影子监护人目镜部的一名高级审查员，如今却位列Iferys混合舰队的幕后最高权力机关——影子监护人的顶峰，成为17位议员中的一人。

这不是对我才能的认同、不是对我功绩的奖励、甚至不是对我人格的赞赏，而是对我等悲哀命运的弥补——八个小时前，Iferys混合舰队失去了五分之二的舰队、三分之一的殖民星系、7位影子监护人议员、4位影子监护人候补议员，目镜部也失去了最高审查员dol'Luch与大半的监禁对象。由于残余的议员与候补议员已不足17人，我被直接任命为议员。

造成这一切的不是虚空魔物的入侵、不是地蛇级监禁对象的突破、不是星汉联邦的进攻，不是知识管理者的愤怒，而是我们自己的愚蠢——因为“对我等滥用解释权，篡改至高神意志的不满”，五分之二的舰队、三分之一的殖民星系、7位影子监护人议员、4位影子监护人候补议员在八个小时发布通告称自己为“Iferys福音同盟”并向混合舰队——如果它现在还配得上“混合”二字的话——宣战。

剩余的混合舰队成员毫不犹豫地接受了宣战通告，那10位议员也一致否决了进行外交斡旋尝试的提案，窗外再也听不到令人心情平静的祈祷声，只有狂热的“烧死异端！”“新生混合舰队万岁！”“以至高神的名义！”的吼叫。

我无意评价他们观点的对错，我从不认为至高神能引导我们做出一切决定，指明一切道路。我只知道，跨越了无数矛盾与冲突、经历了无数波折与危机而诞生的Iferys混合舰队，就这样因为至高神而分崩离析了。

星汉联邦向福音同盟派去了大使、知识管理者嘲笑我们的愚蠢，Chicora恒星联盟与福音同盟签订了矿石交易……我们的混合舰队原本是为了抵御他们入侵而生，如今却沦为了它们的笑柄与玩具。

我更担心的是，我们丢失的那些监禁对象恰好都是危险度比较高的那些，我了解最高审查员dol'Luch，若是有机会，他绝对会毫不犹豫地将监禁对象作为武器来使用。

我去拜访了一次SE - 163 - J，他知道了内战的消息后一言不发，只是头部的肌肉扭曲成了奇怪的样子，我不知道这种动作在他的文明中代表着什么，但我对他的眼神很是熟悉——那是前任最高审查员ov'Dren死于收容事故时，dol'Luch的眼神。

不，SE - 163 - J的眼神中没有那种恶意，而是带着——好奇心？

走出收容间时我忽然想起来了，参与叛乱的那七位议员最近都和SE - 163 - J交流过。然后我放弃了扭头回去和它对质的打算，考验我们的不是SE - 163 - J，而是至高神。

如果我们没有打开那艘能源耗尽的飞船，我们的世界会不会更美好？

注意：本通告需对全体长剑部成员宣读。所有已接到此通告的长剑部成员将被剥夺休假与辞职的权利直至作战结束。

作战时间：2202年晓5月12日

动员部队：长剑部全体人员。

作战地点：Sadolic星系第五行星“龙穴”，目镜部第8特别监狱城

作战目的：突破福音同盟军守备部队并阻截混合舰队军第一、三舰队，破坏SE - 2317。

由原第二舰队的残部与原目镜部第一机动宇宙特遣大队组成的守备舰队已在“龙穴”近地轨道驻扎超过20小时并在轨道上建造了一个简易武装堡垒，尽管没有配置战列舰，但这支守备部队的火力依然在长剑部的总战力之上，将其全歼是不现实的。计划以旗舰Seper'Sagredioth号为核心发动突击以破坏其阵型，陆战部队需趁机空降至目镜部第8特别监狱城。

驻守在目镜部第8特别监狱城的原目镜部机动部队的数量与质量都远不如长剑部的陆战突击队，但由于无法对地面进行轰炸压制，陆战部队仍将面临苦战，此外，第8特别监狱城内的监禁的异常对象也可能对破坏工作造成妨碍。

混合舰队军第一、三舰队已于3小时前自Iferysia基地出发，考虑到福音同盟军的阻截，预计将在6小时后到达“龙穴”近地轨道。为尽可能为地面部队争取时间，计划在混合舰队军与福音同盟军开始交战后，利用混合舰队军尚未将我军视作敌军的机会对混合舰队军发起突袭。突袭的首选目标为运载陆战部队的阳陆舰与混合舰队军旗舰。随后，尽管仍会遭到福音同盟军的攻击，我军舰队仍将全力攻击军力占优的混合舰队军。

由于无法为扬陆艇制造起航空域，此次作战将不会回收陆战部队，各舰艇也不得以任何理由脱离战线，务必尽可能为陆战部队争取时间。

附录：长剑部最高指挥官Codrales的讲话

诸位长剑部的同僚，这可能是我最后一次以最高指挥官的身份与你们讲话。

你们没有听错，此次作战中，我们将同时与福音同盟军与混合舰队军为敌。

但是，这并非背叛，而是在履行我们长剑部、履行影子监护人的责任——为了保护Iferys而战。

这不是空口无凭，而是得到了共计9位影子监护人议员的批准的一次行动。尽管这9位影子监护人议员分属不同阵营，但他们依然是影子监护人，依然是为了保护Iferys而抛弃私欲、奉献终身之人。

还有人对此次行动的合理性抱有怀疑吗？很好，接下来我来解释一下此次作战的目的。

在座诸位中，稍年长者可能还记得在开拓Sadolic星系时发生的“事故”：在Sadolic星第五行星附近宙域发现了一只巨大的宇宙生物，在长剑部、目镜部的全力协助下，第二舰队付出了惨痛代价才将它无力化。为了防止它再度苏醒，目镜部在第五行星表明建造了第8特别监狱城以监禁其尸体。

但这只是表象。

事实上，那只巨兽不会再苏醒了，我们已经将它彻底粉碎，第8特别监狱城最初的收容对象不是它，而是一扇门——一扇巨兽用来进入这个宇宙的门。

关于这扇门的具体情报我就不详细描述了，你们只要知道一件事：这扇被编号为SE - 2317的门背后一直传来一个声音，用各种诱人的代价来诱惑我们打开SE - 2317，毫无疑问地，之前我们都将这扇门紧紧关闭。但随着福音同盟的节节败退，他们决定打开它来孤注一掷了。

那些反对此提案的福音同盟高层带着这个情报与混合舰队接触，希望能阻止这个疯狂的计划，但混合舰队做出的决定却是由自己来打开这扇门，作为在这场愚蠢内战中损失的战斗力的代替。

所以最后他们找到了我们，拟定了这个计划——这个有去无回的，需要投入长剑部全体战斗力的计划，从而彻底摧毁SE - 2317，阻止它被打开。

如果我们成功，那我们拯救的不止是Iferys文明，还有整个宇宙；如果我们失败了，那至少我们可以问心无愧地死，而不会在成为那个怪物的饵食之时感到后悔。

最后特别提醒全体同僚一件事，你们知道，为何福音同盟与混合舰队的上层会一致赞同打开SE - 2317吗？

SE - 2317背后传来的声音有很多种自称，正因为每个听到他低语的人听到的自称都不同，我们才能断定SE - 2317口中只有谎言——包括他的许诺。

但现在，那个声音只有一种自称——至高神。

我们是黑暗中的长剑，现在正是出鞘之时。




我只是给他们补了补课。

—— 圣克里斯汀娜星间书院St. Christina Cosmos College ██级毕业生 林██
