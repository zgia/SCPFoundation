项目编号：項目編號：351
等级等級3
收容等级：收容等級：
euclid
次要等级：次要等級：
none
扰动等级：擾動等級：
keneq
风险等级：風險等級：
需谨慎
备忘录链接

特殊收容措施：全部SCP-351拷贝存于安保基金会收容数据库中，只可由模因部人员访问。若要进行测试，将使用仅包含SCP-351-1的“空白”SCP-351拷贝来无效化此异常对测试对象及可能受感染人员的影响。

机器人I/O-SHANNON及I/O-MINERVA 被编程为定期扫描网络网站，寻找SCP-351拷贝是否出现。被发现的副本将在下载后依照标准网络异常收容协议从网站上移除。机动特遣队ψ-13（"EOT"）被指派在SCP-351影响大量人口时阻止集体性记忆篡改发生。特遣队特工必须遵守标准模因隔离协议，配备能显示前述空白拷贝的电子设备，且必须每日查看两次。

正在展开测试以确认该异常植入记忆的边界范围。

描述：SCP-351是一ASCII纯文本（.txt）文件，内含病毒性模因触媒，会向观看该文件的人类植入表现为ASCII艺术画的假记忆1。异常的主要组成部分为一30个符号的字符串SCP-351-1，其字符不与任何已知编码标准相符，并将整篇文件转化为模因性媒介。

添入文件的额外文本是SCP-351植入假记忆的基础，文字的细节度与对象所能回想起的“记忆”细节度相符。所有记忆均能被生动回想，且抵抗一切记忆删除性治疗和效应。文件的拷贝版（SCP-351-A）表现出同种性质。然而，若暴露于除SCP-351-1外再无其他文本的SCP-351（称为“空白”）或其拷贝，将导致记忆被“空白”记忆替代、彻底移除。这也是移除此种模因性感染的主要手段。

虽能清楚回想，对象报告称这些记忆缺乏其他记忆的直觉性和丰富性。理论认为这是因为记忆所基于的文本字符串本身缺乏正常记忆所基于的感官可感性。

受此文件模因感染的对象（编为SCP-351-B个体）可以将自己的假记忆植入给相关人员，诸如朋友、同事。与个体以任何媒介的对话都会使个体谈论SCP-351记忆，将对话转化为一段影响所有参与者的模因性媒介。这些受影响者也会成为 SCP-351-B个体，且可以相同形式传播此模因。

除添入此前不存在的记忆，SCP-351-B还会经历到其他几种认知影响。个体在不与非感染人员对话时表现同于感染前，而若向其问及SCP-351记忆与真实记忆的冲突，个体会予以无视。 SCP-351-B个体中出现嗜睡症、快感缺乏和冷漠症状的比率远高于未暴露人口。

附录1：SCP-351文本示例

下列内容摘录自发现时的SCP-351-A个体。完整拷贝内容可在文件351-RT/267查阅。

个体编号：SCP-351-A5

你在树林中行进，一片漆黑，只有幸存者的手电筒和头顶飞过的敌人炸弹。它们的爆炸看起来就像致盲的闪光。John的血还在你和你的枪上温热着，每次你回头你都期望能看到他在那里，一如既往地微笑着。但他不在。凝固汽油弹让一切变成了焚身炙热。

棕榈树摇摆断裂，灰烬进到你的鼻孔里。敌人的士兵从树荫中现身，顺着绳索降下，向你开枪。你这一辈子都没感受过如此灼热、刺人的疼痛，直到这子弹将你的身体打中，感觉如同一千只火蚁挤在一道开放伤口上。幸运的是你掉落在地上，脱离了他们的视线。其他人没有。你听到了他们的惨叫。你只看到沾血的泥土在你的面前。

你知道敌人做了这些，你为此仇恨。你恨敌人。那时你发誓要用余生与敌人斗争，你发誓你要跋山涉水去到[坐标已编辑]为与敌人的永恒之战受训。为John，为幸存者，为世界。

发现：发现于阿拉斯加州德纳里国家公园保护区内一处废弃军事建筑内的电脑中。该地的其他文件表明曾有计划将SCP-351-A5的副本散播到多个网络论坛，但因未知原因没有执行。

个体编号：SCP-351-A19

你没能开车穿过红灯。是绿灯，街道上没有行人车辆。你的视线好不模糊-你非常清醒。天很暗，但你的头灯开着，你可以看得到道路。你的妻子没有对你大吼大叫。她的头斜靠在肩膀上。

你看到另一辆车驶来，这样你可以及时转弯。你在街上对的一侧，他们在错的一侧，所以你没做错什么。你们的车本来不可能撞上，挡风玻璃不可能碎掉，但即便这发生也没有意义，因为你会及时转弯。你的安全带扣上了所以没有人会飞出窗户去。你没看到血。或者玻璃片在你身上。或者她身上。

你和她的一周年庆典在第二天。是个很棒的庆典。她在笑，微笑，说着她的实验室工作是多么的好。第二天很开心。之后几个月也是。她和你一直在一起，她没有离开。周年后是四个月的快乐。

这不是你的错。

发现：于Sandra Holt自杀后在04/12/2006发现于她的个人电脑中。

采访者：潜伏特工Sascha Hahn

受访者：Alisya Dean

前言：Alisya Dean是Sandra Holt生前的同事，且因为频繁与Holt交互变为了SCP-351-B。采访结束后，Hahn特工和Dean均进行了空白暴露以抹除模因感染。

< 开始记录 >

Agent Hahn：我能问些关于Sandra Holt的问题么？

Alisya Dean：这不是她的错。

Agent Hahn：这不是她的错，我肯定，但我还是得问你些问题。你们认识了多久？

Alisya Dean：我……大概三年吧，我想，从我在[已编辑]开始工作起。她是个好老板，有点安静吧大概，但也不是个铁石心肠之类的，真的很敏感。然后她开始和Jane Kysely约会…我有提到过她嘛？她的妻子。好吧，那时候是未来的妻子。然后就想一道光照亮了她的内心。

Agent Hahn：Jane是什么样的？

Alisya Dean：我只见过她一次，就是在她的周年庆典上见过她，之后她就在车祸里死了她活着，我也在车上。是个很好的庆典。她在大笑，微笑。她没有飞出窗户去。

Agent Hahn：是的，庆典很棒。呃……（停顿）我们这跑题了。她看起来到底怎样，Jane，她的妻子？

Alisya Dean：啊，她像是…五二，鼠棕色头发，有点直直的，像是个U+007C垂直线，明白吗？虽然不是我的事但你看得出来他们真的深入彼此，很甜。她的身体上没有玻璃片。

Agent Hahn：没有玻璃片，对。

Alisya Dean：他们两个都没有。或者血。我没看到血。

Agent Hahn：对……（停顿）抱歉，有些事…你提到说一次事故？

Alisya Dean：不，我是说，对。她及时转弯了。没有人飞出窗户去。很清楚，没发生过。不是她的错。

Agent Hahn：不是她的错。但……有一次事故。

Alisya Dean：是，他们周年后的那天。然后他们开了庆典——是场好庆典——然后几星期过后是她的葬礼。然后他们有四个月的快乐，再然后Sandra……自杀了。

Agent Hahn：是的，这有道理。她自杀前几天心理状态怎样，上班时候？

Alisya Dean：她一直把这文件在电脑上开着，一遍又一遍重看，一边哭。这很诡异，因为她已经有了几个月的快乐。她也在喝酒，这也很怪，因为她很清醒，她的视线没模糊——她可以看得清道路。

Agent Hahn：是的，就这样的。她看得清道路。她的灯是开着的。

Alisya Dean：她的灯是开着的。不是她的错。

< 记录结束 >

附录2：实验记录351/1

于09/01/2007, 研究员Ferdinand Fortier获得许可进行基础测试，确认SCP-351记忆改变能力的边界。测试中Fortier及其研究团队通过自动程序在SCP-351-A个体内加入文本，然后展示给一名D级对象，令其感染后要求其描述记忆。之后会向对象和整个研究团队展示空白，以最小化模因传播的可能。

▷ 实验记录351/1
▽ 实验记录351/1

测试编号：1

所用文本（概要）：文段详细描述一只长有绿眼的黑白双色猫，说成为对象当前所有。

结果：对象构想的猫为猫形，由ASCII字符组成了其身体、皮毛和影子。所有字符为灰度，但眼部为“@符号的虹膜”。未提及绿色，表明记忆的构想只限灰色。

过去式测试结果相同，但对象无法描述为何当前不再拥有此猫。

测试编号：2

所用文本：你有一只黑猫，一只条纹猫，和一只白猫。

结果：ASCII块字符组成的黑猫被描述为“像个洞”，条纹猫是连字符和类似符号组成的猫形，白猫是由句号和点组成的猫轮廓。无法对其详细描述。

测试编号：3

所用文本：你有一只猫。

结果：对象称“我有一只猫”。无法对其详细描述。

测试编号：4

所用文本：你将有一只猫。

结果：对象描述了一段长而不连贯的记忆，内容涉及“反转失明”、猫科生理结构和太空旅行。更多关于将来时态文本的测试得出类似结果。

测试编号：5

所用文本（概要）：测试1中的文本，但翻译成了西班牙语2。

结果：所植入的记忆被描述为十分让人迷失方向，对象无法说明其中缘由。其中的“猫”被描述为有“歌唱的四肢”，快速变化的ASCII符号形状，以及由未知语言广告组成的眼。不同对象的构想不一致。

测试编号：6

所用文本（概要）：遭遇了SCP-3840的不安童年,刻意写成提到颜色、声音、具体的物理形象和其他可感内容。

结果：在被给予SCP-3840相关图片和信息时，对象表现出预期的恐惧回应，但在说明或描绘其记忆时并无此回应。虽然细节度高于此前测试，记忆仍然具有ASCII艺术质感，有“模糊”的周边环境，未被清楚描述的物体为不定形形状。没有非灰色彩被描述。

测试编号：7

所用文本（概要）：测试6文本修订版，但字数翻倍且细节更多。

结果：预期的恐惧反应，且表现出对密林的恐惧，但ASCII艺术质感造成的“真实感”缺失仍然削弱了整体的情绪冲击力。关于细节度和植入记忆生动度的关系假说得到确认。

测试编号：8

所用文本（概要）：在测试#7同样场景下被一支机动特遣队救援。

结果：出现同样的恐惧反应但较弱。对象对机动特遣队产生更大的亲密感。然而其仍然因过去在基金会的D级测试保有怀疑。

测试编号：9

所用文本（概要）：一系列事件，内容为对象曾是机动特遣队成员，最终作为对某（虚构的）异常收容的一部分被降为D级人员。

结果：对基金会产生更大亲密感。作为D级人员的经历未被植入记忆覆盖，有部分作为事件被整合到了特遣队经历中，造成其仍保有对基金会的重度怀疑。ASCII艺术式构想对此有何影响目前未知。

测试编号：10

所用文本：你从未怀疑基金会。你一直忠于基金会。

结果：对象被发现赞同基金会的协议，诸如收容和对帷幕的维护。对D级测试的矛盾心理未被完整移除。

正在计划使用更长、更详细的文本来就改变人格特征进行实验。

备注：对此性质的更多测试已由伦理委员会下令禁止。

附录3：锡兵计划

2007年3月29日，在Lucinde Behringer研究员对SCP-9906进行常规监控时，该异常的认知危害性质绕过了全部模因过滤系统，将Behringer转化为模因媒介。在收容突破发生前，Behringer注射了她身边所有可用的记忆删除剂，并将自己暴露于实验性抗模因符。异常和此手段的累加效果造成她的人格、记忆和近乎全部心理功能被彻底抹除。在被联入生命维持后，未能发现将Behringer恢复到之前状况的办法。

2007年4月21日，伦理委员会批准实验性使用SCP-351重塑Behringer的人格及记忆。机密计划“锡兵”、所有关于Behringer的数据库信息、同事通信3和个人日记被收集起来，写成极为详细的文本，有可能恢复她的心理状况。该文本内将以编年顺序包含入所有已知的想法、生活事件、记忆和人格特征。到01/10/2007该文本已达50000字。

在无法找到新信息后，文本被添入一特别编号的SCP-351-A拷贝（SCP-351-AΔ）。 于2007年10月10日研究员Behringer被暴露于文件。起初仅出现轻微肢体移动，但一小时后她开始在房间内行走。第二天对她进行了采访。

▷ 采访SCP-351-BΔ/1
▽ 采访SCP-351-BΔ/1

采访者：研究员Rayyan Bousaid

受访者：Lucinde Behringer（编为 SCP-351-BΔ）

日期：2007年10月11日

前言：Bousaid 研究员在SCP-9906突破前是Behringer的同事和亲密好友。他是锡兵计划的研究领导人之一，因对她的熟悉，他专门申请进行初期采访。

< 开始记录 >

Bousaid：Behringer?

SCP-351-BΔ站在房间中心，茫然地注视着一堵墙。

Bousaid：Behringer，你能不能移动到窗边的椅子去？

他敲打了下隔离房间与观察间的玻璃。没有回应。

Bousaid：Lucinde？

SCP-351-BΔ慢慢转过头。她把桌椅移动到窗前，就座。

SCP-351-BΔ：你好。

Bousaid：你……你还好吗？

SCP-351-BΔ：我感觉很好。

Bousaid：行，那就没事。（深呼吸）那什么，我对不专业抱歉。已经好久没见你看起来，好吧，是活着的了！我希望你理解。

SCP-351-BΔ：我知道。

需注意，在此时SCP-351-BΔ的面部表情没有任何变化。

Bousaid：（又一口呼吸）所以，你之前躺下的时候感觉怎样？我们给你展示文件之前？

无回应。

Bousaid：没有吗？不能想到什么吗？（沉默继续）好，就这样吧。我不该吃惊。所以——

SCP-351-BΔ：我没发现有必要多言。简明总是更好。

Bousaid：好吧，这和我们写进文件里的是一样，但你也没搞错。 (发笑chuckles) 所以，你记得你的丈夫是谁，对吧？

SCP-351-BΔ：Mike Landon Page，我绝不会忘记。

Bousaid：行，很好，你是否还记得你的婚礼？你知道，当你们在——

SCP-351-BΔ：我们在通往林地连弯道尽头的教堂。太阳是发光的破折号环，树叶是无数摇晃的角引号一直飘动。风寒冷。云是鼓鼓的括弧。是美丽的一天。

Bousaid：……行，听起来不错。

SCP-351-BΔ：因为我们牵涉模因部基金会监控了婚礼并限制来宾为我们的同事但还是一样的美丽。

Bousaid：很好，不错。你记得其他关于云的事吗？树呢？

SCP-351-BΔ：云是鼓鼓的括弧，树叶是无数摇晃的角引号一直飘动。还有什么要说的？

Bousaid：我知道你不是爱说话的人但你还有别的想说吗？哪怕就一句 “我记不得别的事”？

SCP-351-BΔ：我记不得别的事。

Bousaid：我……那好吧。那你记得你有何感觉吗，任何情绪？

SCP-351-BΔ：我一生中从未如此开心。Mike Page面容的每个破折号和字母都令我满心欢喜。他一直在提那些我们几年前看过的星际迷航烂剧集而我止不住要笑。夜里当我们躺下看着天空的星号我感觉我再一次陷入爱河。

Bousaid：记得这些会让你开心吗？

SCP-351-BΔ：是的。

Bousaid：那么为何你的语气一点不变？你的面部表现为何毫无变化？

无回应。

Bousaid：你说这让你开心但我可看不出来，Lucinde。我记得婚礼之后的每天我哪怕稍微一提你都止不住面带笑容，就像Mike脸的每个破折号和字母——

Bousaid看向他电脑上的空白后继续说话。

Bousaid：我稍微一提，之后我就会看到你微笑。

SCP-351-BΔ慢慢的微笑起来

Bousaid：你现在在做了，但这像是你复制了我说的话。感觉不真实。

SCP-351-BΔ：我一生中从未如此开心过。

Bousaid：但你一直重复这句，我是如何前所未有的开心……（深呼吸）这感觉不是真的，Lucinde我感觉和我说话的不是真的……真的你。

SCP-351-BΔ：我就是我自己，Ray。我不懂你什么意思。

Bousaid：（叹气， 停顿）我之前不想告诉你，因为云是鼓鼓的括弧，是美丽的一天。但我觉得我应该说。

SCP-351-BΔ：很美丽，但是什么？

Bousaid：Mike死了。

无回应。

Bousaid：他在你之后几个月也遇到了收容突破。他留下了最后的信息但特工们再没找到他的遗体。

SCP-351-BΔ：噢。

SCP-351-BΔ的微笑变为皱眉。

SCP-351-BΔ：噢上帝。这怎么可能发生。

Bousaid：你就只有这种反应吗？

SCP-351-BΔ：我在心神错乱。这怎么可能发生。噢上帝啊。

Bousaid：我躺下看着天空的星号但这就完了吗？求你能不能，求你想想你和他一起的其他时光？能不能让你想出其他的东西？

SCP-351-BΔ：我不明白你什么意思。我爱他胜过世上其他任何人。

Bousaid：但这和我们写到文件里的一模一样，我们在这写了我们在教堂在尽头……求你，能不能想想文件以外的东西？任何东西？这些记忆没有给你任何之后的东西吗？

SCP-351-BΔ：我爱他胜过世上其他任何人。

Bousaid：这就是…我们写过的。是逐字逐句我们写过的。

SCP-351-BΔ：他有如此美丽的灰梯度井号而且眼里是和号。我爱他胜过世上其他任何人。

Bousaid：这就是…

SCP-351-BΔ：我爱他胜过世上其他任何人。

Bousaid：我爱他胜过世上其他任何人。我很抱歉，Lucinde…

研究员Bousaid离开观察间。SCP-351-BΔ继续看着Bousaid之前所在的空间皱着眉。没有做出更多动作。

< 记录结束 >

在情绪紧张下，Bousaid研究员离开房间时没有观看电脑终端上的空白版，违反了标准安保协议。这造成SCP-351-BΔ的记忆构造模因性传播到所有在场的锡兵计划人员，直至站点安保被触发并得到处置。

不同寻常的是，在暴露于空白个体后，这些受影响人员对SCP-351-BΔ的全部记忆近乎彻底丢失。理论认为这是因为 SCP-351-AΔ植入记忆与SCP-351-BΔ的真实记忆过于相似而发生关联，造成植入记忆的抹除将真实记忆一并牵连。由于所有SCP-351-BΔ的相熟人士都被锡兵计划招募，也都在Bousaid违反协议时在场，所有构成SCP-351-AΔ基础的原始记忆均已丢失。

Bousaid研究员从所有SCP-351相关研究中调离，安保协议被修订以防此类事件重演。然而，SCP-351-BΔ被证实能有效地在孤立环境内阻隔模因传播。在修订SCP-351-AΔ以移除情绪相关信息、加入提升忠诚文本4后，SCP-351-BΔ可继续每日进行有助于模因研究的活动。

若自动为对象提供健康支持的生命维持系统计划实现，SCP-351-BΔ被预期能无限期服务于基金会。

1. 一种图形设计艺术，使用可印刷ASCII标准和适当字符形成图形。
2. 本测试的对象D-3468不说西班牙语。
3. 为收容SCP-9906之必要，分配到此异常的研究员必须切断与模因部外一切人员的联系，以降低意外模因传播的可能。
4. 基于Fortier研究员在实验记录351/1中的发现。
