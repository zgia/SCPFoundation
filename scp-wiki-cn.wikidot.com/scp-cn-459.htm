SCP-CN-459
评分: +18+–x

项目编号：SCP-CN-459

项目等级： Keter

特殊收容措施： 鉴于SCP-CN-459现象出现的无规律性，目前无法对其进行有效收容，基金会需保持时刻监视全球各地的低纬度高亮度极光现象及该极光可见地区的休谟指数以判断是否为SCP-CN-459现象。

一旦确定了SCP-CN-459现象的发生，需立即就近抽调机动特遣队携带斯克兰顿现实稳定锚尽快赶赴现场，且该机动特遣队必须保持与与其直线距离在1000千米以上的指挥中心的视频连接。若机动特遣队抵达SCP-CN-459-2地区时，SCP-CN-459现象进程并未进入第二阶段，则需在SCP-CN-459-2内启动斯克兰顿现实稳定锚10分钟以对其进行抑制处理，并对SCP-CN-459-2地区民众视情况进行对应等级记忆清除操作。一旦机动特遣队抵达SCP-CN-459现象中心点时，该现象已经进入第二阶段之后，则需通过视频连接确认SCP-CN-459-1，并在该现象结束后，对社会上SCP-CN-459-1未受影响的相关档案进行归档与销毁处理。此外，对SCP-CN-459现象开始后进入SCP-CN-459-2地区的人类个体，需实施B级记忆清除操作。

目前本项目的相关研究由Site-CN-91的Dr. Howard Zheng Dr. Gunnarr Ch'in领导，在单一的SCP-CN-459事件结束后，需将相关音像材料与必要情况下的归档情况交付Dr. Zheng Dr. Ch'in主导的研究小组进行研究。

对囿于客观原因无法归档的已知SCP-CN-459-1相关档案，需有序引导相关研究人员的观点走向，以近缘正常概念的档案错误等理由对其进行掩藏。

描述： SCP-CN-459现象为一种通常在中低纬度出现的异常类极光现象1，但其出现通常与地磁活动与高层大气带电粒子的活动没有较大关系。与通常的极光现象不同的是，几乎所有的SCP-CN-459现象均呈暖色调，且具有在白天仍可清晰可见的高亮度。

SCP-CN-459事件会作用于某一特定地区（下通称SCP-CN-459-2）的某一特定概念（下通称SCP-CN-459-1，目前观测到的SCP-CN-459事件记录中的SCP-CN-459-1概念包括实体与信息两大类概念，若在SCP-CN-459-2地区内有多个该概念个体，则这些概念个体都会受到SCP-CN-459-1现象的影响而一同消失），并使其完全消失。在SCP-CN-459事件发生之后，在SCP-CN-459-2及邻近地区对该概念的所有信息记录也会消失2，包括但不局限于文字记录、影像记录、记忆记录等。

SCP-CN-459现象通常发生在所在地点的冬半年夜间，亦即南半球的3月至10月，北半球的10月至3月。早期观测到的SCP-CN-459现象恰巧大多发生于冬半年夜间，但现有长期观测记录已经表明，SCP-CN-459现象在发生时间上没有明显的规律性。

SCP-CN-459现象的发生地点没有明显的规律性。

在对SCP-CN-459事件的表述中，通常还会使用SCP-CN-459-3表述SCP-CN-459事件中出现的一种特殊光柱，详见下文。

1998年12月26日更新：SCP-CN-459现象的实质与过程
关闭文件

1998年12月26日更新：

在1998年5月23日于湖北省荆门市发生的SCP-CN-459事件（下称“98•5•23事件”）中，观测到极高的休谟指数，据此可初步判断SCP-CN-459事件的实质是一类目前仍未完全解明原因的大规模现实重组事件。

通过分析98•5•23事件及同年稍晚一些时间内其他类似事件中的观测记录，SCP-CN-459显示出以下四个阶段。

阶段 距离事件开始时间 表征
第一阶段 0~约60分 大量带电粒子开始聚集在SCP-CN-459-2（一般是以某一点为中心，半径为500米至1千米不等的圆形区域）上空，并在地磁场作用下显现出渐次变亮的光芒。在这一时间段内，SCP-CN-459-2地区的休谟指数逐渐开始整体降低，直到维持在1左右。SCP-CN-459-2地区内的具意识个体此时不会有任何异常举动。
第二阶段 约60~70分 SCP-CN-459-2内部的休谟指数会突然显著降低，且同时使用多个康德计数器测量会导致休谟指数的进一步降低3，这一阶段总时间与SCP-CN-459-2内部的休谟指数呈正相关。在以SCP-CN-459-2为中心，可在SCP-CN-459-2中心点海拔水平面处肉眼观测到SCP-CN-459现象的地区也会呈现出不同程度的休谟指数降低。此时在SCP-CN-459-2内部会出现很多从SCP-CN-459上映射下来的小型光柱（下通称SCP-CN-459-3）4，且SCP-CN-459-3的投影会在SCP-CN-459-2内部的地面及建筑物外表面上无规律移动。值得注意的是，这一时间段内，在事件开始时即位于SCP-CN-459-2内部的物体运动速度会显著放缓，但在事件开始后进入SCP-CN-459-2内部以及在进入第二阶段之前即离开SCP-CN-459-2的物体速度并不会受到影响，且这一现象能被事件开始后进入SCP-CN-459-2内部的人员察觉。
第三阶段 约70~90分 所有SCP-CN-459-3会就近向SCP-CN-459-1个体聚集（若SCP-CN-459-1为信息，则向其物理载体聚集）。在所有SCP-CN-459-3完全聚焦于所有SCP-CN-459-1个体后，SCP-CN-459-2内部的休谟指数达到谷值，类似此时SCP-CN-459-3会呈现出类似透明非晶体固体的状态，且无法被外力穿过或破坏。若SCP-CN-459-1为实体概念，则所有在SCP-CN-459-3内部的SCP-CN-459-1个体及其附属物会同时承受巨大的未知切割力量而碎裂成肉眼观察由其组成物质类似物组成的小块，该小块很快会在类似力量的作用下进一步碎裂直至肉眼不可见。若SCP-CN-459-1为信息概念，则其物理载体会逐渐失去相关信息而呈现混沌状态。当SCP-CN-459-1个体完全碎裂成肉眼不可见颗粒或无法辨识的混沌状态后，在SCP-CN-459作用范围内关于SCP-CN-459-1的信息记录会逐步消失。
第四阶段 约90~120分 SCP-CN-459-3消失。随后受影响人群中会随机出现对SCP-CN-459-1（尤其是信息类）的虚拟记忆5。在这一过程中，SCP-CN-459现象逐步消散，SCP-CN-459-2区域中的休谟指数也逐步恢复正常。

目前SCP-CN-459事件中异常带电粒子与休谟指数异常的来源仍然未知。

2002年9月2日更新：SCP-CN-459现象的可能应对方式
关闭

2002年9月2日更新：

在2002年6月29日于福建省晋江县发生的SCP-CN-459事件中，斯克兰顿现实稳定锚首次被投入使用，并成功在5分钟内使处于第一阶段中后期的SCP-CN-459-2内部的休谟指数从1.5升高至2，且在关闭斯克兰顿现实稳定锚后，SCP-CN-459现象并未重新出现。这一实验结果表明斯克兰顿现实稳定锚对早期SCP-CN-459事件有抑制作用。

在2002年7月9日于台湾台北县发生的一次SCP-CN-459事件中，由于交通的客观原因，当基金会员工携斯克兰顿现实稳定锚抵达SCP-CN-459-2时，当地的SCP-CN-459事件已演替进入了第二阶段。基金会员工贸然打开斯克兰顿现实稳定锚后，SCP-CN-459现象不仅未被抑制，反而加快了其演进速度，导致在60秒内迅速进入第三阶段并对SCP-CN-459-2地区内的莫氏树蛙（Rhacophorus moltrechti）种群带来了毁灭性打击。这一结果表明斯克兰顿现实稳定锚的使用对中后期的SCP-CN-459事件不仅无法起到抑制作用，反倒会进一步提升其休谟指数，导致其演替速度大幅度加快。这一现象的机理目前仍未知。

发现与观测： 1983年，供职于台湾██大学██杂志社的基金会员工Dr. Shz-hsiu Hsiung发现有两篇投稿至该杂志社的历史论文从不同方面对位于广东台山一带的同一聚落“万滩”的存在进行论证，但对该聚落的存在没有其他同时期资料佐证。Dr. Shz-hsiu Hsiung开始觉得这可能是某一异常现象的结果，但在报知时任Site-CN-██主管后，并未得到基金会方面的足够重视。Dr. Shz-hsiu Hsiung随后于1985年经多国辗转亲赴台山调查相关情况，但很快因被群众举报为特务而被大陆当局控制。1986年2月，经基金会中国分部斡旋，Dr. Shz-hsiu Hsiung返回台湾，随后被诊断出轻度精神分裂症，提前退休并赴帕劳静养。

1988年，在甘肃省敦煌县观测到了低纬度极光现象，造成了当地高度恐慌。基金会介入并对当地居民实施大规模记忆清除后，发现基金会使用的冷战时期美国测绘的地图在敦煌县██区（今属██镇）多绘有1个不存在的村落，在比对台湾方面资料时，也查到了该村落的存在，但在随后的实地走访中对地图上该村落所在处调查并未发现任何村落存在的痕迹。其时于Site-CN-91供职的Dr. Howard Zheng察觉到其中的异常之处，并比对1983年Dr. Hsiung的提议以及Dr. Hsiung在台山的一些调查记录，发现了二者之间的吻合之处，因而提议对这一异常现象立项。1989年1月，SCP-CN-459立项，由Dr. Zheng领导研究。但由于基金会内部一些声音认为这一异常现象的异常性质仅仅是导致档案错误，因而SCP-CN-459项目一直未得到重视。

1989年6月，基金会员工Agent Ma直接目睹SCP-CN-459事件并利用电话详细向其站点主管描述了SCP-CN-459事件的经过（详见附录-I），这是首次对SCP-CN-459事件的远程监控，随后，全国多处均报告出类似事件，这些事件均交与SCP-CN-459研究小组研究。Dr. Zheng等逐渐总结出了SCP-CN-459事件的基本性质。值得注意的是，1990年5月，西班牙分部首次报告了秘鲁普诺大区的████████村地区发生异常低纬度极光事件后当地克丘亚母语者改说艾马拉语或西班牙语的异常事件，这是首次确认发生在中国境外的SCP-CN-459事件。

2002年6月基金会首次将斯克兰顿现实稳定锚运用到SCP-CN-459事件中并取得了重大成效后，基金会随即开始将斯克兰顿现实稳定锚广泛用于抑制SCP-CN-459事件之上。

截至2017年12月31日，基金会已观测到6██次SCP-CN-459事件，发生地点分布于2█个国家，其中基金会成功抑制了共计3██次SCP-CN-459事件。

附录-I： 1989年6月13日，Agent Ma在江西省宁都县参与Site-CN-██建设的调研相关工作时目睹SCP-CN-459现象（下称“89•6•13事件”），并通过电话与位于Site-CN-██内的Dr. Yu通话并成功记录下了第一手资料。Agent Ma事后表明对89•6•13事件中的SCP-CN-459-1实体没有任何记忆，经检查Agent Ma的大脑没有包括记忆清除痕迹在内的任何损伤。

权限核验通过，欢迎

前言：1988年，为了填补赣南与粤东地区异常事件应对体系的空白，基金会计划在赣南地区修建Site-CN-██，因而派遣包括Agent Ma在内的16名基金会员工到赣南各县与当地政府磋商相关事宜。其中Agent Ma和Dr. Yu被派遣至宁都县。Dr. Yu由于Site-CN-██相关工作于1989年5月返回，随后Agent Ma便见证了89•6•13事件。

<记录开始，18:12>
Site-CN-██接待人员： 您好，这里是██市███医院，请问有什么可以帮助您的吗？

Agent Ma： 我是基金会特工马██，请帮我转接3级研究员Dr. Yu。

（电话转接）

Dr. Yu： 什么事啊，小马？

Agent Ma： 于老师，我有很重要的事情向您汇报。我现在在江西省宁都县██区██村。您对这个地方还有印象吧？

Dr. Yu： 有啊，发生什么事了？

Agent Ma： 17点10分的时候，这里突然发生了看起来类似极光的现象，我那时候在村子后面的山上，可以看到橘红色的光正好照到村子的上空。后来进到村子里面来的时候，可以看到村民都很惊讶地望着天，我问他们这是怎么回事，村长，那位78岁的老人家说他一辈子都没见过这种东西。

Dr. Yu： 请尽量精简你的语言，我需要更多的有用信息。

Agent Ma： 好，那我尽量。大概18点07分的时候，从那片极光里面射下来一些橘红色的光柱，在地上扫来扫去。我不知道怎么描述，就类似于舞厅里面那种灯的感觉，您能明白我的意思吧？

Dr. Yu： 嗯，继续。

Agent Ma： 然后现在那些光柱开始逐渐往村里的樟树上面靠，就像我以前去您家做客拿镜子往您家猫脸上照那样。你看，现在那些光柱完全罩住那些樟树了，我还可以从窗户这儿看到很多樟树，村头的、村子左右的，都被光罩住了，不过我还是可以看到，就跟隔了一层橙色玻璃一样的。（Agent Ma突然兴奋。）哦哦哦！那些樟树开始、开始碎掉了，就好像有什么东西把它们捏碎了一样！哦哦哦！

（Agent Ma持续不停惊叫着。）

Dr. Yu： （提高了声音）小马！小马！冷静一点儿，小马！告诉我你看到了什么！

Agent Ma： 于老师……那些樟树，“咔擦咔擦”地就被捏碎、哦不对，是被跟什么东西碾碎了一样，我可以看到那些木屑在光柱里头，但是一点都没飞出来，一点都没有！

Dr. Yu： 嗯，然后呢？

Agent Ma： 然后……

（Agent Ma停止回答，Dr. Yu呼叫了长达三分多钟。）

Agent Ma： 啊，于老师。

Dr. Yu： 我的天哪！小马你终于回话了。那些樟树，怎么了？

Agent Ma： 樟树？什么樟树？这边好像没有看到什么樟树。那个，于老师，我打电话是想说，我大概下个月十几号就能回来了，这边不太愿意配合选址。没有什么别的事，我就先撂了啊。

（Agent Ma挂断了电话。）

<记录结束，18:29>

后记： Agent Ma于1989年7月返回Site-CN-██后，由Dr. Yu主持对其进行了审问与检查，确认其没有受到记忆清除类似操作。在比对Agent Ma带回的照片与5月Dr. Yu带回的照片资料后，确认Agent Ma所带回的照片中所有樟（Cinnamomum camphora）个体的存在都被抹去。

离开Amacão6不到半天，我们便到达了Mantão。

Mantão在Lampacão7西面，位于圣约翰8附近，离Amacão大约有16个légua de 20 ao grau9，是一个很繁荣的集市。不过我们到的时候已经是黄昏时分，显然已经太迟了，码头上已经没有很多人了，而远远地望过去，镇子上的房子里都已经飘起了炊烟。我们来时在马六甲买下的仆人穆罕默德告诉我们，平日里在集市上可以看到很多檀木售卖，这些檀木大多数都来自马六甲或者香料群岛的其他地方，Mantão的中国人很多都迁居到了马六甲，他们在这些地方收购檀木，贩卖给他们迁居Cantão10的亲戚，然后这些人再在Cantão收购中国内地出产的瓷器、茶叶，并将其交付给他们的亲戚在马六甲贩卖。

大副佩德罗听了穆罕默德的话，很好奇Mantão人的日常生活，执意要留在此处过夜。我看天色有些晚了，便依了他的话，停靠在了港内。佩德罗和我带着穆罕默德去了镇子上，穆罕默德说这里属于中国的Cuição11地方，这里的人们讲的语言和Amacão附近差异很大，和Amção12、Cantão一带的语言也有很多不同之处，我不说中国话，听不出来有什么不同，不过路上的行人见到我们并没有像之前在Amção那样指指点点，看来应该是有白人来过此地。我正想着，突然就看到镇子前面有一座小房子，顶上竖着十字架。是一座教堂。不过现在太晚了，教堂的大门紧闭着，考虑到明天正是礼拜日，虽然不知道在如此远东之地的基督徒还会不会严格地做弥撒，我还是提议Pedro我们先行回去，明天再来碰碰运气。

1585年1月13日

佩德罗一大早就把我从床上拉了起来。我起来的时候，港口已经很热闹了。我和佩德罗穿着斗篷带着穆罕默德刚一下船，便收获了很多指指点点。看来我之前的想法可能错了，这里的人们还是对白人不太熟悉。穆罕默德用马来话跟他们解释了一下来意，我和佩德罗便趁机罩着兜帽匆匆从人群中穿过，就听见后面他们在很大声地说着什么，但是没有人追上来。

循着昨天的记忆，我们找到了教堂。我们推门进去的时候，看见有几位中国老者正在祈祷。那几位老者对有陌生人来访明显感到很奇怪，停止了祈祷，看向我们这一边，为首的那位开了口，是西班牙语，问我们从哪里来。佩德罗告诉他我们从波尔图来，他便微笑地招呼我们坐下，并对另一位老者说了些什么，那位老者应了一声，便进了里面的屋子。

老人自称教名阿方索，1552年由方济•沙勿略神父13受的洗，后来沙勿略神父去世之后，他便开始在本地传教，然而一直都没有什么成效。不一会儿，进了里面屋子的老者端来了一些水和蛋糕一样的东西，我们就就着这些食物，开始畅谈起来。佩德罗问了他很多关于本地情况的问题，老人都一一回答了，老人也问我们葡萄牙国内的事情，我们知道的我们也都回答了。我是不太关心这种事，所以也就随便听着。

不知不觉，时间已经过了中午。穆罕默德突然推门进来，告诉我们有人昨天看到我们的船进了港，去Cuição城里报告了官员，而中国对外国人的入境管控很严，我们最好赶快离开。佩德罗却舍不得那些确实很好吃的蛋糕，老人见状便招呼仆人给我们装了一些。我们谢过了老人，便径直回到了港口那儿偷偷溜上了船，赶快启帆离开了Mantão港，向马六甲驶去。

Mantão，或即據漢語文獻稱的“萬灘”，按葡萄牙人的說法，和先前葡據的上川島以及同時期的澳門港不同，是肯定處於明朝朝廷治下的。根據“檀木”相關內容，可以推斷這一時期的該地區事實上很可能是檀木走私的據點之一。然而關於這一方面並無更多文獻資料，也只能從缺不表。

——摘录于Dr. Shz-hsiu Hsiung的笔记

黄公的名字是德宣，表字阿九，是台山万滩人，万历二十七年14出生。很小的时候就失去了父亲，九岁的时候母亲去世，只剩下了哥哥姐姐。哥哥的名字是德明，年轻的时候就因勇猛而出名。黄公十三岁的时候，哥哥把姐姐嫁给了万滩的张家人，带着黄公做了海盗。黄公在海上漂泊了很多年，行的是梁山好汉那样的事。黄公三十一岁的时候，哥哥去世，便有了退隐之心。三十五岁那年，迁居到了西贡，娶了钟氏做了妻子，生下了三个男孩和两个女孩，他们的后代就是河仙黄氏。在西贡定居之后，黄公经常怨恨叹气，说自己在南洋，亲戚朋友都死去了，只有一个姐姐还在万滩，但是路途遥远，也没有办法相见。后来黄公迁居河仙做乌贼生意。永历六年15，有从台山来的客人，黄公问他关于万滩的消息，客人不知道。黄公拿着地图仔细地对他讲，客人还是不知道，说万滩之类的地方，现在大概已经是一片荒滩了吧。黄公非常感叹，认为整个镇子已经遭遇了兵戈之祸，于是就作罢了。

——编译自Dr. Shz-hsiu Hsiung笔记中的《河仙黄氏宗谱》照片，原文为越南语。

《河仙黃氏宗譜》中的這一段，可能是現存少有的關於萬灘這一據點的漢語文獻記載了。我記得之前看到的其他資料還有提及崇禎年間臺山地區觀測到的所謂“長夜如晝”的現象，雖然沒什麼證據，但是這一事件，總覺得和萬灘消失之謎有什麼關係的，恨原件無以得見。大概需要去一趟大陸了，雖然也不指望能調查出什麼東西來，而且██也警告過我可能會有危險。實在不行，問問大陸那邊異學會的人，看他們有沒有保下來明末廣東那邊的黃冊16好了。

——摘录于Dr. Shz-hsiu Hsiung的笔记
