SCP-4338-2
SCP-4338 » SCP-4338-2

SCP-4338。此为无人机于2018/08/11发生的事件DN-4338-43中拍摄的视频中截取的静止帧。

特殊收容措施：3艘FMT安保巡洋舰，连同水上研究船海伦号将围绕莫哈岛构建四点封锁带。1在每个阴历月份开始时2，必须向岛内火山口中投入多种食品，包括：

  • 2200-2800kg饼干（如：奥利奥、奇宝牌巧克力薄饼干）
  • 1600-2000kg薯片（如：品客、乐事多力多滋和Ruffles）
  • 1300-1600kg各式巧克力、糖果、甜点
  • 1000-1400kg面点（如甜圈、丹麦面包、turn-overs）
  • 1200-1800kg经熏烤腌渍的猪或牛肋排
  • 1200-1800kg经调味的鸡翅
  • 1000-1200kg雀巢热口袋速食
  • 1300-1600kg冷冻披萨

这些供应是基于SCP-4338的持续要求制定，被接纳的概率未知。若食谱被拒绝，受任命的“领导人”应前往孤立地点，以免基金会财产损毁及额外生命损失。更多细节参见下文“4338-DN事件概览”。[正在修订。参见下文记录]

描述：SCP-4338是一具有智能的浮空火山岩，呈4.6km3大小的球形，居住在莫哈岛下方的岩浆中。多件文物和口传传说将此实体称为“Ōsobiue”、“Irn'acha”或“燃海之神”：在19个早期海洋文化群的创世神话—及灭绝事件—中扮演核心角色。

采访及事件记录19/08/2018 | Rowell / 4338
采访及事件记录19/08/2018 | Rowell / 4338

前言：在过去八个月里，SCP-4338的要求逐渐变得更为夸张，行为越发无规律。在谈话时，Rowell / 4338会向各方向吐出熔化的元素，必须采用高强度有机玻璃隔离板，且定期更换。表皮岩石生长率增大230%。

受访者：Rowell / 4338

采访者：主管Amelia Thornborrow

Thornborrow：你要求见我吗，4338？

Rowell / 4338：虫子！尔等还是不懂如何同神对话吗！？

Thornborrow：…致敬。噢4 3 3 8。您的仆从很荣幸受到召见。

Rowell / 4338：这就好了。本尊有新的要求。

Thornborrow：傻逼 我是说，您的仆从很荣幸听候。

Rowell / 4338：尔等要给本尊奉上尔等每日兽类之肋，份量要是上次供奉的双倍。

Thornborrow：您的仆-

Rowell / 4338：还有更多的酸果，裹上甜沙。

Thornborrow：您的仆从-

Rowell / 4338：还有那填着奶油的黄海绵！再行奉上！

Thornborrow：…

Rowell / 4338：…

Thornborrow：您的仆从。我会去-

Rowell / 4338：还有五倍的松脆吉百利，搅拌填满琥珀色-

Thornborrow：等下不行，不行我们讨论过这个的；这些是从很远的地方运来的进口巧克力。记得吗，我们同意过高级食品数量减少可以减轻现在的航运紧张—

Rowell / 4338：够了！尔等的“紧张”与本尊何干。尔等必得遵照我令，不然尔等和尔等的SCP都要烧成灰烬！

Thornborrow：听着这—你不能才过了三天就丢给我们—

Rowell / 4338：不能？不能！？ 尔等居然敢教训本尊什么能什么不能！？尔等顽抗已让我厌倦，AM EEL YA；本尊将不再忍受！

DN-4338-42：莫哈岛发生一次4.0级地震。所有人员撤离到附近船舶上，高速驶离。根据协议，Thornborrow主管留在原地。

SCP-4338经过45分钟没有出现。震动从持续变为周期性，表现为猛烈但微小的晃动。派出无人机拍摄低空视频，发现SCP-4338多次尝试均未能脱离火山口；和六个月前不同，现在的它已不具备合适的速度和大小。

安保巡洋舰翠鸟号不久后返回岛边，Thornborrow主管继续采访。

Thornborrow：4338，您的仆从想问为何您没有—嗯，“落实”您的威胁。

Rowell / 4338：…尔等让我开心了。

Thornborrow：再说一遍？

Rowell / 4338：尔等让我开心了，AM EEL YA。我要大发慈悲，尔等性命可得苟延。但勿要再质疑我，不然尔等所知就只有痛苦与死亡！

Thornborrow：…那好吧，您的仆从无比荣幸。我们继续？

Rowell / 4338：是的。本尊要吉百利-

Thornborrow：其实，我在想现在时机正好，该讨论下彻底修订我们现在的安排了。

Rowell / 4338：什么！？

Thornborrow：30%总量削减，附带替换—

Rowell / 4338：污秽！尔等就这么一心求死吗！？

Thornborrow：—减人力和安保，因为只有一人真正需要—

Rowell / 4338：够了！够了！够了！

DN-4338-43：再次侦测到地震，这次只有2.8级。采访时在场的安保人员准备撤离，但主管Thornborrow示意他们停下。SCP-4338再次未能离开火山口。和过往事件不同，只有5%的熔岩排出。

如预期，在DN-4338-43期间，Rowell/4338被观察到发出大喊、喷吐、对着束具挣扎。此行为持续8分钟，最后Rowell/4338精疲力尽，瘫倒在地板上。

Thornborrow：…完了吗？

Rowell / 4338：可恨。尔等要激起我的愤怒但-

Thornborrow：您的仆从不希望惹您生气，4338。如果您不高兴的话，我和我的人就先走一步—

Rowell / 4338：不！

Rowell / 4338仍在往上跳，试图将自己沿着桌边往上托，但只能跌回。

Rowell / 4338：不！尔等不能！尔等-本尊要尔等…我…我要…

Rowell / 4338慢慢躺回地面。

Thornborrow：…你准备好继续了吗，4338？

Rowell / 4338保持不动且沉默8秒，然后叉着手臂开口。

Rowell / 4338：是。

Thornborrow：很好。那么，关于这次修订？

通知：你正在查阅的文件自19/08/2018起已经过期。
