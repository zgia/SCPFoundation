SCP-4111

SCP-4111内部某洞穴。

项目编号:  SCP-4111

项目等级:  Euclid

特殊收容措施:  要制造SCP-4111-1个体，需获得三级以上权限。任何存在但尚未投入使用的SCP-4111-1个体必须被储存在安全的异常项目收容室中，除非向SCP-4111转移的行动被批准，否则不允许接触盐水溶液。

希望长时间在SCP-4111中探索的基金会成员需配备标准干旱气候紧急逃生背包。出于任何原因被困入SCP-4111的工作人员都应尽快被找回。

在SCP-4111内发现的任何其他人员将被回收，在被采访后接受记忆清除并重新回归社会。

描述:  SCP-4111是一个可通过特殊手段抵达的位置，目前被认为不在地球上1 。 该手段能够将多名人员（或物体）转移至特定环境下的某一区域 (详见 SCP-4111-1说明)。

SCP-4111整体位于地下，有可供呼吸的气体环境，表层土地多为凝固岩盐2，偶有高热熔融岩盐形成河流、湖泊与大片海洋。除熔融岩盐发出的光外没有其它可见光源，且进一步调查显示此地存在一高度未知的天顶，同样由岩盐构成。所有SCP-4111中的自然结构均为立方体结构，目前尚未在其中发现天然形成的弧线。

一个未激活的SCP-4111-1个体。

SCP-4111-1被用于将实验员或物体传送至SCP-4111。制作SCP-4111-1所用客体不必完全相同，但其填充物与形状需遵循随PoI-418被发现的指南之规定，该指南由Gemenskap Coalition组织3完成。制造SCP-4111-1个体的具体信息见附录4111-A。成功制造的SCP-4111-1个体在完成后短时间内会闪现黄光并发出特定响声。如将一SCP-4111-1个体浸入盐水溶液，所有与其直接接触的实验员和物体4（包括该SCP-4111-1个体）均将被传送至SCP-4111。

唯一从 SCP-4111 返回的方法是死亡。触发SCP-4111-1进入SCP-4111的实验员如在任何情况下死亡，将会与其直接接触的人员和物体共同传送回触发前之位置。实验员返回后，所有在SCP-4111内产生的伤病皆会消失，SCP-4111-1 个体也将出现在实验员附近。实验员将保持在SCP-4111内部活动及自身死亡的记忆。

勘探记录中发现了若干疑似不属于SCP-4111的物品，详见下方清单。该清单表明，在基金会发现SCP-4111前，已有人员多次进入SCP-4111并对其进行了探索。

显示从SCP-4111发现的物品列表
隐藏列表
  • 人类和动物骨架若干。
  • 装有价值不等原钻的木箱若干。5
  • 一包剩一半的黄金奥利奥。
  • 一部iPhone 4手机。
  • 一把M1911手枪。
  • 一把鹤嘴锄。
  • 一个带过滤机的人造水库及一个金属杯。6
  • 一个苹果园，以及数个腐败状况不同的苹果核。7

PoI-418图像

发现:  在9/28/20██，首个已知曾经进入过SCP-4111的测试者PoI-418，突然出现在2009年前后沉没于大西洋的████████残骸附近。 PoI-418的图像来自该海域一水下机器人。报道称，一男子在此次海域探测过程中突然出现，基金会获悉后立刻清除了相关目击者记忆并封锁消息。PoI-418的图像被封锁，记录该图像的水下机器人被销毁后沉入海中，基金会对外宣称这是由于探测过程中的机器故障。

以下为 PoI-418 康复后的一段采访。

展开采访-4111-██
隐藏采访-4111-██

<开始记录>

Agent Swietner咳嗽一声并坐下。

Agent Swietner: 好的，你觉得还好吗？考虑到报告中你获救时身处的环境，我们认为你不会想坐在任何金属或者盐制成的东西上了。希望那个坐垫能叫你满意。

PoI-418调整坐姿。

PoI-418: 非常感谢，事实上……我很高兴过了这么久终于看到了别的人类。

Agent Swietner: 没错，我们听说了你关于那个娃娃和……“盐的世界”的描述。

PoI-418看上去想要开口，Agent Swietner举手打断他。

Agent Swietner: 在你开始担心这个之前……我们相信你的话。嗯，我们不大相信你声称的那里真的是地狱，但我们觉得你的确去了某个别的地方。

PoI-418: 那你们想怎么称呼我去过的那个地方，除了“地狱”以外？

Agent Swietner: 好吧，毕竟我没有在那里呆过，所以我大概没这个发言权。但是现在具体怎么称呼你去过的那个地方并不重要。重要的是你是怎样返回的。

PoI-418: 我……我不知道是怎么回事。我甚至都不知道自己是怎么进去的。我突然就到了水下，然后我看到了那个娃娃，我一碰它眼前就全黑了。然后我好像看到了白色的地平线。我试着走向地平线，接下来我…… 就在那里了。

Agent Swietner: 我们认为你进入那里和那个娃娃有关，这点很明显，但我们需要知道的是你脱身的方法。你最后记得的事是什么？

PoI-418: 我最后记得的事？呃……我最后记得的是……是……

PoI-418看起来很为难。

PoI-418: 还有别人和我一起逃出来吗？

Agent Swietner: 有没有别人……不，从我读到的来看，当时除你以外没有别人。

PoI-418弓起背，开始拉扯自己的头发。

PoI-418: 不，不，不不不不……

Agent Swietner: 你是想说和你在一起的某人没有出来吗？

PoI-418揉着眼睛，长叹一口气。

PoI-418: 我需要回去。我需要那个娃娃，我需要回去。

Agent Swietner: 如果里面还有别人，我们会试图进入那里搜寻并带回他们。但在那之前我们要知道你是怎么回来的。

PoI-418叹气。

PoI-418: 我最后记得的事是我死了。Sydneigh和我正试着通过一条河，然后我脚下的地面突然崩塌了。

Agent Swietner倾身向前。

Agent Swietner: 悉尼？

PoI-418: Sydneigh。听上去像悉尼，但她是一匹马。我给她取的名字。

Agent Swietner: 你找到了……一匹马？

PoI-418: 不，是她找到了我。当我躺在那里的时候——听不到声音，半死不活，经受着压力变化—— 她来到我的身边，把我拖到了一个相对凉爽的地方。她是一匹好马，我需要把她带回来。

Agent Swietner: 好吧，可是你不就是因为死了才回来的吗？就凭她自己，一匹马能在那里活多久？

PoI-418: 她不会这么快就死的。她幸存了下来，我打赌她到那里的时间比我早很多。

Agent Swietner: 你怎么知道？

PoI-418: 她带我去了一个农场。呃，至少看上去像是农场。显然，有人试图在那里建一个。我想这是为什么她会出现在那里，不过我不知道她在那里呆了多久。

Agent Swietner: 好吧，如果你坚称你回来之前经历的最后一件事就是自己的死亡，那么我们很乐意在进行若干实验以后替你杀掉你的马。

PoI-418抬头看向Agent Swietner。

PoI-418: 不，这件事我想自己来做。你和她之间没有那种感情，如果她非得死掉才能回来，那么我得当那个杀她的人。

Agent Swietner: 我不确定有没有这个可能，考虑到——

PoI-418: 我比任何人都更了解那个地方。我在那里呆了，让我想想，一个月？我探索了整个区域，我知道要找她我们需要去哪里。

Agent Swietner: ……考虑到我们仍然不知道这是否是唯一返回的方法。

PoI-418: 怎么，你们还要做实验？我就是你们实验用的豚鼠，这我不在乎，只要你们能允许我去找她并把她带回来，你们让我干什么都行。

Agent Swietner和 PoI-418都沉默了片刻，接着Agent Swietner叹了口气。

Agent Swietner: 你听起来是个好人。我会看看自己能做什么，但我无法作出承诺。

Agent Swietner停了一下。

Agent Swietner: 在我们这个行当找到一个自愿的测试者并不容易，所以你的确可能如愿以偿。

PoI-418: 谢谢你。只要你确保我是最后进入那里的人，我会告诉你任何你需要的其它信息。

Agent Swietner: 我再说一次，我无法承诺任何事。

<记录结束>

自这次采访以来，PoI-418完整地回答了基金会提出的所有问题，并自愿参与所有要求出入SCP-4111的实验。在同意完成提取后接受记忆删除的条件下，他企图救援“Sydneigh”（也称PoI-418-H）的要求得到了批准。

附录4111-A: PoI-418康复后，基金会单独对████████发起了一次针对Pol-418图像的调查。在PoI-418被拍摄地点附近发现一SCP-4111-1个体。后续探测中共有四个SCP-4111-1个体被发现，均完好无损。此外，被发现的还有一本写有SCP-4111-1制造及使用方式的指南，部分书页浸水后遭到破坏。该指南被修复后，基金会得知在SCP-4111-1个体制造时有以下关键点:

限3/4111或更高权限浏览
权限等级确认
  • SCP-4111-1个体需由布、纱线或丝绸制作。
  • SCP-4111-1个体应具有人形的四肢。
  • SCP-4111-1个体应由盐、土壤、黑曜岩及灰烬填充。

基金会自此开始根据研究需求制造SCP-4111-1个体以进入SCP-4111。

对PoI-418-H的提取: 在11/16/20██，机动特遣队Epsilon-9 ("食火者")及PoI-418共同进入SCP-4111以找回 PoI-418-H。四位特工被选择与PoI-418同行，代号为Agent Pele，Agent Chantico，Agent Hephaestus及Agent Lalahon。

展开勘探记录
权限确认

勘探记录 EX-4111-27: 对PoI-418-H的提取
日期: 11/16/██
序言: 以下片段均摄自Agent Pele的随身相机。

<开始记录>

画面从基金会的安全测试设备切换到了一个黑暗的洞穴中。Agent Pele转向队伍中剩下的人。PoI-418正笨拙地调整身上的装备。

Agent Pele: 418，你还好吗？

PoI-418: 呃，还好，不用担心我。我只是搞不懂你们都是怎么适应这么重的防护服的。

Agent Chantico: 没有它我们都得热死在这里。穿着防护服移动很困难，但另一方面它也隔绝了热度。

Agent Hephaestus: 相信我们，我们四个已经在这里对它们做了够多的压力负荷实验。

PoI-418: 等等……呃……

Agent Chantico转向PoI-418。

Agent Chantico: 还有别的事吗？如果你在害怕，别担心，我们很快就会把你的小朋友从这里救出去。总之，不要着急。

PoI-418: 不是这样，只是……没这些装备我也在这里活了一个月，也没有因为高温死掉。见鬼，大多数时候我都穿着潜水服。

Agent Pele: 嗯，的确很奇怪。

Agent Hephaestus: 你肯定是远离了那些熔岩湖，对吗？

PoI-418: 我尽力远离它们。但如果我需要水源，我就得穿过一条熔岩河，除此以外大部分时候我都在阴影里活动。

Agent Hephaestus上前拍着 PoI-418的肩膀。

Agent Hephaestus: 那么这或许就是原因。

Agent Pele: 不管你那时候穿着什么，这些防护服的设计就是为了在极端环境下保持体温。你现在受的折磨已经够少了。

PoI-418: 听上去真是鼓舞人心。

Agent Pele: 事实上咱们现在的处境不容乐观。来吧，该前进了。

Agents Pele和 Lalahon领头，PoI-418在中间，Agents Chantico和 Hephaestus殿后。他们在洞穴里穿行，朝疑似出口的光源走去。

Agent Hephaestus: 所以，这就是出口了？一个活人也没有？难怪你最好的朋友是匹马。

Agent Chantico打了Agent Hephaestus一拳。

Agent Hephaestus: 嘿，你这是在干嘛？

Agent Chantico: 别那么混蛋，Hef。你无权指责他的做法。

PoI-418: 某种程度上，他是对的。我想在这地方看见别的什么人，这念头快把我逼疯了。只有Sydneigh总是陪着我，给我我需要的一切。

Agent Hephaestus: 真肉麻。

Agent Chantico又给了Agent Hephaestus的肩膀一拳。

Agent Hephaestus: 很明显，我只是开玩笑！我知道他什么意思……

Agent Lalahon: 出于好奇我想问，为什么是“Sydneigh”？

PoI-418: 什么意思？

Agent Lalahon: 有那么多名字可以选，为什么最后选了它？

PoI-418: 我想……在家里我有一只叫Meowster Chief的猫，几年前还养过一条狗叫Bark Obama。我给宠物取名字时喜欢玩点文字游戏。

Agent Hephaestus: 天啊… Bark Obama?

Agent Chantico: 好吧，没错，我没法替你说话了，这名字确实不适合一条狗。

PoI-418笑了。

PoI-418: 听着，我坚持自己取名字的规矩。这是我自己的事！

Agent Lalahon: 噢，上帝……

PoI-418: 嘿，这是你自己要问的！

Agent Lalahon: 我本来期待着什么有意思的回答，结果听到了几个我听过最糟的俏皮话。

队伍抵达了洞穴出口。他们面前有一条宽阔的熔岩河。Agent Pele看见了河岸线旁岩浆留下的惊人痕迹。她转向PoI-418。

Agent Pele: 418，这个地方你认识吗？

PoI-418看着这条河摇头。

PoI-418: 我……我不确定。我在这里的时候见过好几条河，也可能是同一条，只是它非常长。我没有画地图，但…… 好吧，我也没有画地图的材料。我只有一堆苹果和水。只有苹果和水，真的。

PoI-418紧张地笑。

Agent Pele: 好吧，如果你看到了任何能把我们带到目的地的标志，请务必告知我们。我们可能到过周围的什么地方，但你才是所有人里面在这里呆得最久的。

Agent Hephaestus: 再说了，那毕竟是你的马。

PoI-418: 没错……是的，我能做到。

Agent Pele转过头。队伍继续沿河岸行进。

Agent Hephaestus: 话说回来，你能拿一匹马干什么？你会骑马吗？

PoI-418: 要是我能出去，我立刻就去学。

Agent Chantico: 你在这里的时候从来没有骑过她？

PoI-418: 我怕我会骑着她直接撞进河里。

Agent Lalahon: 你不就是这样死的吗？我是说，你第一次到这里的时候。

PoI-418: 不完全是。我那时候走在她旁边，我正要走上一座桥。是天然形成的那种，所以它承受不住塌掉了，我猜。

Agent Lalahon: 天啊，第一次经历这个一定很可怕。

Agent Chantico: 而且你还不知道那之后自己能够回来。

PoI-418: 我说……你要是几周之前这么问我，我还有点没法接受，可是……我想想，我死过有，九次了？我原本可能只是个Costco（好市多量贩店）店员，但死过几次以后，现在什么都吓不到我了。

Agent Pele: 好家伙。其实我挺惊讶你能在那种环境里活那么久，你值得我们的尊敬。

Agent Hephaestus: 是啊，甚至有很多特工都没死过那么多次。可惜你做完这些就得接受记忆清除，当然你也可以加入我们，你会是个相当好的特工。

PoI-418: 嘿，很高兴能听到你这么说……真的。我一直觉得自己挺没用的，除了潜水什么也不会。而且潜水也不是我的主业，只是我假期里的休闲娱乐而已。

Agent Chantico: 你学得很快，这点很重要。而且你也明白，到了这种地方就得不怕死。

PoI-418: 好吧……我会考虑的。不过我不确定我是否适合待在一个秘密组织里——

Agent Chantico: 放轻松！我们不会强迫你加入的。不是所有人都能接受这种工作。

Agent Lalahon: 况且,你看到那些文件的时候说不定就会开始考虑退出了。

Agent Lalahon，Agent Chantico和 Agent Hephaestus笑。

PoI-418: 好了，好了。还是感谢你们提出这个建议。

队伍继续沿河岸前行，直到他们碰见一座天然岩盐形成的桥。远处有三根岩盐柱，其中一根塌掉了一半。

PoI-418: 等等，我想……我想就是它。

队伍停止移动。Agent Pele 转向 PoI-418。

Agent Pele: 你认得这个地方？

PoI-418: 是的，是的……毫无疑问……不，等等……

Agent Pele: 有什么不对吗，418?

PoI-418: 这些柱子和这座桥我都认识，再走几码就能到水库，但……

Agent Hephaestus倾身看向 PoI-418。

Agent Hephaestus: 这就是你从上面摔下来的那座桥？

PoI-418: 我告诉过你我没有从那桥上摔下去。那座桥自己塌掉了，我发誓是这样。我一下子掉到了河里。

Agent Pele把相机镜头向那座桥拉近。它完好无缺，没有任何损坏或者修复的痕迹。

Agent Pele: 418，你是从那座桥掉下来的？你确定？

Agent Chantico: 我说，毕竟你经历了很多事，死了很多次，明白我的意思吗？你可能把某些经历混淆了。

PoI-418 摇头。

PoI-418: 不……我发誓我是从那里掉下来的。

Agent Lalahon: 好吧，重要的是你能不能从这里找到你的马。

PoI-418: 要我说……如果这就是那座桥，我就会知道到水库和果园去的路。如果Sydneigh还在的话，她只可能在这两个地方之一。

Agent Pele走向 PoI-418，抓着他的肩膀走到最前面。

Agent Pele: 那就带路。我们都带着娃娃，即使发生意外也没事。

PoI-418: 好……嗯，那我们先去看果园。就在那些柱子中间。

Agent Hephaestus: 这些柱子已经塌了一根，把果园建在这里真够危险的，不是吗？

PoI-418耸肩。

PoI-418: 反正不是我建的。

队伍靠近远处的岩盐柱，和苹果园的距离逐渐缩短到约20米，PoI-418 放慢脚步停下来，身体前倾，手撑在膝盖上。

Agent Pele: 有问题吗，418？

PoI-418没有回答。他突然直起身体向前冲刺。

PoI-418: 我看到她了，我看到她了！她就在那里，看，你能看见她！

其他人试图跟上PoI-418。他直接跑向坍塌的柱子，他们靠近时隐约看到 PoI-418-H从柱子后面闪过，停在了一棵树下。

Agent Hephaestus: 天啊，伙计，慢点！小心那些柱子！

Agent Pele: 418，那些柱子不够结实，把脚步放轻点！

PoI-418脱下面具和头盔扔在一边。Agent Pele在他身后捡起他的装备。

Agent Pele: 418，不要着急，如果你犯了错，我们承担不起后果！

PoI-418: 我们可以！要是我搞砸了我们还能再回来！我不会浪费这个机会！

PoI-418到 PoI-418-H身边抱住她。PoI-418-H发出嘶鸣。

PoI-418: 噢，Sydneigh，我就知道我还能看到你！

PoI-418亲吻 PoI-418-H的口鼻。

PoI-418: 真是个好姑娘、一匹好马。你怎么样？

队伍中其余人放慢脚步，谨慎地靠近果园和 PoI-418-H。Agent Pele把面具和头盔递给 PoI-418。

Agent Pele: 你该庆幸这里的空气可以呼吸，418。难以想象……明明有大量火山中的气体排入空气……

PoI-418: 我得让她知道是我！我们戴着面具看上去都一样。

Agent Pele自言自语。

Agent Pele: 真搞不懂一匹马是怎么活下来的……

Agent Pele，Agent Lalahon和 Agent Chantico查看整个果园。

Agent Lalahon: 奇怪……

Agent Chantico: 怎么，我觉得他俩待在一起挺好。

Agent Lalahon: 我不是在说 418 和他的马，我是指这个果园。

Agent Lalahon指向一棵树。 Agent Pele看向它，表示它很正常。每个枝杈都生长着大量苹果,它们身上也没有异常。每过几秒钟就有一个苹果从树上落下。苹果树不是扎根在土壤中，而是直接长在岩盐上。

Agent Lalahon: 没有土，直接长在矿物表面。也看不到水源。

Agent Chantico: 也许是那匹马在照料它们？要不然就是……魔法？

Agent Lalahon: 现在看来魔法也不是那么不合理的解释，尤其我们还是通过一个巫毒娃娃进到这里的。

Agent Lalahon耸肩。

Agent Lalahon: 我不知道。Pele，你在录像吗？

Agent Pele: 别担心，Lalahon，我给果园录了像。研究人员会对这些感兴趣的。

Agent Hephaestus靠近了 PoI-418 和 PoI-418-H。

Agent Hephaestus: 所以这就是她了？

PoI-418抬头看着 Agent Hephaestus。

PoI-418: 在哪里我都认得她身上的白斑。

PoI-418-H 冲他嘶鸣。 PoI-418笑起来。

PoI-418: 我想她也认得我。

Agent Hephaestus看向倒塌的岩柱，小心地把手放上去。

Agent Hephaestus: 她活这么久真够奇怪的。你说她在你之前就在这里了？

PoI-418: 是啊……树上有那么多苹果给她吃，河那边有水源，还有这么大的地方可以跑动……

Agent Hephaestus: 我听说有时候马会靠吃人活命。也许就是因为这个这里才一个人都没有？

PoI-418局促地笑笑，转向Agent Hephaestus给了他一拳。他没有命中，反而打在柱子上。柱子发出响亮的嘎吱声，像是即将崩塌得更厉害。

PoI-418: 闭嘴，那只是个谣言。

大块岩盐从柱子上崩落下来，直接落向 Agent Hephaestus，PoI-418和 PoI-418-H。

Agent Hephaestus: 听着，你回去之后，趁着记忆还没被清除，去读读那个俄国士兵被马吃了的故事，听上去很疯狂，但挺有意思的。

Agent Pele: 418，Hephaestus，赶紧从那里离开！

Agent Hephaestus看向上方，快速从所站的地方闪开。PoI-418迷惑地看着 Agent Pele。

PoI-418: 等等，发生了什么？有什么——

几大块岩盐落到 PoI-418和 PoI-418-H头上。PoI-418立刻被打中，然后就消失在原地。PoI-418-H不见踪影。

Agent Hephaestus: 该死！

Agent Chantico: 呃，至少他们出去之后能联系上。这就说明Sydneigh被成功提取了，对吗？

PoI-418-H嘶鸣一声。

Agent Lalahon: ……操。

剩下的人靠近断柱搜寻 PoI-418-H。PoI-418-H似乎就在断柱旁边，情绪激动，四肢以扭曲的角度地张开。它不断嘶叫，但不管它怎么动作，那根断柱都纹丝不动。

Agent Hephaestus: 这他妈怎么回事？

Agent Lalahon: 看上去像是某种程序故障……

PoI-418-H僵住了，四条腿水平向两侧伸展。下一刻它又站在断柱了旁边，毫发无伤。它开始朝果园走去，还吃了一个树上落下的苹果。

Agent Pele: 这不像是程序故障，Agent Lalahon。

Agent Lalahon: 你想说什么？

Agent Pele: 你也看到刚刚的怪事了，这只是一个拙劣的程序，写它的人肯定连常识都不知道。空气坏境不符合常识、人可以穿过地面、反常的温度……

Agent Chantico: ……所以这整个地方就是一个程序？

Agent Lalahon: 那匹马只是它的一部分，对吗？

Agent Pele: 嗯，考虑到它没有和外界进入的人一样死亡然后消失，我想确实是这样。

队员们沉默地看着 PoI-418-H吃掉了更多苹果。

Agent Chantico: 所以……我们要干什么？

Agent Pele: 把这件事报告给基金会。他们会做好需要做的事。

Agent Hephaestus: 那么 418 呢? 我们怎么给他交代？

Agent Pele: 反正他自愿接受了记忆清除。即使我们告诉他他的马压根就没存在过，对他来说也没有区别。

PoI-418-H离开果园走上桥，队伍再次陷入沉默。

Agent Lalahon: 可怜的家伙。

Agent Pele: 本来可能更糟的。无论如何，那只是一匹马。

Agent Lalahon: 是啊，但……他为了找回她经历了那么多。

Agent Pele: 那不重要。他知道这件事的危险性，也知道这次行动不一定会成功。实际上，他因此帮我们更深入地了解了这个地方。

Agent Hephaestus: 别以为他是为了科学研究跟我们来的。

Agent Pele叹口气。

Agent Pele: 我们走吧。回去之后就知道是怎么回事了。

Agent Pele拔枪对准自己的头，然后开枪。

<记录结束>

机动特遣队Epsilon 9离开SCP-4111后将行动过程中的发现报告给了基金会研究人员。PoI-418先于他们返回后被护送离开测试场所，他未被告知SCP-4111或 PoI-418-H的本质。

PoI-418现居住于基金会监控下的华盛顿州的█████地区。到目前8为止，他完全安全且未受到心怀不轨者的侵扰。
