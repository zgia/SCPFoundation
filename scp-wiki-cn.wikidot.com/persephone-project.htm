拉撒路项目

项目代号：Lazarus（拉撒路）

项目编号：PRJOLM-14310193

权限与归档编号：NPF-00051473

首席研究员：K. P. Crow教授

项目目的：重新效验，进一步升级，利用SCP-244-ARC，使K. P. Crow教授能无限期受益于基金会。

SCP动用：

  • SCP-244-ARC - 作为基础构建
  • SCP-447 - 作为原材料
  • SCP-500 - 作为原材料
  • SCP-914 - 用于变更原材料
  • SCP-890 - 用于变更原材料和最终成品

原材料：

  • 一片SCP-500。
  • 由SCP-447所产的五百公升的粘液，作为生物防腐剂、消毒剂和润滑剂。
  • 一个心肺机
  • 一个血液透析机
  • 250公斤的防弹玻璃。
  • 一个安德森机器人神经适配器

完善与修改程序：

心肺机、透析机、防弹玻璃经由SCP-914精加工。安装这些装置可辅助呼吸并防止脑部/呼吸系统萎缩。

SCP-890协助进一步成型，使之与SCP-244-ARC底架融为一体。此为容纳SCP-447产生的黏液的必要之举。虽然黏液无法构成对此降ΩK的威胁，然其预防性能可抑制身体体质下降，防止进一步的肌肉骨骼疼痛的发病。

安德森机器人神经适配器将作为翻新版SCP-244-ARC的控制系统装备。此外，还需要一名训练有素的神经外科医生，将脑神经感受器附着在适当的科技上。

附加更易备注：

  • 移除过去安装的武器 - 不再需要
  • 移除紧急自毁系统 - 不再需要
  • 移除手动控制系统 - 不再需要
  • 移除空气动力推进系统 - 不再需要
  • 安装四个辅助肢体 - 以提高工作效率
  • 安装带有枕部支撑的陀螺仪摄像头 - 以提高可视性

备注：

项目计划书获批。- 2022年3月19日
翻新SCP-244-ARC完成。- 2023年9月21日
试验开始 - 2024年1月24日
试验结束 - 2024年10月20日
手术安排在2025年3月1日 - 2月19日
Crow教授要求推迟手术 - 2025年2月28日
手术安排在2025年6月12日 - 3月18日
Crow教授要求推迟手术 - 2025年6月11日
手术安排在2025年7月30日 - 2025年7月12日
Crow教授于2025年7月30日与设备连接。- 2025年8月1日
通过校准测试 - 2025年8月15日
进行维护 - 2025年10月23日

          项目代号：Persephone（珀耳塞福涅）

          项目编号：[未提交]

          权限与归档编号：NPF-00051473

          首席研究员：K. P. Crow教授

          项目目的：创造能够终结人类生命的实体、物体或事件

          需测试SCP：

          • SCP-053 - 为ΩK前致命性而选
          • SCP-049 - 为ΩK前致命性而选
          • SCP-2333 - 为ΩK前终结人类意识的能力而选
          • SCP-963 - 为ΩK前终结人类意识的能力而选
          • SCP-2935 - 为可能引发一场CK级重组事件的可能性而选

          测试程序：

          • SCP-049、SCP-053和1个SCP-2333样品将从收容室中取出并移至SCP-2935入口。

          你所选择的任何异常的收容失效都无法被忽视。实际上，我们需要同时获得所有这些异常。此外，2935入口埋在数吨混凝土下。我无法确定这要如何起效。

          不要在公开场合谈论这件事，把它放在心里。Alto最近变得很爱管闲事。
          —Jack

          没问题，我会把抢夺的细节交付你手。如果你有机会帮上什么忙，那就是现在。 —Kain

          • SCP-053将受到限制并暴露于SCP-2333的雾化溶液中。
          • 将SCP-963置于受试者皮肤上。
          • 如果SCP-049不自行开始进行手术，将引介并指示其进行手术。
          • 一旦操作开始，所有异常将移进SCP-2935入口。

          这是一个远景，但见鬼，现实本身似乎这么奇怪，这么不真实。我相信它就像物质和信息一样，有可能达到、呃，死亡的临界质量。如果我是对的，我们应该可以看到在跨越门槛的那一刻发生的一些事情。我一直在研究这个聚焦彩虹可作用于奇术、电力，甚至是气体能量上的功用，所以它应该能够接收到死亡能量，如果那玩意确实存在的话。

          不过老实说，我不知道到底会发生什么。不过由于我们都发现自己坚定地站在“除了往上走，别无他法”的阵营中，那在我看来，这就是我们最好的选择。 —Kain
