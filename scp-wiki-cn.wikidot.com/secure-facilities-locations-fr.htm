法国分部各类设施

法国分部各类设施列表

欢迎各位研究院在下方列出自己所研究的项目，所提及的工作地点与工作人员都是可选的。

本分部的设施运作，提名制度与本部基本相同，但是由于法国分部的存在，在此文档中与英语区的设施区分开来是必要的，以免在以后的时间里归档出现问题。

Sites

א Site-Aleph (维度研究与武装收容设施-Aleph)

主管 : Bruce Garrett

Site-Aleph曾经位于[数据删除]城市北部，于██/██/████被SCP-▉▉-FR引发的维度崩塌事故所摧毁，Site-Aleph目前正在进行大规模翻修，为此动员了制造部门（Site内运作）与DI&ST及大量的▉▉级与0级人员。尽管一些SCP的研究工作由于翻修工作被推迟，仍然有三分之二的设施能够投入使用。
该站点极为庞大，设施齐全，可为Site内所有研究人员提供住宿及其他服务，以及为需要大量研究人员的众多SCP提供收容措施。
外交事务部在该Site极为活跃，并在一个作为前D级人员的监狱处设立了行政办公室。
最后，Site-Aleph是安全与情报部门的枢纽，也是该部门近▉▉▉名成员的工作与生活地点
以下是收容于该Site的SCP: SCP-020-FR, SCP-019-FR, SCP-035-FR, SCP-006-FR, SCP-098-FR, SCP-074-FR, SCP-075-FR 与 SCP-080-FR。 SCP-140-FR正在收容中。

ה Site-He (Multi-sites de Confinement Provisoire-He)

该站点主要收容无法或不应移动的单个/多个SCP项目. 例如SCP-002-FR 与 SCP-010-FR

ל Site-Lamedh (气候研究与异常收容设施-Lamedh)

主管 : Jonathan Pierce

Site-Lamedh被设计成一个能够承受极端恶劣气候及高强度作战的传统混凝土地堡工事。目前正收容SCP-053-FR.

ᘸ Site-Kybian (仓储设施 -Kybian)

主管 : Kathryn Mayers

仓储设施-Kybian的前身是一个地下仓库，现已作重新翻修，只用于存储小型物件，无需特殊收容措施与特殊安保程序。现收容SCP-134-FR 与 SCP-012-FR在内的SCP项目。

י Site-Yod (武装收容设施-Yod)

主管 : Alexa Earl

Site-Yod位于[数据删除]镇，专门从事Euclid级与Keter级SCP项目，阻止其进行掠夺性行为。该Site是基金会最为敏感的Site之一，尽管整体设计合理坚固，安全措施齐全，但依然是死亡率最高的Site之一。因此，该Site又被选作机动特遣队Delta-4的永久驻地，并作为机动特遣队队员与特工的集训中心。

现Site内收容的项目有：SCP-023-FR, SCP-025-FR, SCP-049-FR 与 SCP-062-FR等等。

ז Site-Zayin (武装收容设施 -Zayin)

主管 : Geoffrey Doyles

武装收容设施-Zayin位于一个废弃的军事基地，其设立目的是收容具有军事外表或者需要以军事手段收容的SCP。因其距离Site-▉▉有▉▉▉▉公里，许多人使其为Site-▉▉的一个附属设施。

目前收容的SCP有SCP-125-FR.

ב Site-Beth (人形生物收容设施-Beth)

主管 : Théodore Harris

Site-计划-Beth最初是一个收容位于中东的异常人形个体的计划。
现Site-Beth是一个收容有多个Safe与Euclid级人形SCP与多名基金会员工工作与住宿的场所。
现收容的项目有 SCP-084-FR, SCP-104-FR 与 SCP-180-FR 以及 SCP-178-FR-3.

ע Site-Ayin (空间站设施 -Ayin)

主管 : Garry Rugaini (每年只有3个月于Site内停留)

Site-Ayin是一个静止于地球东经线3°上空的空间站，用于收容需要被收容在真空中的实体。目前该Site仍处于建设阶段，计划中仅对SCP-086-FR进行收容。仅有获得O5许可的人员才可通过太空胶囊运输至Site-Ayin。

ת Site-Tav (空间仓储设施 -Tav)

主管 : Juste Ivesern

Site-Tav是专门用于收容属于任何与同行组织“光明骑士团”有关的物件的站点。该Site曾用于收容SCP-056-FR以及其他一些文档。在收容期间基金会便挪用了此站点。

ג Site-Ghimel (收容与仓储设施-Ghimel)

主管 : Ismaël Saïdi.

Site-Ghimel被设计为一个能承受恶劣天气，高温以及高强度作战的传统混凝土地堡工事，在地下共拥有14层的作业区域，且无任何外部设施。该站点位于撒哈拉沙漠中心地带，以更有效地收容对温度与湿度极其敏感的异常物品。为了实现该功能，整个设施安装有先进的通风系统、温度控制系统与紧急除泥沙系统。目前收容的SCP有SCP-166-FR.

Zone1

☉ Zone-Shell (维度研究与收容区域-Shell)

主管 : Alison Per

Zone-Shell专门用于SCP-123-FR的监视与维护，作为“蠕虫之洞”计划的一部分。目前由▉▉▉执行，O5-2与O5-3批准，由Vleken医生和Ezcyo及其自身的安全部队以及机动特遣队 Phi-23和Tau-17组成的研究小组管理该Zone。该Zone位于地下，为保持最佳研究活动效率，其将远离人口密集地区，以尽量减少附带损害。

צ Zone-Tsade (武装收容区域-Tsade)

主管 : [数据删除]

Zone Tsade用于收容具有空间属性且具有极小破坏力的SCP项目。

随着SCP-▉▉▉-FR的出现，以达到最佳收容效果为由，Zone-Tsade被扩建且高度军事化，成为武装收容区域。

在一次核爆中，Zone-Tsade被彻底摧毁，现正商议寻找与重收容先前的SCP的任务。

ק Zone-Qof

主管 : [数据删除]

Zone-Qof最初设计用于收容一定数量的低威胁度SCP，但在后来的时间里医疗中心扩建，以更好地收容SCP-091-FR.

ם Zone-Mem

主管 : [数据删除]

Zone-mem位于大西洋的一个海上平台，其作为法国分部的海军基地的同时，也收容少部分异常物品。
