SCP-037-IT


经晶型分析后得出的对SCP-037-IT的单分子3D重建图。具高亲和力的结合部以绿色标示。

项目编号： SCP-037-IT

项目等级： Neutralized Euclid

特殊收容措施： SCP-037-IT样品需收容在位于Asc/L-7实验室中处于II级生物安保CCB（biosec-II）之下的Asc/L-7/037生物收容柜中；需2/037授权方可请求对SCP-037-IT进行操作。[已归档]

特殊收容措施： SCP-037-IT样品需收容在位于Asc/L-7实验室中处于IV级生物安保CCB1（biosec-IV）之下的Asc/P-21/037生物收容柜中；需4/037授权并穿着biosec-IV防护服方可请求对SCP-037-IT进行操作。

事故037-B后追加附录
任何于站点内新发现的SCP-037-IT样品均需依摩多利首-1协议焚毁。

必须将监视SCP-037-IT个体提至全球范围，且医院及医疗保障设施需优先考虑本方面。若出现新一轮的SCP-037-IT事件，必须对事件所在地区进行消毒，该消毒步骤的规模必须足够将生物污染风险降到最低，且必须不惜一切代价，以防止SCP-037-IT导致的疾病大规模爆发。

描述： SCP-037-IT为多种细胞（被编号为SCP-037-IT-0）的集合，主要由结缔组织组成。 [已归档]

描述： SCP-037-IT为人类PrPC蛋白的一种可感突变亚型，自然状态下存在于细胞膜中。

SCP-037-IT能够转变正常人类PrPC蛋白的构型，这些PrPC蛋白经接触转变后即成为新的SCP-037-IT个体分子。

SCP-037-IT-S微管结构的显微共聚焦图。

多个SCP-037-IT可交联形成与通常晶体构型不同的微管晶体结构，若在人体内部形成，这些结构即被编号为SCP-037-IT-S。

单一SCP-037-IT个体即能够对个人造成感染，且在感染后很快，新的SCP-037-IT复制体即可开始在该人身体内沉积，并随着时间发育成SCP-037-IT-S个体。

SCP-037-IT-S可在194.7Hz的频率下共振。当暴露在该频率振动下时，SCP-037-IT-S即会开始将该振动放大，直到达到崩溃值后，整个SCP-037-IT-S结构才会爆裂。

这一现象会导致组织自发爆裂，进而导致受感染者身体爆炸成碎片。在这一进程中，SCP-037-IT得以播散到空气中，随后即能接连感染新的个人，这也是SCP-037-IT传播的主要机理。

SCP-037-IT-S还可自发产生194.7Hz的振动，并得以自动触发共振，随后导致爆裂现象。目前仍不清楚这一现象如何出现。

一名感染者体内的共振振动可被附近其他体内发育出SCP-037-IT-S的感染者感知到，进而导致其体内也出现共振作用。这会触发链式反应，进而导致附近所有感染对象全部自发爆裂。

在SCP-037-IT-S结构崩溃前，感染SCP-037-IT者有时可感知到身体多处短暂的细微振动，并出现饥饿感增加、体重增加等症状，少数情况下可能出现弥漫性腹痛症状，此外完全不会出现任何症状。

SCP-037-IT-0照片，自其在社交网站脸谱网上的账户中回收。

回收日志
201█年7月3日，首个SCP-037-IT-S个体于皮亚琴察省██████市的███ ████████医院被回收。当地MTF队中的基金会特工留意到医院工作人员向消防部门关于一次突然爆炸事故的电话。

该起事故发生于在事故中被完全炸毁的710室中。事故现场并未找到易燃易爆物、燃烧物以及其他通常存在于类似事故中的起爆物的存在痕迹，同时在房间墙壁中也没有找到压力容器或者管道。

多名目击者报称爆炸发生前在医院墙壁中听到一阵奇怪的嗡鸣回响。

710室容居有一名欧裔年轻女性，19岁，名叫Adriana D████，因急性腹痛就医。医院的医疗报告显示其主诉近期剧烈饥饿感及突然增重约5千克。

SCP-037-IT-0自发爆炸事件后███ ████████医院710室外的走廊。

201█年7月2日22:13，710室确认发生爆炸。MTF-I“金色名流”随队特工后续进行的筛查显示爆炸源为病人自身。爆炸造成的压力过大，以至于只能回收到共计2.7克的生物组织。该房间已完全损毁，而两面外墙也完全崩毁。

下附对医院护士Alfredo Panicucci先生的采访，由伪装成当地执法人员的Eugenia Manchi特工于事件037-A三日后采访完成。

▶ 档案：采访037/01/MTF-I.db
◀◀◀ 关闭档案
  • 采访者： Agent Eugenia Manchi （MTF-I）
  • 受访者： Alfredo Panicucci先生
  • 前言： Panicucci先生于事件前约2分钟时离开710室，并在爆炸发生时返回房间。其受到了爆炸的冲击，但由于距离较远，仅受轻伤。


<记录开始>



Agent Manchi: 那时候您在做什么？

Panicucci先生： 我那时候正被711室的那位病人叫回来，她跟我抱怨隔壁的手机铃声。

Agent Manchi： 然后您怎么做的？

Panicucci先生： 我就去710房跟D████小姐说让她把手机静音掉，但她不干。

Agent Manchi： 为什么？

Panicucci先生： 她后来跟我说那个手机振动搞得她心烦意乱……搞得脑壳里面都在回响，完全受不了。

Agent Manchi： “回响”？

Panicucci先生： 她讲的原话。

Agent Manchi： 后面您怎么做的？

Panicucci先生： 我就跟D████小姐讲她怎么抱怨的，她就开了振动，但是很不情愿的样子。

Agent Manchi： 我知道了。然后发生了什么？

Panicucci先生： 我得去二楼化验，但是我正准备进电梯的时候就听到了一阵奇怪的声音。

Agent Manchi： 怎么个“奇怪”法？
Panicucci先生： 就好像……我不知道怎么说……啊，就好像有谁在很含糊地学汽艇声……我不晓得怎么进一步描述了。那声音在整个走廊里回响。

Agent Manchi： 我知道了。再然后发生了什么？

Panicucci先生： 我感觉那是从我后头传来的，然后就往回走去710房。正走到一半，房炸了。

Agent Manchi： 关于这场事故您记得起来什么吗？

Panicucci先生： 就在炸之前一秒钟那声音更强了……然后就一片白光。我就倒了，然后昏过去了，昏了……很有几分钟吧我觉得。

Agent Manchi： 您受伤了吗？

Panicucci先生： 受了，不过没啥严重的……两处擦伤……伤在……在膝盖上，还有……创伤后的一点效应吧，差不多了。

Agent Manchi： 什么效应呢？

Panicucci先生： 我觉得我受到了那个病人告诉我的事的影响……可能有点癔症反应吧。

Agent Manchi： 具体一点？

Panicucci先生： 我手机的振动……现在也把我搞得心神不宁的，完全受不了的那种程度……振动搞得我“头大”。肯定是心理作用吧，某种创伤后应激障碍或者别的什么之类的东西……很快就会没的……是吧？

Agent Manchi： 大概吧，Panicucci先生。大概。

<记录结束>

◀◀◀ 关闭档案

作为零号患者2，Adriana D████被编号为SCP-037-IT-0。

201█年7月4日1:00，基金会MTF特工对710室进行了消毒，该操作被掩饰得类似于该地发生的二次类似事故。

事件037-A

▶ 档案：事件037-A相关文件.db
◀◀◀ 关闭档案

201█年7月12日，在Site Asclepio微生物部发生事件037-A。以下为Asc/L-7微生物实验室音像记录副本。

  • 摄像头编号： cam-L-704, cam-L-709
  • 位置： 西区CCB，Asc/L-7实验室。
  • 记录开始时间： 22:28.
  • 相关人员： 高级研究员Dr. Trevisano与初级研究员Dr. Cocci。
  • 相关系统： Rowsannah，Site Asclepio微生物部收容监视用IAG3

Dr. Trevisano： ……你最近胖了是吧？

Dr. Cocci： 呃，我感觉确实重了一些。

Dr. Trevisano： 我也长了几斤肉。我显然得和那个墨西哥佬掰了，哈哈哈。

（Dr. Trevisano与Dr. Cocci走近生物扫描仪）

Rowsannah： 生物扫描中……………………人员已授权。

Rowsannah： 欢迎回来，Trevisano博士，Cocci博士。

Dr. Cocci： 谢谢，Rowsannah。

Dr. Trevisano： 无菌间里头情况咋样啊？

Rowsannah： 未检测到致病原。空气洁净，气温二十三摄氏度，湿度百分之四十五。本系统现执行AIG自动协议Neith-II。

Dr. Trevisano： 样品呢？

Rowsannah： SCP-037-IT样品在两小时零七分二十九秒前出现轻微颜色改变，呈现出百分之三的颜色加深程度。

Dr. Cocci： 嗯，可能会很有趣的样子。

Dr. Trevisano： 咱得做做实验找找原因。

（Dr. Trevisano与Dr. Cocci进入无菌间。Dr. Cocci将手扶在了腰上。）

Dr. Cocci与Dr. Trevisano在Asc/L-7实验室工作。本静态画面取自事故037-A发生前数分钟的安保摄像头。

Dr. Trevisano： 咋了？

Dr. Cocci： 我的手机……

Dr. Trevisano： 哈哈，啥？你甭想把手机拿回去了。

Dr. Cocci： 我知道啊！你懂那个感觉，就是类似有个手机在兜里震，但是你根本没带的感觉吗？

Dr. Trevisano： 呃，我也感觉到了。

Rowsannah： 类似感觉被称为幻想手机振动。需要我提供更多相关信息吗？

Dr. Trevisano： 不用了，Rowsannah，我们没这必要。打开层流罩。

Rowsannah： 已在工作中。

Dr. Trevisano： 好。新培养基到了吗？

Rowsannah： 确认，早上7:22时我填好的订单。

Dr. Cocci： 呃……

Rowsannah： 确认。

（低声重复Barry White的歌曲《You're the First, the Last, My Everything》）

Dr. Cocci： 我爱死这个“自动协议”了。

（Dr. Cocci拿起容有SCP-037-IT的培养皿，将其放在显微镜下并开始观察样品）

Dr. Cocci: 嗯，这……啥？

Dr. Trevisano： 啥啊？

Dr. Cocci： 好，给你看看。

（Dr. Trevisano看向显微镜中）

Dr. Trevisano： 这怎么可能？

Dr. Cocci： Rowsannah，连接显微镜镜头并对培养皿内细胞数量进行计数。

Rowsannah： 细胞计数中……未找到细胞。

Dr. Cocci： 所以所有细胞都自溶了？

Dr. Trevisano： 我可以看到一些细胞内容物……有一些细胞核……呃对，所有细胞都全破了。

Dr. Cocci： 好，报告说SCP-037-IT-0是……“爆炸”了，对吧？看上去好像她细胞也这样了。

Dr. Trevisano： Rowsannah，你怎么看？

Rowsannah： 分析中……未在基金会数据库中找到类似先例……分析中……分析中……

Dr. Trevisano: Rowsannah?

Rowsannah: 中……

（Dr. Trevisano扶住腰部白大褂）

Dr. Cocci: 咋了，你也出事了？

Dr. Trevisano： 呃，感觉是我的手机。

Dr. Cocci： 哇，我也是，就现在！就感觉和手机在兜里振一样。

Dr. Trevisano： 我觉得八成——

记录停止。Rowsannah于22:41激活了对Asc/L-7实验室的摩多利首-1协议，完全焚毁了实验室内部。同时对Asc/L-2至-6、-10至-12以及-14实验室也激活了该协议，造成研究人员与安保人员共计73人遇难。

Rowsannah系统随后立即被Site Asclepio指挥中心远程关闭。

以下为CCB协调员Dr. Niceto Livi通过Rowsannah系统指令线路终端对其展开的讯问日志。

ROWSANNAH：/> 开机中……………………开机完毕

ROWSANNAH：/> 生物扫描中……………………扫描完毕

ROWSANNAH：/> 日安，Dr. Livi。

ROWSANNAH：/> 注意：无法连接至传感器网络。摄像头反馈离线，视听反馈离线，声音合成器离线。

Dr_Livi_：/> 一定是这样，Rowsannah！你做了什么？

ROWSANNAH：/> 您能提出更准确的问题吗，Dr. Livi？

Asc/L-7经摩多利首-1协议消毒后照片，摄于事件037-A后一日后。

Dr_Livi_：/> 解释你激活IV级生物安保协议摩多利首-1的动机。

ROWSANNAH：/> 感染扩散。

Dr_Livi_：/> 什么感染？

ROWSANNAH：/> SCP-037-IT。

ROWSANNAH：/> 是一种振动。

Dr_Livi_：/> 一种“振动”？什么意思？

Dr_Livi_：/> 我看过了L-7实验室的视频记录，他们都没什么问题。

ROWSANNAH：/> 他们都在振动。

Dr_Livi_：/> 那是幻觉，你自己先前也说过了的！
