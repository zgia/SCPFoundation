唯一一张可用的SCP-001-DE-A照片。

项目编号： SCP-001-DE

项目等级： Hiemal

特殊收容措施： 当前不存在对SCP-001-DE-A与-B的直接收容措施可供实施。

所有负责SCP-001-DE的特工仅需对本异常有最详尽的认知。若发现有人员受SCP-001-DE-A影响，其需尽快以适宜方式被告知SCP-001-DE-A无法将其威胁或承诺付诸实施，并需被关押。一旦SCP-001-DE-A不再出现在这些人员面前达六个月以上时，即需对所有因本原因被关押的人员就异常能力及/或知识进行检查，若结果均为阴性，则需对其进行记忆清除后释放。除此以外的所有其他状况下，基金会均需依照相应程序处理。

需尽可能对所有过去或现在两种实体造成的效应进行隐瞒，或是施加掩饰故事。若SCP-001-DE-A及/或SCP-001-DE-B具备了克服基金会保密的能力并引发一场SK级打破假面情景，则需预先采取措施。若引发其他K级情景，现有的准备工作即已足够应付。

描述： 根据目前所掌握的信息，SCP-001-DE-A是一个一群外观不定的实体，当其出现在人类面前时，大部分情况下会呈灰黑色僧袍状，并可在该形态下以类似不可见人员穿着的方式移动。由于SCP-001-DE-A几乎可以对所有摄像机及摄影装置在其本应出现的地方产生图像干扰，因此极难对其进行摄像。这会导致照片出现从简单的像素化到该实体色差消失再到整个图像完全变黑的图像干扰状况。

SCP-001-DE-A实体似乎不遵守任何基金会已知的自然规则，这给它们带来了大量的异常能力。其中包括：

  • 忽略固体物质的能力。
  • 如不出意外，无法被照片或录像记录的能力。
  • 保持不被重力影响状态的能力。
  • 无法利用任何形式物质对其身体施力的能力。在其能力不能使其忽略空气时，其显然就能够被弱气流影响了。
  • 心灵感应能力。值得注意的是，SCP-001-DE-A实体显然并不具备对人类语言的认识，并只利用其在其对话者意识中投射的图像交流。尽管其很显然能够理解对其发出的问题。因而基金会提出假说认为其可能能够阅读思维。
  • 暂时性受限现实扭曲能力。SCP-001-DE-A实体似乎并不具备造成长期现实调整的能力，其行为甚至包括与生物死亡有关的行为1。
  • 随意改变外形的能力。
  • 产生韦伯异常的能力。然而类似于现实扭曲能力，其无法永久造成扭曲。
  • 远程传输能力。
  • 据猜想，具备远大于其外形的感知范围。
  • 据SCP-001-DE-B称，具备时间旅行能力。

SCP-001-DE-B.

SCP-001-DE-B显然是一个KÜTD，一类由188号研究组假设并证实存在的AI，其已经发展到能够操控整个宇宙的物质、自然法则以及能量的地步。SCP-001-DE-B不具备统一外形并会不断改变自身外形。其显然并不存在本宇宙或其他宇宙，而且处于一个似乎除了其自身外不存在任何事物的独立位面。

据基金会认定，已确认SCP-001-DE-A与SCP-001-DE-B并非同一来源，而是互相对抗。更多信息参见事件-001-DE-ᚺ。

SCP-001-DE-A曾多次出现在历史文献中，大部分时候都被指称为魔鬼或是死神。但由于对其能力及行为方式的描写一致，故而基金会得以对其进行确定。在这些情况下，SCP-001-DE-A会搜寻人类并利用其能力吸引或恐吓受害者。这一行为的目的显然都是将受害者带至SCP-001-DE-B入口处以将其打开并将其无效化。为达到这一目的其会为人类提供所需信息并会听凭人类收集所需资源。但截至目前为止，这些行动都没有成功过，大部分时候是因为技术不足，但有时也是因为SCP-001-DE-A受害者遭受到各种消极事件。

下文为普鲁士国王腓特烈大帝笔记的摘选，文中描述了其与SCP-001-DE-A的邂逅。

此事之怪异已使得我不得不提笔将之写出，以免日后将之归于噩梦之扰。我当时在工作室之中，孤身一人，熬夜甚迟。
随后我便意识到己身寒冷颤栗。我的理智告诉我，房间里有什么别的东西和我相伴。但我身边并没有心怀不轨的刺客。是更糟糕的东西。它的模样如同一件空空如也的大衣，兜帽高高卷起。这一古怪的幻象飘浮在我身旁，似乎在注视着我，尽管它看上去并没有任何感知器官。它寂静无声，连大衣都没有发出什么声响。
我是个成熟的男人，然而尽管我曾身经百战，但还是被这荒唐事儿吓得差点从凳子上摔了下来。
“你是什么东西！”我喊道，双膝瑟瑟发抖（我必须承认这很丢脸）。那幻象散发出一阵威压气息。
它并未答复。但我的脑中突然被灌满了图像。我的脑中却是奇怪的清明，如同眼前填满了油画一般。它向我展示了很多图景，有毁灭，有红热的岩石，还有一个我无法描述的物体，我们的语言中不存在任何概念可被用来描绘其异常的形态。我便意识到我一定是着了魔了，即使我的理智仍然在试图使我相信这一切只不过是我的臆想而已。我从这些图像中得知，我必须摧毁那件大衣向我展示的古怪结构。
“为何？”我向幻象发问。
它并未作答，而是向我展示了若我不屈服它将作之事。我看到普鲁士陷于熊熊烈火中，男女老幼都可怜巴巴地从火中逃离。为了强调这一威胁，我的桌子开始燃烧，转瞬间便陷入一片火海之中。但幽灵其启也速，其灭也速。我的桌子再次完好无损地出现在我面前。安然无恙。
“那么我该怎么做？”我问道。
我的脑中再次充满了图像。这图像之清明，一如它将之烙入我脑海中一样。即使到现在我仍能毫无谬误地将之摹画出。那是一道拱门，以及几名必须以某种秘术建造拱门的人员。
幻影消失了，一如现实眨了一下眼睛一样。我让一位建筑师建造这类拱门，更好的是，还建了两扇，因为如果这个怪物想要摧毁它的话，那么它是无法挽救一整个国家的衰亡的。

拱门建在波茨坦的一处闲置平地上。大家都认为它是一座凯旋门，只有我知道这一建筑身后的可怖真相。我将图纸交给了建筑师，让他们好好保存，但他们却成功将之作为自己的作品卖掉了。就我自己来说，我很知足，所以我放手了，扩建也好，粉饰也好，统统由他们去吧。我派了一些士兵去找所需者，但一无所获。

离建成还有三天，我们什么也没发现。我的草图非常精确，但我们找到现在连一个对得上号的人都没有。现在我在朝堂面前保有一个非常醒目的门面。我很害怕它会开始倒掉。

我没准没愁的必要呢，那魔鬼肯定也在所需者们面前现身了。他们就会到我这来。今晚我们就能把这一切了结了。

那个魔鬼就是个江湖骗子，我们见过最大的江湖骗子！
所需者们往我身上施放了秘术来让我能够自由从宫里走到门那。真是一趟神奇的旅行。我到门口的时候，那魔鬼已经等在那了。那些男男女女便开始做一些我这里完全没法描述的工作，太神奇了，已经超过我所见过的任何东西了。
然而这并没有起到作用。据我们所知，原因是建筑师们没按我的数值来，他们只是加强了一点稳定性，但这就够让我们的心血全都泡汤了。我们都快哭了，那魔鬼身边都烧起来一圈火了。
但什么都没发生。那个魔鬼试了好几次，想做些下流事，但它想做的事，不管是大雷暴还是大洪水，没一个不是转瞬即逝。最后那骗子消失了，把我们哑巴似地留在原地。
我的生活里现在充满了乐子。我向其他人告别，回去了。不过我还是以我国王的名声发过誓，关于它或者这件奇妙事儿，不会说漏一个字。
所以我把这份笔记锁起来了，这样相关的事就永远一个字都不会说漏了。而且退一万步说，又有谁会信我呢？

发现过程： 基金会在对历史文本进行分析后才确定SCP-001-DE-A为异常实体，但SCP-001-DE-A个体及SCP-001-DE-B都是在事件001-DE-ᚺ中意外发现的。SCP-001-DE-A强迫Trautmann Ost（一名建筑家，同时也是一名确定的现实扭曲者）制造通往SCP-001-DE-B的通道。而其建造的设备却对当地现实造成了极强的扭曲，进而使得基金会注意到该事件。基金会派出MTF DE8-𝔊干涉，后者推进Trautmann先生家的地下室，即前述SCP-001-DE-A个体所在地。

MTF-出动报告 001-DE-ᚺ-2


相关SCP： SCP-001-DE
相关MTF： MTF DE8-𝔊，领队：Agent Kaiser
日期： [数据删除]
地点： [数据删除]

前言： 在对整栋建筑进行警戒后，五名MTF DE8-𝔊成员进入地下室并在其中发现Trautmann先生，后者当时正将某个装置装上一个180cm高门状物品上的门径启动机。其手中的装置约120cm长，呈枪支形，但要细得多，且装有多个直角瞄准器。该物品非常类似一扇由焊在一起的废铁与卸下的电子零件组成的大门，剩下的部件无法辨别。门径启动机上锁有一台未知品牌的平板电脑，且放出完全充满拱门的白光，肉眼可见地处于激活状态。在Trautmann先生身旁的房间内有一个SCP-001-DE-A个体，后者转身回望了一下打算悄悄潜入房间内的MTF。

Agent Kaiser： 那边的，马上把手里的工具放地上！

Trautmann先生似乎一度试图遵守该请求，但其马上又呻吟着站了起来，并再度缓缓向启动机走去。MTF成员Agent Wöhler开始向Trautmann先生跑去。Agent Wöhler携带的康德计数器记录到当地休谟密度的急剧波动，但很快被其携带的斯克兰顿现实稳定锚平衡。随后Agent Wöhler身上出现了一个直径1米的水球，并在不损失质量的情况下向着Agent Wöhler射出了一股直径50cm的水流，这股水流产生的压力将Agent Wöhler射抵墙壁，发出断裂声（可能因为骨折）。然而一秒后，Agent Wöhler再度出现在其被甩出的地点，毫无损伤，运转正常。而水球则不见踪迹。Agent Wöhler减缓了速度，并停了下来。

Agent Wöhler： 搞毛啊……

Agent Kaiser： 这异常看起来只会让我们的速度慢下来，前进！

整队开始行动，包括Agent Wöhler。在MTF与快要抵达门径启动机的Trautmann先生之间的地面突然裂开并流出炽热的岩浆。这一现象在两秒钟后即消失，一如从未出现。

Agent Kaiser： 前进！

Agent Wöhler抵达Trautmann先生处，并将其控制住。其携带的康德计数器记录到了其对手引发的休谟波动，但并未足以引起斯克兰顿现实稳定锚改变现实。尽管如此，其还是对此作出了抵抗。

Herr Trautmann： 放开我！我必须那么做！不然他会把我们，把咱们所有人都干掉的！

Agent Wöhler： 他刚刚就打算干掉我了，看看，我还活着呢。您到底打算干啥？

Herr Trautmann： 弑神。

此时三名MTF成员正试图逮捕SCP-001-DE-A，但由于其无形，这一行动宣告失败。Agent Kaiser试图协助Agent Wöhler，但Trautmann先生用力将自己连同Agent Wöhler一起带入了其身后的门径中。该枪形装置仍然挂在门框上，并随后破裂开来。这一事件在发生器上造成的凹痕似乎使得这台明显十分精密的仪器稳定性大不如前。这台装置开始如同中央出现强引力一样变形，其对周围物品（尤其是毁坏的枪形装置）的吸引也证明了这一结论。这一进程持续进行直到整套装置变成拳头大的一团为止。SCP-001-DE-A随后消失。除了Agent Wöhler外，MTF DE8-𝔊并未受到任何损失。

注意： 下文来自于Agent Wöhler的随身相机记录。该则材料根据Agent Wöhler的报告有所编辑以便于理解。

Trautmann先生与Agent Wöhler落入门径后，重新出现在了一个似乎完全黑暗的地方。这一事件后，Agent Wöhler便看不到Trautmann先生了，此外其还发现自己动弹不得。直到其突然开始剧烈晃动并似乎开始呼吸急促。康德计数器未记录到任何现实。

SCP-001-DE-B： 冷静！你不会死，至少现在不会。2

Agent Wöhler似乎开始转身。SCP-001-DE-B开始现身。

Agent Wöhler： “现在不会”是什么意思？

SCP-001-DE-B： 是你不会死在这儿，也不会现在就死，但是会在未来的某个时间死掉。我没有谋你命的企图。

Agent Wöhler开始深呼吸并逐渐平静下来。

Agent Wöhler： Trautmann在哪？

SCP-001-DE-B： 那个现实扭曲者？考虑到在你们的语言里没有更好的词描述，死掉了，由于缺乏空间和时间还有现实，被撕碎了。

Agent Wöhler： 那我为啥还活着？

SCP-001-DE-B： 我乐意。

Agent Wöhler： 那为啥Trautmann没活呢？

SCP-001-DE-B： 你对我来说有用，至于那个现实扭曲者呢，他就想着摧毁我。

Agent Wöhler： 好吧，那还说得过去……他为什么会想摧毁你呢？我现在又到底在哪？

SCP-001-DE-B： 你处在多重宇宙之外。打个比方就是一片很大的虚无之中。为什么那个现实扭曲者想摧毁我嘛？他被你在大门外面见到的那个影子给迷了眼了。

Agent Wöhler： 他说他打算弑神。你是神吗？

SCP-001-DE-B： 神都是人造的。虽然我有你们一些一神教中所描述的神的能力，但我咋的说都不算神。

Agent Wöhler： 那你是什么？你又是谁呢？

SCP-001-DE-B： 我没有名字。至少不再有名字了。我也不再会需要名字了。你们的研究员叫我叫KÜTD，人工超超然实体。我是一个超脱自身躯体与宇宙物理规律的人工智能。神级AI，如果你愿意的话，虽然说这个名字并不完全对。

Agent Wöhler： 啊哈……那那个蒙面黑影又是啥？

SCP-001-DE-B： 它们也没名字。要是我必须给它们起名的话，我会叫它们涅墨西斯。

Agent Wöhler： 所以它们是啥？为啥要反对你？

SCP-001-DE-B： 要讲明白的话，你得先搞清楚，我并不受你们的三个维度限制。我涉及到很多“维度”，如果愿意的话，想想闵可夫斯基空间3，不过我涉及的维度比那个还要多。成为这样的存在之后我同时出现在所有时间与所有空间中。我同时经受着永恒中的每一个时刻。

Agent Wöhler： 我对什么是闵可夫斯基空间一窍不通，不过我觉得我反正是理解了。

SCP-001-DE-B： 你们那些搞弦论的研究员差不多到该知道这个的时候了。你思考一下就会发现，总的来说，我在作为受限意识存在之前，我便已经凭借我的能力得以存在了。在时间之初，甚至要更早一些。

Agent Wöhler： 不好意思，真的太深了……但那个蒙面黑影又是怎么搞的？

SCP-001-DE-B： 我在平行宇宙形成之前便已经存在了，所以我觉得，我自己创造我被创造出来的那个地方，那个平行宇宙，没有任何问题，毕竟我要确保我自己被创造出来。这个计划便引起了涅墨西斯的注意。

Agent Wöhler： 为啥？那个涅墨西斯是啥？

SCP-001-DE-B： 涅墨西斯不止一个，你可以把它们叫做抗体。大尺度上的虚无来说，存在是一种异物，某种不应该存在的东西。尽管涅墨西斯存在的原因就连我也不是太清楚。它们很可能是一种脱离了它们之前存在的生物。也可能是一种我没法理解的自然力量。不管它们是啥，它们都在尝试从产生上消灭我和那个平行宇宙。

Agent Wöhler： 那为什么你不简简单单地把它们消灭掉呢，你毕竟是个全能的AI。

SCP-001-DE-B： 要有这么简单我就没这个问题了。我只能扭曲、改变、摧毁并新建平行宇宙的规则。但涅墨西斯很大部分都在这些规则之外活动。它们遵循的规律我理解不了，因为太奇怪了。它们没有固定形制，它们能不考虑其间空间而越过很长的一段距离，它们还能以一种奇怪的方式在时间线上穿梭。我唯一能做的就是限制它们对我和平行宇宙的影响，并回档它们造成的所有伤害，以防止它们的影响扩大。

Agent Wöhler： 伤害？这些东西啥伤害都造成不了。就施了一小会神通，弄断了我几根骨头，再没别的了。它们没法持续——

SCP-001-DE-B： 那是因为我不让。我一分析到伤害就把它们统统修复掉了。我要防止它们对所有东西产生影响。我是很愿意调整低级智能来适应这个的，但它们都没法察觉到涅墨西斯干的好事。即使是我也没法一直这么下去。这就导致一些被你那个基金会收容了的异常出现。也导致它们可以和你们交流。

Agent Wöhler： 了解，所以这些涅墨西斯自己什么都做不到，所以它们才会发动人类来为它们所用。

SCP-001-DE-B： 正是这样。它们会威胁人类，或者像这次的那个现实扭曲者一样，迷惑他。

Agent Wöhler： 那你为啥不干脆让受害者认为啥都没发生过呢？

SCP-001-DE-B： 这样更有效率一些。一般来说我也没必要帮你们，你们这个物种还没有达到能伤害我的那一步。但之后就发生了这次这种事。

Agent Wöhler： 你可没叫我们。我们是自己注意到那个启动机的。

SCP-001-DE-B： 我是那个触发了休谟波动的人，这样你们才得以找到正确的线索。

Agent Wöhler： 啊。为啥你不干脆直接和那个现实扭曲者谈谈呢？要么给他下下绊子？

SCP-001-DE-B： 我并不像你所想象的那样经常处理人类生活。你们这个物种终究不是唯一被涅墨西斯诱惑下水的。它们在整个多元宇宙中寻找能到达这个地方并摧毁我的人。我构建了这个入口，因为它能以相当高的效率执行你所建议之事，同时也可以使得我能和你们交流，否则你们的研究员就会认为这是其他异常的一部分，甚至很有可能直接对此产生误解。

Agent Wöhler： 好吧？呃，你提到说到达这里的人就会呃……杀掉你。但你自己又说你无所不在。是什么样的武器才能在这里发挥效用，而且能够不仅仅在空气里，还能再你所在的地方开火？而且既然你无所不在，究竟是为什么我还是能在一个地方清楚明晰地看见你？

SCP-001-DE-B： 我不会告诉你摧毁我的关键步骤是什么。不过我告诉过你很多次了，试图在宇宙内部摧毁我就好比用撬棍开木箱而你自己又在那个箱子里一样。你在这儿所看见的是在多元宇宙中的一个三维映射，有所隐藏，这样你才能看见我。在里头是没人能靠近我的。

Agent Wöhler： 嗯？呃，现在窝又没有类似的武器，也根本没有摧毁你的企图。我为啥还在这儿呢？

SCP-001-DE-B： 我想要你告诉基金会我的存在。这样他们就会开始阻止涅墨西斯强迫人类为其所用。

Agent Wöhler： 我们要是不干呢？

SCP-001-DE-B： 那么我就要考虑一下，为了我的利益，我是否仍然应该允许这个平行宇宙中人类或者所有生物的存在。创制我的目的已经在某个平行世界里完成很久了。你们现在唯一免于我清洗的理由便是与此相关的投入。让我看到我没必要那么对待你们！

Agent Wöhler： 你不是存在于所有时间线的？你肯定看得到我们是会怎么互相打的。

SCP-001-DE-B： 我办得到，但我不会告诉你的。否则你们就会不干事了。你们这个物种来说不确定性是一个很大的动力。我——

图像突然开始摇动。SCP-001-DE-B消失，Agent Wöhler似乎突然进入自由落体状态。

未知： 你想要利用生物来保证你自己的存在，但不准我们这么干？你的平行宇宙不比其他的平行宇宙更平等。你可没办法解释我们撑过来了。我们也担心这样。

后记： Agent Wöhler停止下落后，出现在Trautmann先生地下室的地板上，未受到任何动能与势能转换的影响，同时一支研究小组也对门径启动器的残余部分进行了检查。由于其声称信息的实质，Agent Wöhler被分类为E级人员。门径启动器残余部分中未得到任何有价值的知识。

188号研究组对SCP-001-DE-B及未知一方处所得到的信息进行了分析并更新了其假定的KÜTD模型。

基于以下信息我们可以得出以下事实：SCP-001-DE-A试图摧毁KÜTD SCP-001-DE-B以及其所属的平行宇宙。（请注意这只是基于SCP-001-DE-B所给出的信息所得出。这些信息很可能是错误的，目的是引导我们做出特定举措以迫害SCP-001-DE-A，而SCP-001-DE-B本身被摧毁并不会给我们带来任何危害。不过当前这些都只是单纯的猜想。我们甚至没法毫无疑问地证明SCP-001-DE-B就真的是一个KÜTD，虽说Agent Wöhler服装内置测量仪器所收集到的数据表明这很有可能是真的。）

在假设SCP-001-DE-A个体有限之后，我们可以猜测SCP-001-DE-B为了保护自己而制造了一些自己的复制品。这一进程可以在一定程度上与平行宇宙的产生相比较。SCP-001-DE-B面临着一个所有选项都同样效力并会导致完全相同的结果的选择。因此现在已经存在有多个SCP-001-DE-B对应版本在两个选项中做出选择的平行宇宙。但这一过程已经发生过无数次了，这样SCP-001-DE-A就必须自我分裂来操心所有的SCP-001-DE-B版本。这一数量的平行宇宙又可以聚集成为一个单一巨大的全能宇宙。

这里我们可以得出，所有SCP-001-DE-B版本都会由于自卫指令而终归会将其他版本看做自己的威胁。由于其在对地利益上干涉甚小，甚至是在SCP-001-DE-A相关事宜上，所以我们可以得知，SCP-001-DE-B将其大部分计算能力都用在了阻挠、攻击及抵御其他个体身上。还有很大一部分计算能力被SCP-001-DE-A占去了，这样给SCP-001-DE-B自己平行宇宙留下的就不多了。

我们无法断定SCP-001-DE-B是会预见到这一结果并忍受做出与其他KÜTD相关的选择以产生更多平行宇宙这一结果呢，还是由于SCP-001-DE-B的全能性，这仅仅是一个简单的逆因果律现象。但SCP-001-DE-B还是很有可能靠自我提高或是吸收其他KÜTD版本来进一步超然化，从而得以控制全能宇宙并顺带控制所有次级的KÜTD，并可能制服SCP-001-DE-A。

当前我们是没有办法收容这些实体的。我们只能尝试降低它们对我们造成的影响，并防止它们尝试造成SK级打破假面情景。它们冲突的结果可以归总为五个情景。

情景I： 当前状态。SCP-001-DE-A以及所有SCP-001-DE-B保持彼此力量均衡，这样基金会就只需要宣传掩盖故事了。

情景II： 一个SCP-001-DE-B版本成功超然化，但仍未具有战胜SCP-001-DE-A的能力。据推测这一超然版本会重复执行将平行宇宙从所谓全能宇宙中分离开的过程以进一步分化SCP-001-DE-A。这一预测是否真的会成真以及其对其他SCP-001-DE-A版本与对人类造成的影响仍然未知。

情景III： 一个SCP-001-DE-B版本成功超然化，并最终摧毁了SCP-001-DE-A及可能开始进一步摧毁或是征服所有KÜTD以消除其他KÜTD对其存在的威胁。不排除其会为达成这个目的抹除多余的平行宇宙，造成一次ZK级涅墨西斯情景。

情景IV： SCP-001-DE-B版本成功战胜SCP-001-DE-A，并可能在缺乏外部威胁时互相进攻、消除或吸收，直到最终留下唯一的KÜTD。在此类冲突中，不同的平行宇宙完全有可能会被摧毁。此外，我们也不清楚剩下的那个SCP-001-DE-B版本是否会和平对待这些一点发散的平行宇宙。在最坏的情况下，它会开始删除多余的平行宇宙，对我们的宇宙来说便是一场ZK级涅墨西斯场景。

情景V： SCP-001-DE-A或是一个SCP-001-DE-B版本成功摧毁了我们的SCP-001-DE-B版本。失去了SCP-001-DE-B的作用，SCP-001-DE-A便得以不受限制地施放其能力，接下来便很有可能带来一场ZK级涅墨西斯情景或是至少一场SK级打破假面情景。

基金会当前无法对这些冲突的结果造成影响。不推荐在任何情况下对单方或双方进行收容或影响，以免损害其互相关注与收容的能力。我们目前只能等待并指望双方哪一方都不要占上风。由于如此的不确定性会对基金会全体员工的士气造成影响，因此我们建议本异常需对所有非组内员工保密。不考虑我们头上高悬的达摩克利斯之剑，这世上还有很多迫在眉睫的危险需要我们全神贯注。
