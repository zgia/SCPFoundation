古法修研

审批者：██████ ████

第一章 —— 法术简介

处理好魔杖Wands 的尺寸和位置会有更好的效果。1

如果你更愿意使用法杖staff 的话，只须摇动其头部，而不是一整根。2

法器casting tool应该要轻便，如果你的施法道具较重，就把对象放在容易操作的东西里。


袖里乾坤术Bag of Holding Spell

所需物品： 法器 x 1，袋子或可容纳物品的东西 x 1

将法器指向你想要的加持的事物，然后摇动施法道具画一个完美的方形，并用你的母语吟诵：“汝怀之，吾颂也。汝愈怀之，吾呼愈乐也。”此术可使对象化作无限容量的包裹。若想要给其加锁，则将法器指之并以母语吟诵道：“吾因汝馈而喜甚，然他者欲窃之，吾望护汝，以斯言为之闭锁，*‘暗语’*”。尔后，密匙就形成了。


炼器术Casting Item Spell

所需物品： 法器 x 1， 未受祝的新宝具 x 13

将法器指向新宝具4，然后用你的母语吟诵：“自为之，为之甚。”此举可将你的法器的功能转移到你选中的新宝具里。如果你不喜欢新宝具的运作方式，你可以很容易地重复刚才的流程再做另一个。

注意：你不可以对早就铸能过的宝具施法来替换原本的魔力，
例如：你尝试对宝具的铸能之处item’s casting power使用该咒语。
结果:无效。

例如: 你得到一根未受祝的棍杖，并施加此咒。
结果: 新法杖。


初等护体术Basic Protection Spell

所需物品: 法器 x 1

快速将法器移到你将受攻击的部位并用母语吟诵：“伤害速退！”这可能会防御任何基础物理伤害或初等魔法咒术，但对更强的法术无效。


巫毒心惶咒Voodoo And Phobia Curse

所需物品： 法器 x 1，棉娃娃 x 1，你想要诅咒的对象的体液、皮肤、毛发、等等之类的身体的部分 x 1， 燃烧的蜡烛 x 4，针线包 x 1

将你从你想要诅咒的那个人身体的一部分放入巫毒娃娃里面，再缝合其背部。之后再放置巫毒娃娃到一间黑暗的房间里，并在四方点燃蜡烛环绕。接着拿起法器指向巫毒娃娃，并用被诅咒者目前使用的语言吟诵：“孤欲使受咒者罹此苦既哀且惧，孤欲使之俱，*受咒者真名* ，孤欲使吾可刑尔如刑此偶欤。”然后，你就有了巫毒娃娃替身。若你伤害巫毒娃娃，对方肯定也会同样受伤。

如果你想要诅咒他们终日惊恐，将法器指向你想要诅咒对象的巫毒娃娃替身，并划个方形，用受咒者当前使用的语言吟诵：“*受咒者真名*，尔负万般极苦，绝望乎！尔，孤之小弱玩偶，*其名者汝欲怖人以何*，应惧乎！”

无论你选择什么，哪怕是水，受咒者也会恐惧于此。


缴械术Disarming Spell

所需物品： 法器 x 1.

快速挥动你的法器划闪电，并指向敌人的法器，再用母语吟诵：“此力归他处。”除非对方使用防护咒或他们的法器相当强力，否则这个法术总会生效。5


景明术Sight Spell

所需物品：法器 x 1，任何可能的光源

你可以在不聚集任何光的情况下施放这个法术，但是聚集光能可以提高法术的亮度和持久度，所聚集的光能的外界光源的亮度必须和蜡烛差不多。6

为了收集光源，将你的法器指向光线，用任何你喜欢的语言大声呼唤：“与我共耀”。然后你就可以聚集光能了。

从法器中释放光能时，用任一语言吟诵：“予以绽灵光。”

这会让你的法器发光。


初击术Basic Attack Spell

所需物品：法器 x 1.

将法器指向对手7，并用你的母语严厉地吟诵：“天轰此贼！”这将向你的敌人释放出强大的能量，可能会击倒他们或使之瘫痪，这取决于你击中的地方。


初等屈位术Basic Warp Spell

所需物品： 法器 x 1.

将你的法器指向空中，比划个圆，并用哥布林goblin的语言吟诵：
“揭砵柯 依窈 吽尼姆 赭Gïburk Eįú hûñįm jęw，*地名*”
翻译过来就是，
“移凡夫至*地名*.”

这将把你传送到你所选择的目的地。当然，你也可以将你的法器对着某个敌人念此咒语，它会把敌人传送到那。


威能化兵术Energy Weapon Spell

所需物品：法器 x 1.

像你想要拿的武器的姿势一样拿着你的法器，接着在 [已编辑] 中吟诵武器的名字，然后你的法器就像是你想要的武器一样运作。


驯仆术Domestic Spell

所需物品： 法器 x 1, 生灵 x 1

用你的法器划个心形，使生物发出噪音会让其更容易驯服。或者如果已经驯养了该生物，会直接驯服。

附记：此法术不适用于更智能的生物，比如人类、哥布林、龙族、█████，等等。


羽化术Bewinging Spell

所需物品： 法器 x 1，羽毛/蝙蝠、鸟类、昆虫的翅膀 x 1

首先拿翅膀或者羽毛放到平面上，用法器指向其，再在你的头顶的正上方直指，并用精灵语elven吟诵：
“毐 威契 提 呼毗 嗡 苏欲吉 尼柯 悉菩，德提 恩涅Į vįkc tì hūpi œ sū yūge ñq sīwp,dêt mñie
提 怯尼卬格 咼啒於 尼尼 阿威伐 阇格
tì kæńyärg wïquw nini ęwīf shėgb.”

翻译过来就是：
“挟飞仙以遨游，
於，轻且怜如许，
但愿昊天举吾足以上。”

这会给你生成你所选择的翅膀样式，而其会与你的身形相适应，甚至你去选择昆虫的翅膀也会如此。


御空术Flight Spell

所需物品： 法器 x 1，罐装烟/雾 x 1

将法器指向罐子上方，用精灵通用语elvenian吟诵：
“怛呗羯，廓叱 啒 赭柯，啰吉 阿威伐 约依Dîęqb gę,Køwčį qø jōwqē,rįax ęwīf yęœ
唵儿威呗 呼卢 鄂伮 奴啒目

lvįßė hūįrw ēñu ńūxvmù.”
翻译过来就是：
“且柔也，莫遮掩，随吾去，如珍美姬，君愈进奇哉。”


元灵法术Elemental Spell

所需物品： 法器 x 1，造化之书 x 1

若你没有造化之书，你需要以下材料制作一本：
空白的书 x 1，草叶 x 16，水滴 x 1，灰烬 x 1，金属薄片 x 1，卵石 x 3，砂砾 x 1，血滴 x 1

在把这些一同混入空白之书里面后，用法器指向那书，划个方形，并用你的母语吟诵：
“如人之生，似命之合，自其同成。”
这会给你带来一本造化之书。

造化之书包含多种你可以使用的元灵法术。但如果你仅是带着这本书，那只要你简单地用你的母语说出你意图使用的元素，你的法器可以释放相应的元灵法术。


焉为煞咒How To Craft A Curse

*这部分的页面上仅仅覆盖了一层血，对血液的分析显示没有任何现有的生物与之相匹配，唯一与之紧密关联的存在是█████。*


██之术██████’s Spell

所需物品： 法器 x 1

将法器指向你自己的嘴，并用███语来吟诵：
[已编辑]
翻译过来就是：
“往者已逝，余皆敬之，
愿余与其同语，以表敬意。”

这将会让你通晓并流利地说出任何语言。


军群召唤术Horde Spell

所需物品： 法器 x 1，邪血Vile of Blood x 18

你面向你要施法的地面后，在你颈部用邪血涂个三角形，并以蛮兽人中的饰巫者9之语吟诵：
“羯磨 呗赭惹 乌嚧 咼乌鲁Gîmú Będrz Łį Wæwr”
翻译过来就是：
“修我矛戟，与子偕作！”

这将召唤一支统一的岩兽军队，其数量取决于你的法力。这些野兽装备着剑或弓，将为你奋战到死。


怪仆术Labor Spell

所需物品：法器 x 1，章鱼触须 x 8，人血血滴 x 8

让每一根触须都沾上其专属的一滴血，而不是把它们堆成一堆。摇动你的法器划个圆，此时你必须用你的母语吟诵：“分之，合一，吾友，与汝杂务矣。”这将使你拥有你自己的人外乌贼仆，其几乎可以做任何你要求的事情。


过誉咒Glorious Curse

所需物品：法器 x 1，盛有英雄之血的圣杯 x 1，用长臂猿制作成的香料 x 1

将英雄之血倒入圣杯，将其与用长臂猿制作成的香料混合，并用吸血鬼vampiric之语吟诵：“依赭那悉 阿优伐 乌勒 约依德悉 威阿忒 阿 唎阿母， 德 萨儿醯 阿优伐 威阿母悉 知乌斡 廓 斡 佛优母Igenois oiuf ure joidēs vīėt a rìēøm, de sēirs oiuf vēæms zôl kîõ wô fœwn”。
翻译过来就是：
“尔之长也，忘之何憾也，为尔欢庆之徒，必噬尔命于名扬。”

然后再让某个人喝下这些，那人现在会被他人认为是了不起的。而人们则总是过分要求受咒者去做某些事情，不过这种影响并不会改变施法者对受咒者的看法。


猫之皮衣咒Cat Skin Curse

所需物品： 法器 x 1，猫皮 x 1

扔猫皮到你想要诅咒的对象上，然后将法器指向那个人，并轻声吟诵：“此也仿，彼也仿，尔非天生真狸奴，尔为小弱仿制猫10。”这会将对方化形为小型猫科动物，并模仿你命令其模仿的动作。

附记：曾经有一位非常强大的女人，在学到永生的秘密后，给自己施加一个可逆的咒语。这就是为什么说，世界上有的猫儿有无穷的寿命的传言，是明智的。


初等化形术Basic Metamorphoses Spell

所需物品： 法器 x 1，血滴 x 1

获取你想变形成的生物或人的血液并使用，

法术持续的时间取决于你所耗用血滴的数量。

将法器指向盛血的容器，用你的母语吟诵：“令神魔塑我，我身化他物。”一旦饮下被施咒的血液，你将会在一段时间内变成其原主的形态。

示例：
一滴血 = 一分钟


妖之赌局Goblins Bet

警告：虽然这并非禁咒，但使用时请务必小心。
所需物品： 法器 x 1，门径 x 1，某些物品用于充当筹码

拿起法器指着一条通道，划个钥匙形，并用我即哥布林之语I’m goblin language吟诵，“依毐柯 殃母 赭乌 菩依德 咼柯 欸优威柯 哥布林 Eiq Yeim jew puied wik eiueq goblin”，翻译过来就是“吾至此同尔质赀。”当这句话念出时，你，也只有你才可以通过进入次元之门。而当诈欺之咒释放后，其他人仅仅穿过此门如寻常一般。当施法者，也就是你，进入并通过门径后，将会看到交易正在快速进行。一只哥布林悄然坐在旁边，它会提供给你三种赌局，并与你进行其中一种。

你必须在赌局中拿出任一事物下注，否则将不会获得任何奖励或者没有机会进行游戏，不然你后面会被踢出局几个小时。你可以拿出除了实际的货币外，包括记忆、感觉、绘画品、玩具在内的任何东西作为筹码。

以下是供你选择的三种游戏：
哥布林抽卡，
猜价，或者，
除你武器、格挡或升降。

猜价游戏中，你必须精准地估测哥布林选择的物品实际值多少。这将会很难，因为哥布林会选择类似回忆、幼女的贞操之类的近似不可能定价的东西。

相抵或升降是要运用到你的法器的游戏，你可以选择提高你的赌注来格挡哥布林的攻击，或者缴械。

如果你和哥布林同时缴械，那么两个玩家都留在游戏中，如果你们相格挡后无事发生，你就是唯一一个会提高奖励的人。若哥布林选择缴械时，你选择了提升奖励，你就输了。倘使你缴械时，哥布林选择降低损失。你不能再选择更低的了。


习技术Skill Spell

所需物品： 法器 x 1

将法器指向胸口划五芒星，并默念你想要习得的技能，再用五种不同的语言吟诵。这会赋予你十小时的相关技能，这依据于你的能力大小而决定能再持续多久。


龙涎Dragon Spit

所需物品： 法器 x 1，整杯的任一液体 x 1，任何龙的体液 x 1

将龙的体液与其他液体混合，然后把法器指向混合液并施加 混制要诀Mixology Spell11。

施加该咒时应用你的母语吟诵：“分尝亦奇之，共品则超乎至善。”

当这些混一时，你自己就拥有了多元宇宙中酒精含量最高的美酒。

警告：那些对酒精承受能力较弱的人，或者在过去12小时内喝过酒的人，最有可能会死于酒精中毒。

魔像术Golem Spell

所需物品： 法器 x 1，至少八英尺高的任一石头或金属 x 1

将你的法器指向石头或金属，划个三角形，并用██之语吟诵 [数据删除] ，翻译过来就是“闻吾所召，乃见天光。感足底之吐纳，嗅吾雠之血焉。吾之从者，以醒以生。”这将会直接召唤外道魔像并听命于你。而其尺寸，则由你魔法能力的高低，以及你使用的原材料的大小，来决定。


六识幻戏Pheromone Magic

所需物品： 法器 x 1，罐装弗洛蒙 x 1，狼人之牙Wearwolf Tooth x 1 ，永恒粘剂Forever Glue x 1

用永恒粘剂把狼人之牙黏到你的法器上。

将法器指向你的罐装弗洛蒙，并以矮灵pixinain12语吟诵：
“*两个相互缠绕的螺旋，形成类似于大写A的图案。一个实心矩形，仅在带有个空心三角形的顶部那里，没有被填充。一个八边的星形标记，一个近似鹰眼的符号，一个锤形记号，一个构成近似字母V的相缠绕的双螺旋，一个菱形符号而内有三个黑点组成三角形。十二边星形符号，内有六个点的菱形标记，内有三角形的方形图标，带有十条白纹的两个红瞳印记，六边星形符号。*”

翻译过来就是：
“于吾神器中吸之，为吾需时而吐气。”

现在你将有一个持久的法术。当你需要施放弗洛蒙时，咏唱其名，一阵魔力暴动将会传到你的目标那，并用相应信息素的特性侵染他们。

例如：花的信息素。
结果：人会真切地感受到幸福，举止中透露着甜蜜和爱。

例如：臭鼬的信息素。
结果：人将日日作呕，失去沐浴的兴趣，易染疾病。


色相幻戏/高位攻防术Color Magic/Advanced Attack and Defense Spells

所需物品： 法器 x 1，光棱镜spectrum glass x 1， 镜子 x 1

将光棱镜放到朝向镜子的平面上。
用你的魔棒指向光棱镜，并以矮灵语吟诵到：
“闭目见黑相，背暗则为昼之白。煦以明光，激之而成怒色赤；抚之为橘，缓之甚乃得黄。取天苍合之，以发生灵其青青。舀瀚海之力，酌熔岩之炽，混同化为紫。如此，君得霓虹焉。”

如此一来你就可以用法器施放色相幻戏。当用矮灵语吟诵时你将会得到不同类型的防护，用吸血鬼语说不同的颜色名你将能做出各种各样的攻击。

例如：守护之棕
结果： 形成一个流沙深坑，能阻滞攻击者。

例如：灰之强袭
结果：引动石屑和其他物品等尖锐的碎片，会聚集且环绕在敌人身旁，并形成飓风。

例如：天苍之击
结果： 致使对方被锁喉，此时你可以把他们四处乱甩。如果你把他们遗弃在那里，他们会窒息而死。

例如：伏击之棕
结果：此术可召唤一条饥饿的流沙之蛇，它将猎杀敌人。

例如：黄之守中
结果：凝聚成一个灵能力场，环绕在你身旁以提供保护。或者让它去做你想要的动作。
