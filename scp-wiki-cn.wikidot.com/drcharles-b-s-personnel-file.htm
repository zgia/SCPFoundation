Drcharles B S Personnel File

CB系統V 2.0.0.1

查尔斯博士在Site-09的一次收容突破演习中的档案照

姓名：Charles.Bowen.██
現使用CB作為代稱
暱稱：隨便叫什麼都可以，通常被稱作CB，查爾斯，垃圾人以及6%和沒救的死宅

職位：外派駐站員，研究員，特殊情況下可以加入戰鬥單位

專長：自然哲學，玄學，生物科學，政治學

外觀：CB博士每天有50%的機率戴上他的銀色圓框眼鏡，一般情況下身高1.66cm， 偏瘦，右手肌肉比左手肌肉奇怪的強壯許多。 臉型屬於圓臉。在不戴眼鏡的情況下，眼睛的大小是戴眼鏡時的兩倍。且頭髮常常在修剪後一小時內長出5cm或以上。

Site-CN-09站点主管。原担任Site-CN-05副主任职位。情緒無常，現有紀錄為在一小時內表現出23種不同的情緒，這個特性十分危險，參考事故報告CB2029-████，雖然只造成了12名D級人員死亡以及一名2級研究員輕傷的損失，但是網路太卡並不是你開不出龍狙的原因，更不是你在D級拘禁單元拿著P-90爾西莫夫（嶄新出廠）胡亂掃射的理由！，在此事件後，查爾斯博士被調往人煙稀少的Site-CN-09綜合站點擔任主管職位。
