SCP-048-FR

项目编号： SCP-048-FR

威胁等级： 白 o

项目等级： Safe

特殊收容措施： SCP-048-FR目前被收容于Site-Aleph的一个加固库房中。考虑到SCP-048-FR的潜在危害，需2级人员授权，任何具有风险的实验必须在特工面前进行。由于SCP-048-FR的潜在能力，SCP与SCP之间的任何交互都可能会对其造成损害，因此需要获得4级授权才可进行交互实验。

石碑上方铭文的照片可供随意查阅。

描述： SCP-048-FR是一块具有不规则边缘的砂岩块，尺寸高1米，宽50厘米，厚约10厘米，J-C硬度在2500-3000之间。石板上雕刻有模糊不清的阿卡德语字母语句的铭文。

该砂岩块发现于伊拉克某考古挖掘场地附近一个名为██████的村庄。挖掘现场已被基金会收购，基金会人员正寻找其他SCP-048-FR的其他实体和受影响者。目击SCP-048-FR的平民被予以A级记忆清除并遣返。

当一个或多个人与岩块保持身体接触时，SCP-048-FR的异常效应显现。在这段时间里，每位受影响者都将能理解现存的语言/方言，即使他们先前完全没有学习，使用过。

在与岩块接触的过程中，前者将会如沙漏计时器般缓慢分解成细碎的岩沙。这些岩沙没有异常。

目前，无法确定SCP-048-FR是否已经完全被收容，但可以确认的是基金会所持有的是未被其异常形质消耗的实体。

附录： 以下内容为岩块上阿卡德语文字的近似翻译:

我们对这位当初对我们不敬的神予以了绝罚。
我们将一起承担他的所有领域。
我们团结一致，我们既是一切
我们将是上帝。

+ 实验记录
- 实验记录

日期 : 25/10/14
受试人员 : D-2510 (说法语与英语), D-2514 (仅说法语)
实验过程 : 这两名D级人员被要求将手放在SCP-048-FR上，并以各自的母语发言。
对话 :
D-2510 : (法语) 好极了，一切尽在掌握。
D-2514 : (法语) 啊，老哥我懂。
D-2510 : (法语) 这很正常，我会说法语。

分析 : SCP-048-FR开始分解。

日期 : 25/10/14
受试人员 : D-2510 (说法语与英语), D-2514 (仅说法语)
实验过程 : D组的两名工作人员被要求将手放在SCP-048-FR上。D-2510必须用英语说一句话。
对话 :
D-2510 : (英语) 我们正在进行一次实验。

分析 : 正如在岩块修复过程中所确定的那样，项目的异常效应使得D-2514能够听懂法语短语。应该指出的是，D-2510也以法语发言。但是，放置在房间里的麦克风却记录了英语的语言记录。

日期 : 25/10/14
受试人员 : D-2510 (说法语和英语), D-2514 (仅说法语)
实验过程 : D组的两名工作人员被要求将手放在SCP-048-FR上。D-2514必须模拟用一种不存在的语言说话。
对话 :
D-2514 : 夤夢夰奨奢奜奟奩奭

分析 : D-2510和D-2514没有听到任何交谈。

日期 : 25/10/14
受试人员 : D-2510 (说法语和英语), D-2514 (仅说法语)
实验过程 : D组的两名工作人员被要求将手放在SCP-048-EN上。D-2514必须在用英文在一张纸上表述。
对话 :
D-2514 : (英语) 呃…我们正在做一个实验…

分析 : 尽管这是一个可理解的英语语句，但SCP-048-FR的异常效应并没有表现出来。猜测项目的异常效应仅限于口头表述。

日期 : 28/10/14
受试人员 : D-2842与一条2岁的拉布拉多犬
实验过程 : 要求D-2842与SCP-048-FR保持接触并与动物交谈。安保人员令狗的前腿接触到岩块。
对话 :
D-2842 : 你好，小狗！今天过得还好吗？

分析 : 狗对D-2842的字没有任何异常反应，只是嗅了一下SCP-048-FR，然后舔了一下警卫的手指。 D-2842没有听到任何狗的声音。 发现当D-2842将他的手从平板电脑上取下时，虽然狗仍与岩块接触，但后者没有变为散沙。

日期 : 08/11/14
受试人员 : D-2842和一只经过训练的黑猩猩，能够使用基础手语进行交流。
实验过程 : 要求D-2842与SCP-048-FR保持接触，并提前向猩猩提问。基金会动物协会的一名精通手语的成员陪伴黑猩猩，并通知其需要触摸石头。
对话 :
D-2842 : 呃……天空是什么颜色的？

分析 : 黑猩猩对D-2842的话语没有任何异常反应。他转向他的同伴，使用手语询问动物协会成员。岩块在黑猩猩接触时不分解。

日期 : 09/11/14
受试人员 : D-2842 (说英语与法语), D-2496 (说意大利语与英语), D-2471 (说俄语与德语), D-2852 (仅说西班牙语)
实验过程 : 四名D级人员被要求将他们的手放在SCP-048-FR上并以他们各自的母语发言。 考虑到D级人员号码识别，另外的特工被安排在场。
对话 :
D-2842 : (法语) 你好，你能听懂我说话吗？
D-2852 : (西班牙语) 听得懂，但是我们在这里做些啥？
D-2471 : (俄语) 和我们一样。都在做同一件事情
D-2852 : (西班牙语) 也许，但不会很久。我对他们的实验简直厌烦透顶。
D-2496 : (意大利语) 然后，你打算怎么做？请假吗？
D-2852 : (西班牙语) 正有此意……

在交换结束时，D-2852抓住了SCP-048-FR并将其扔向其中一名特工。该代理试图将平板电脑和收据放入胸部，打破两根肋骨并立即将其敲出。 然后D-2852投身于服役武器并将其击落的二等军官。 其他三个D班立即走到了地面，以免出现威胁。 SCP-048-FR在混战中没有被损坏。

分析 : SCP-048-FR可以同时为多人翻译对话。 然而，平板电脑的退化与接触的人数成比例地增加。现在已确定了SCP-048-FR的能力，无需再进行更多的D级人员实验。
