齿轮画廊的晴朗一日

欢迎来到齿轮画廊！现在展出的有三个部分，请享受你在这里的时光。在这边支持艺术家。

折叠展开
目录
第一部分：收容项目
第二部分：故事项目
第三部分：视觉项目


第一部分：收容项目

SCP-2481 - 射日

简介：

这个古老的王国渴望弑神。聆听孤独的幸存者的故事。

特性：历史，悲剧，被遗忘。

SCP-2711 - 定海神针 如意金箍棒

简介：

一个传说中的造物在湖中被发现。里面隐藏了什么惊天奥秘？

特性：神话，金属，启发。

SCP-2732 - 《缢王本纪》

简介：

通过一出怪诞剧解开一个国王背后失落的历史。不过要小心。

特性：历史，艺术性，可怖。

SCP-2847 -禹王不死

简介：

是什么样的执念让这个灵魂驻守在这铁铸的大渊之中？原因已经亡佚了吗？

特性：金属，英雄，崩坏。


其余展品：

  • SCP-2841 - 虺蛇的雪仇
  • SCP-3610 - 填海
  • SCP-3847 - 但愿我能与鸟交流

第二部分：故事项目

我写了一篇外围

简介：

一场与Megaprime的绝妙经历。让它进来。你会明白大计算的需要。

特性：神圣，破碎，困惑。

晴日已到，而他又聋又瞎

简介：

你试过来到一个地方，但是你的记忆却离你而去吗？你感受过无法抵抗的痛苦和无法填补的饥饿吗？来找出到底发生了什么。

特性：冰冷，饥饿，痛苦。

铁狼

简介：

"铁狼蛰伏于黑暗，其齿苍白而锋芒。眼望血肉之兽群，以爪猎捕肉羔羊。"

特性：金属，野性，变形

大龙之日

简介：

在另一个世界线，一切会变得不同。在龙的时代，大术士最后的想法是什么？

特性：另类，反传统，神圣。

造主与兽

简介：

于大渊之上，机械造主与噬神者相遇。令人着迷。

特性：神圣，古老，破碎。

蛇说

简介：

过去的幸存者诉说的怪异故事。哪些是真相？哪些又仅仅是故事？

特性：神话，历史，古老。

龙与蛇

简介：

向龙拜伏。向二龙拜伏。向一对龙拜伏。找出我们的父母到底是什么巨兽。

特性：神圣，诗意，破碎，困境。

何故五惧于七？

简介：

乌云聚集于天空之上；虚空之环现于大洋之底；随后，回归虚无。

特性：黑暗，空洞，虚无。

神会

简介：

诸神会议聚会议事。然也，此乃常事。

特性：神圣，滑稽，欢乐。

如是群鸦讪笑不止

简介：

带你领略人们推翻他们的王的古老传统！另外，你可能想留意一下那些乌鸦。

特性：诅咒，折磨，倒吊，鸟类。

午夜游行

简介：

每当午夜到来，幻想就会开始。事实上它从未结束。不要尝试逃跑。

特性：乐趣，无尽，蒙面。

钟必再响

简介：

嘀，嗒。嘀，嗒。嘀，嗒。

特性：未定义，永恒，混沌。

基础观察

简介：

潜入麦克斯韦宗的网络！在展开的新世界当中连接思想！

特性：数字化，基础，隐藏。

流动 中心页
