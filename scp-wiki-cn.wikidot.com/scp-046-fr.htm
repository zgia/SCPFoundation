SCP-046-FR

SCP-046-FR的入口

项目编号： SCP-046-FR

威胁等级： 黄 ●

项目等级： Euclid

特殊收容措施： SCP-046-FR位于美索不达米亚的摩苏尔北部山脉的地下20公里处。SCP-046-FR所在的地下洞穴的唯一入口必须以铀矿开采为由以基金会的武装力量封锁，并与当地部队达成协议。进入洞穴行为本身应仅限于测试和实验，以及确保该区域安全（洞穴长2公里，高100米，宽1公里）。
在没有4级人员的授权的情况下，禁止对SCP-046-01-FR，SCP-046-02-FR以及两者的产物进行任何形式的攻击。

描述： SCP-046-FR由两部分组成：SCP-046-01-FR，似乎是一艘大型木船，还有SCP-046-02-FR，一个类似于老人的实体，其腰部之棒槌和手中的牧羊人拐杖均由纯金制成。
当SCP-046-02-FR进入SCP-046-01-FR时，其能够展示任何未灭绝物种。但SCP-046-02-FR的创作数量似乎有限：实际上，洞穴中存在的动植物等生命体的数量从未超过50个。

SCP-046-02-FR的牧羊人拐杖可以发光，经过分析，拐杖放出的辐射具有催化反应，能够使生物有机体以比正常情况高近1000倍的速度再生。

当SCP-046-02-FR锤子与水接触时，水域以及水源将立即被净化。测试表明，锤子通过发出波动复制或者重组构成水中的各化合物的亚原子粒子，置换出水分子中的所有污染残留物。

如果SCP-046-02-FR在距离洞穴入口最远的墙壁上敲击，则墙壁上将开出一扇门，通过该扇门，创造出的动植物将进入其适当的自然环境。尽管经过多次实验，导致门是如何被创建的仍未得到解释。 但是，假设SCP-046-02-FR的造物仍可作用于任何物质。

SCP-046-02-FR同时也能够与他的造物进行沟通，并利用其来保护其洞穴拒止任何侵略者。

任何侵犯SCP-046-01-FR，SCP-046-02-FR或其任何造物都将导致SCP-046-02-FR敌意骤增。并迫使其使用造物来攻击侵略者。

SCP-046-01-FR所在的洞穴覆盖着植被，并有一个水源，显然是地下水。

SCP-046-01-FR的存在允许繁殖于其上的动植物存活并保持健康而无需额外资源。向洞穴中派遣没有食物配给的D级工作人员证明这种异常效应能影响到的不仅是SCP-046-02-FR的造物，而是洞穴中的所有生命形式。关于SCP-046-01-FR在这种异常中的作用的假说已经建立，但是，除了发现SCP-046-01-FR周边的磁场外，没有进行任何其他实验进行，因为大多数实验导致了SCP-046-02-FR的敌意，后者坚持认为SCP-046-01-FR“可能被损坏”。

洞穴的天花板上存在有晶体，可以产生光线，使其获得充足的光。

SCP-046-02-FR似乎在此洞穴内永久保留了一部分造物。

只要不造成伤害，SCP-046-02-FR或SCP-046-01-FR中的各类生物（包括掠食者）就完全无害。

附录 046-01-FR :

特工Neremsa通过Dr.Grym的通讯收到了关于项目的原始报告。以下是通讯内容。

+ Dr.Grym发送予特工Neremsa的警告函：机密
- 中止传输

Dr. Grym在传输过程中使用了113-B的单线性加密算法加密了以下消息：

你好Nerem.
我知道已经过了一段时间了，因为我没有任何音讯，但这一次，一切都很好。
大约十个小时前，我在美索不达米亚的某个地方找到了“四个”中的一个。目前，我位于澳大利亚中部第一个可以找到的网吧。这其中有成吨的故事可以讲。
这就是我现在所掌握的，PCS和所有一切破事，以及我所知道和发生的一切。
接近它时要小心。我的建议目前可能不准确，但我认为他们会帮助你到达活着的实体面前。
我要去寻找下一个了。在报告的最后，有一份我的笔记和录音，以及探索摘要。
我希望我很快就能回来，祝你好运
Grym。

在将特遣队派遣到现场后，Grym博士的初步报告得到了更新。

Dr. Grym的笔记：

当我开始从我的同行组织联系人那里寻找关于“四人”的信息时，我起初有点犹豫。但是通过尝试，我得到了答案。这相当意外。
我的一位联系人告诉我，出现了一个新的同行组织。他们正在研究摩苏尔北部山区的一样东西。
我立即去了那里，没过多久我就得到了关于这个新组的一些信息。 最后，让我们说他不久前就让自己知道了，但他又回到了光明骑士团。他们或多或少地对神圣的一切食物感兴趣。它可能与SCP-038-FR所谈论的那个着名的“他”相对应。在最坏的情况下，它可能只是一个普通的，会说话的SCP。无论如何，我必须理清这一切。
我花了两个星期的时间，设法与该组织的一名成员取得了联系。我向他们提供了我的能力，但没有明确说明我属于基金会，并告诉他们我已经不得不处理某些异常物体。这些家伙，一伙子绚丽的狂热分子，就在那里。但简而言之，我在这里为上帝服务，那种已经是过去式的上帝。一旦进入小组，我发现了一个似乎没有其他人疯狂的小家伙。他的名字是Nämu。毫无疑问，阿拉伯名字。我很快就和他成了朋友。

他们在山洞中发现了古希伯来文字，并且开了一扇以前不知是谁自愿将其封起来的石门。
需要再过两周才能清理通道并开始探索。我的天啊，在地表以下20公里的洞穴。
在第二周结束时，我们花了一周的时间集结了一支远征军。一切都是为深入地底而准备的。显然，这些狂热分子拿走了一些武器。他们虽然狂热，但显然还不是白痴。Nämu跟随我们一起下去。
我们走得越远，墙上的史前动物壁画就越多。
在下降的第三天，在洞穴的正中间，从远处看到了光。以下是录音。

以下录音来自Grym博士进入洞穴时佩戴的录音机的记录。

Dr. Grym : 他妈的是什么？这里有光！来吧。
探索队成员 1 : 联络基地。
Dr. Grym : 好嘞。
探索队成员 1 : 我们要进去洞穴了，走吧！
传来脚步声以及AK自动步枪的上膛声。
Dr. Grym : 哦，谢特。
探索队成员 2 : 妈的，那是啥？
Nämu : 看起来像一个方舟……
Dr. Grym : 等等，树木是如何在地底下20公里长得这么大的？
探索队领队 : Garett，Arzil，去看看方舟的左边。Grym，Nämu，去检查右侧。你跟我来，我们会看到方舟里的东西。
脚步声，正在移动。
Dr. Grym : 我希望我们不会落在3米高的东西上。
Nämu : 我也是，那里还能有这么多生物真是惊了，就像我第一次见到水牛一样。
数秒后，传来枪声。
Dr. Grym : 发生什么了？你们有谁看见了吗？
Nämu : 听起来像是方舟另一边传来的，看起来是……等等，这个老头子谁啊？
SCP-046-02-FR : 又是人类。我们封锁了所有入口。我得让你们中的一些人知道这个地方与你们的嗜血行径是不相容的。你们不应该在这里乱搞事情。我们给了你一个恢复平衡的机会，你们却要破坏它？可怜的傻瓜，“他”会让你付出代价的！
Dr. Grym : 不要动，Nämu。我认为这就是我们正在寻找的东西。平衡？你就是“四个”中的一个吗？
SCP-046-02-FR : 你怎么会知道“四个”？你来这里是为了结束我们的使命吗？你是不会成功的可怜人！ (用不明语言说话)
Dr. Grym : 等等！我们不要你任何……等等，那恐怖玩意是什么？
Nämu : 快跑啊！
传来人类尖叫与哭喊，并有快速的步伐。
探索队领队 : 他们在后面，射击！射击！啊啊啊啊啊啊……
传来大象踩碎骨头的声音
Nämu : 队长！
Dr. Grym : 跑！快跑！尽全力跑！
Nämu (大喊): 看起来……是个沙漠！
Dr. Grym : 是一个门，跳！期待它们别跟上来！
SCP-046-02-FR (后方) : 你们辜负了这一直包容着你们的家园！你们——
Dr. Grym : 他们在那里！快跳！
传来物体坠入沙子中的声音。
Nämu : 妈的，终于逃走了，还好没跟上来……卧槽！
Dr. Grym (呻吟) : 别担心。它将会……很快……它开始……他妈的老虎，我还没有看到它来到这里……
Nämu : 这是什么鬼？你的胳膊……它把你……？
Dr. Grym (声音) : 靠，好痛啊。
Nämu : 他……他松口了……这发生了什么？
Dr. Grym : 这是一个…………不要担心他会……完全退缩的，这需要一点时间。 我们……赶快上路走吧。

根据Nämu的说法，其中一个白痴向他面前的灌木丛中的狮子开枪，所以在我们反应过来之前，我们遇到了一群狼，然后是整个动物园。他们都像白痴一样死去，我们通过大门逃到了澳大利亚的沙漠中。我们走了很多时间，但终于找到了。我对Nämu解释了一切。在他看到之后，我认为他必须改变他这些疯狂的想法，并尽可能多地掌握的信息，他们可能对基金会有用。在此期间，我把它贴身携带，就是为了监视他……并试着重新接触骑士团。Nämu听说过这个光明骑士团在世界各地的其他研究站点。
我会去那里，我们明天将被一名在ASIS的旧联系人撤走。
别担心。我知道我们现在有一个知道得有点多的前同行组织成员，但他现在对我很有用，我认为那下面发生的事情已经深刻地改变了他们。
别担心，他将成为特工Nämu。或者不会，我以性命担保。
一旦我有新的发现，我就会通知你。以下是洞穴及其入口的确切坐标：[数据删除]。

Grym
