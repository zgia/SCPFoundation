正在验证安保权限…

您的权限为 4级。
需要权限为 4级。

验证通过。您可以访问实验记录301。

“写入”测试
日期： ██/██/20██
实验对象： D-6213
写入数据： G████博士的性格数据
实验结果： 对象性格出现极大转变，表现出了好学、温和、内敛、乐于合作等迹象，与性格来源类似。向对象教授了基础研究方法后，对象被允许对数个中学物理课题展开研究（并适当给予了相应的资源）。对象表现良好，其作品有较高的完成度和创意，表现出对细节和新意的偏好。

“写入”测试
日期： ██/██/20██
实验对象： D-42203
写入数据： 将数据5（-保守/+进攻）从-30%增加到了100%。
实验结果： 对象佩戴了录音设备以跟踪其行为，随后返回其宿舍。在接下来的一天内，对象与同宿舍D级人员发生了数起争执，在其每一次争斗都处于劣势的情况下，依然不断地试图动手。18:32分，对象遭到同宿舍D级人员的围殴，安保人员不得不到场以确保实验对象的健康。对象趁安保人员不注意时挣脱，企图突破收容，对象被当场处决。
“下次不把数据改那么高试试……？”——高级研究员███博士

“写入”测试
日期： ██/██/20██
实验对象： D-1318
写入数据： 将数据10（未知）从-70%增加到了70%
实验结果： 实验对象性格大体上不变，但开始表现出对周围人的不信任，包括有意躲避目光、说话遮遮掩掩、藏匿物品等。当问及为何如此时，对象起初拒绝回答，在得到命令后，称自己“处在不安全的环境下，应当自我保护”。

“写入”测试
日期： ██/██/20██
实验对象： D-44570
写入数据： 将数据10（未知）从-30%增加到了70%
实验结果： 实验对象性格上大体不变，也没有表现出自我保护欲，但开始热衷于出面劝架。在随后的跟踪调查中，发现对象参与进了每一次周遭D级人员的争执中，甚至有一次，出面劝说两名正吵的面红耳赤的研究员。当问及为何如此时，对象表示“我希望主持公正”。
"我找不出这个的规律。"——高级研究员███博士

“写入”测试
日期： ██/██/20██
实验对象： D-320
写入数据： ██████博士的性格数据
实验结果： 对象性格出现较大变化，但拒绝配合一切测试。以武力胁迫对象继续测试。对象的笔迹与部分习惯和██████博士相若，但对象表现出了对基金会的极度仇恨，并数次企图突破收容。起初容忍对象的行为，但在对象成功说服了一名安保人员并差点潜入SCP-███收容间时，将其处决；该安保人员受到处罚。调查显示██████博士对基金会一直忠心耿耿，从未有不良记录；随后旁敲侧击地采访了██████博士。
“这个玩笑不好笑，我绝对不会去违法，更不会把自己放到那个位置上。不过既然你非要问…假如我是一个D级人员，我大概会…不自由毋宁死吧。”——██████博士

“写入”测试
日期： ██/██/20██
实验对象： D-7366
写入数据： 将数据12（未知）从-10%增加到了70%
实验结果： 对象立刻开始大喊大叫无法辨认的音节，并尝试使用暴力突破收容。沟通和武力胁迫均无法成立，推测对象失去了理性思考能力。确认写入过程是正常完毕的。
"这是什么好可怕。"——Swater博士

“写入”测试
日期： ██/██/20██
实验对象： D-7370
写入数据： 将数据12（未知）从-30%增加到了-10%
实验结果： 写入完毕后命令对象取下头盔时没有反应。安保人员进行了防暴准备后取下其头盔，发现对象目光呆滞，对外界环境仅有缓慢且低程度的应激反应。对其脑部检查未发现明显病变，但灰质活性显著下降。
"不改变极性也会变傻吗，果然好可怕。"——Swater博士

“写入”测试
日期： ██/██/20██
实验对象： D-7374
写入数据： 将数据12（未知）从-20%减少到了-70%
实验结果： 对象性格出现较大变化，几乎不会主动说话，仅回答一部分提问。能够服从命令进行动作，但未接到命令时会无视个人生理需求呆滞地看向前方偏左约30°方向。被问及在看什么时，表示“没什么，就看看”。
"改低也没有变聪明而是另一个方向的傻？暂时找不出规律，怀疑和其他编号之间有相互作用。"——Swater博士

“写入”测试
日期： ██/██/20██
实验对象： D-15843
写入数据： 助理研究员R█████的性格数据
实验结果： 对象性格出现极大变化，多次情绪失控与自杀未遂。与其进行常规的心理会谈没有取得显著成效。值得注意的是其讨论自身时常常在短时间内论述自相矛盾的观点。
“怎么回事？我看不出数据样版的性格与实验对象的相似之处。是数据库出错了吗？”——Dr.██████

“写入”测试
日期： ██/██/20██
实验对象： D-11820
写入数据： 助理研究员R█████的性格数据
实验结果： 对象性格出现较大变化，并表现出典型的精神分裂症初期症状且自杀未遂。对象向心理医生袒露称“真主至善，而我都做了什么？但至善的真主教导我行恶，为什么？两个声音在我脑子里吵着，我受不了了。”
“似乎是与信仰有关？或许我们能试试对数据1的测试。”——Dr.██████

“写入”测试
日期： ██/██/20██
实验对象： D-11820
写入数据： 将数据1（未知，推测为信念）从+70%增加到了+90%
实验结果： 对象情绪失控并立即尝试自杀。
“这和我想的不一样，果然这个数据不能动吗？不，或许我能往另一个方面试试。”——Dr.██████

“写入”测试
日期： ██/██/20██
实验对象： D-11820
写入数据： 将数据1（未知，推测为信念）从+90%减少到了+50%
实验结果： 对象陷入抑郁，对所有刺激采取消极回应态度，同时表现出部分精神分裂症初期症状。
“我们又回到普通的结果了。但我想再试一次。”——Dr.██████

“写入”测试
日期： ██/██/20██
实验对象： D-11820
写入数据： 将数据1（未知，推测为信念）从+50%减少到了0%
实验结果： 对象表现出过度亢奋的利他意愿，但通常因其过于强硬的态度得到对象试图帮助的对象的消极回应。对象似乎因此积累了大量消极情绪并开始了自残行为。
“这数据果然不好动，但还是让我再做一个平行对照试验吧。”——Dr.██████

“写入”测试
日期： ██/██/20██
实验对象： D-15843
写入数据： 将数据1（未知，推测为信念）从+70%减少到了0%
实验结果： 对象表现出过度亢奋的利他意愿，但通常因其过于强硬的态度得到对象试图帮助的对象的消极回应。对象似乎因此积累了大量消极情绪并开始了自残行为。
“好消息。我们知道了数据1是0%的时候实验结果会有极高的复现性。坏消息，我彻底弄不懂这个数据是怎么回事了。说起来这样版是谁啊，申请不批下来我完全没头绪了，什么鬼信念能比████教还奇葩可以让这性格的人好好活着的？”——Dr.██████
