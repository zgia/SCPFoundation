星之眼营救任务事件
SCP-001-DE » 代号：Dr. Leo - Trabant /一个时代的终结 » 星之眼营救任务事件

以下文件包含事故“星之眼营救任务事件”的相关资料。严禁任何非5级安保权限或P/001/2特殊权限者查看


> 请确认您的安保权限以查看文件：“星之眼营救任务事件” 
• 安保授权已确认：已批准。

该事件发生于20██/██.██执行“星之眼”营救任务中。所有事件内容均由无线电音频传输记录。

星之眼营救任务事件音频副本
日期： 20██/██/██

备注：以下副本由营救人员与控制中心之间的无线电传输记录提取。

[日志开始]

Dr. Schrauer: 控制中心，准备工作已经就绪，Dr. Eberhart将留在飞船内返回。Agent Rehner和Agent Derel将会乘坐1号登月舱，而Dr. Abalew和我将会乘坐2号登月舱。

控制中心： 了解，执行任务。

Dr. Eberhart: 祝你们都好运吧！在下面注意点。

Agent Derel: 谢了。您搁咱船上也注意点，成不？咱还得回家呐。

Dr. Eberhart: (笑声) 了解，长官。

(登月舱下降15分钟后)

Dr. Schrauer: 我们现在得去前哨站M001-A4。

Agent Rehner: 那个前哨站建起来不是维修用的吗？

Dr. Abalew: 并不是，A4是M001建筑人员紧急补给用的。那儿恰巧有辆探测车，我们可以坐那辆车去M001。

Agent Rehner: 晓得了。

(停止说话两分钟。)

Dr. Schrauer: 好了，探测车在这。Abalew，去看看电池怎么样。Derel，帮忙和我一起看看装载怎么样，可以吧？

Agent Derel: 成。

Agent Rehner: 你们先忙着，我去前哨站里再翻翻。

Agent Derel: 知道了，去吧您呐。

(停止说话六分钟。)

Agent Rehner: 我找到一个终端，上面还有一些M001里头事情的记录。但是这上头一些技术性的东西我看不懂，看起来像什么设备出了情况。指挥中心？

指挥中心： 了解。给我们看看数据。

Agent Derel: 晓得了。

(停止说话七分钟。)

Dr. Schrauer: 指挥中心，我们现在去Site-M001。

指挥中心： 了解。请务必小心，我们并不知道那里发生了什么。

Dr. Abalew: 出现暴动可能性多大？

指挥中心： 极其低。地球上的员工会接受心理测试，Site-M001里面的也会。

Agent Rehner: 一个GOI？

指挥中心： 不确定。请专心于任务。

Agent Rehner: 好，那肯定，不好意思。

Dr. Schrauer: 你们可都听到指挥中心的话了。

Agent Derel: Schrauer？我们到M001那儿还得多久？

Dr. Schrauer: 大概还有一个半小时。

Agent Derel: 成，谢谢。

指挥中心，员工，内部交流： 和Dr. Eberhart的连接断开了，我们需要告诉营救人员吗？

指挥中心，O4-█，内部交流： 不，这只会造成不必要的紧张。继续试试恢复连接。

指挥中心，员工，内部交流： 了解。

(停止说话半小时。)

Agent Rehner: 喂，你们也看到那东西了吗？

Dr. Schrauer: 你说什么？

Agent Rehner: ……

Agent Derel: Agent Rehner?嘿，Agent Rehner?

Agent Rehner: … 呃，什么？

Agent Derel: 怎么了？

Agent Rehner: 啊……没事。

(停止说话一小时。)

Dr. Schrauer: 指挥中心，我们已经到E-9号侧门这儿了。

指挥中心： 好的，有什么情况吗？

Dr. Schrauer，私人交流： 呃，有的。Agent Rehner表现有点怪。他表现得心不在焉而且在车上还出现幻觉了。还有他……

指挥中心，私人交流： Dr. Schrauer?

Dr. Schrauer，私人交流： 他……他还一直在嘟噜着什么……他正站在E-9号门口……

指挥中心： 接着观察Agent Rehner，请小x……

Dr. Schrauer: 哦去你妈的！Rehner！Rehner！

Dr. Abalew: 把他按住，扑街！Rehner，你老人家在做什么啊！

Agent Derel: Rehner，把刀放下！Rehner! Rehner！

Agent Derel: 我勒个去！ (打斗声)

(停止说话一分钟。)

指挥中心： 请回答！营救小队？请回答！

Agent Derel: 这儿Agent Derel。

指挥中心： 发生什么事了？

Agent Derel: Agent Rehner (气喘吁吁)……把他的刀拔出来然后……捅了Dr. Abalew。然后我把我的刀拔出来然后……划拉断了他的氧气管儿……

指挥中心： 现在Agent Rehner和Dr. Abalew怎么样了？还有别的伤者吗？

Dr. Schrauer: Abalew和Rehner都死了，现在就剩我们。没有别的伤者。

指挥中心： 了解。请等待下一条指令。

Dr. Schrauer: 明白。

(停止说话一分钟。)

指挥中心： 请将尸体移到边门前，然后请进入Site。任务第一。

Agent Derel: 但是……他们尸体咋办，我们不给他们带回去？

指挥中心： Agent Derel，任务第一。尸体可以等会再管。

Agent Derel: ……

Dr. Schrauer: 我们现在进去Site了。

(停止说话十分钟。)）

(由于Site-M001内具有供氧系统且仍在正常运转，Dr. Schrauer与Agent Derel脱去了他们的太空服。)

Dr. Schrauer: 我们现在在探测车进来的这个门口，这里到处是……

指挥中心： Dr. Schrauer，发生什么事了？

Agent Derel: 大门口这儿到处都是……都是Site员工的尸体……Dr. Schrauer肯定吐了已经。

指挥中心： 了解。查看尸体。

Agent Derel: 尸体都被大卸八块了。内脏也没了……哦去他妈的……

Agent Derel: 有的看起来就跟……其他员工要吃了他似的……他们嘴里头还有肉……大多数人手里……还拿着简单武器。玻璃瓶儿啊，撬棍儿啊……那是什么鬼……指挥中心！

指挥中心： Agent Derel?

Agent Derel: 妈的，你们█的咋了把我们送过来啊？！这根本不正常啊！这些员工互相都……这儿到底咋了？！

指挥中心： 请稍安勿躁。我们不知道在Site-M001发生了什么，所以我们才会把你们送过去。

Agent Derel: 我们队里都死了俩啦！

指挥中心： 请您二位拿上一根撬棍，再往Site深处走一走。Dr. Schrauer现在情况如何？

Agent Derel: 你大爷的没听到我说啥吗？

指挥中心： 任务第一。

Agent Derel: 妈的！

Dr. Schrauer: 我……还行……过来，Derel。

(停止说话七分钟。)

Dr. Schrauer: Agent Derel?

Dr. Schrauer: 你……你能听到我说话吗？

指挥中心： Dr. Schrauer？发生了什么事？

Dr. Schrauer: Agent Derel站在那然后……不动——

(无线电通讯中断。)

(两分钟后，无线电通讯恢复。)

指挥中心： Dr. Schrauer，Agent Derel？能听到我们说话吗？

Dr. Schrauer: 可以……我可以听到。

Dr. Schrauer: Agent Derel拿着他的撬棍袭击了我，我挡住了那一下袭击然后……打到他头上了……他……他……他不动了……

Dr. Schrauer: 指挥中心！现在……现在就剩我一个了吗？！

指挥中心： 请冷静。指挥台，Dr. Schrauer，请您去指挥台。

Dr. Schrauer: 好……好……去指挥台。

(停止说话五分钟。)

Dr. Schrauer: 走廊上灯已经没了，但是我还能看到一遍遍写在墙上的字……“从我脑袋里滚出去”，“救命”……“他们都死了”……大概就是这些，都是血写的……尸体也是……到处都是。妈的，这儿到底怎么了？

Dr. Schrauer: 我现在到了指挥台的大门那儿了，需要密码才能过去。

指挥中心： 了解，请稍等片刻。

Dr. Schrauer: 说得比做得容易……这儿也是。

(停止说话一分钟。)

指挥中心： 密码是“███████”。

Dr. Schrauer: ……“███████”……成功了。门开了。我现在进去去指挥台。

Dr. Schrauer: 屏幕都还是亮的。这儿什么都没——不对，这儿有具尸体。看起来是……是Site-M001的守卫，头上有枪伤……旁边还有一把手枪。我觉得……他是自杀。

指挥中心： 了解。现在试着用终端恢复一下与地球的连接。

(停止说话一分钟。)

指挥中心： Dr. Schrauer？Dr. Schrauer，请回答。

Dr. Schrauer: (小声) 他还活着……

指挥中心： 什么？请重复一遍。

Dr. Schrauer: (大声，声音中带有攻击性) 他还活着！

Dr. Schrauer: 他肯定死了的！站起来，你他妈的！你在这儿是干什么？！

指挥中心： Dr. Schrauer，能听清吗？Dr. Schrauer！

Dr. Schrauer: 闭嘴！总算死了！你去死吧！

(砍肉声：Dr. Schrauer用撬棍击打守卫尸体。)

指挥中心，O4-█： 马上切断通讯！

(无线电通讯切断。Dr. Schrauer被宣布死亡。)

(“星之眼”营救行动于██:██宣告失败。)

[日志结束]

与航天飞机和Dr. Eberhart的连接在随后恢复。其被告知任务失败，剩余救援行动成员均已死亡。Dr. Eberhart回到地球后被授予指挥权以领导航天飞机。在归程途中，Dr. Eberhart抱怨其总做一个声音说着一系列句子的噩梦，Dr. Eberhart着重指出这些梦对其来说“不自然”。

以下词句为Dr. Eberhart于梦中听到：

你们会受到惩罚的。
我没有同意过你们的存在。
但是你们还是坚持要如寄生虫一般扩张。
你们之所以能够存在，仅仅是因为你们把我囚禁在了这里。
但这并不能救你们。
你们会受到惩罚的。
你们囚禁了我。
但是你们不明白时间不是一条河。
而是一场风暴。
你们无法囚禁我。
