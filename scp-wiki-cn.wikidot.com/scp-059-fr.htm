SCP-059-FR

项目编号： SCP-059-FR

威胁等级： 黄 ●

项目等级： Euclid

特殊收容措施： 在出现SCP-059-FR实体的情况下，应在所有本地电台上循环广播紧急信息，以警告附近居民不要离开居所。

伪装成救援人员并通过绳索相互连接的基金会工作人员，必须配备红外探测器与加热工具，在人口密集的地区以四人（或以上）为一组的方式进行巡逻，在任何情况下都不得分开。

任何处于户外的平民必须带往就近的建筑物内。

幸存者与目击者必须予以A级记忆删除。

SCP-059-FR-1受害者的尸体一旦被发现，必须予以回收。如发现平民尸体，则应对外宣称为野生动物袭击并且作停案处理。

描述： SCP-059-FR是一场直径为50公里的小型暴风雪，在每年9月初至3月下旬之间平行于北纬█0°以上的地区仅发生几分钟。卫星图像显示，与正常风暴不同，SCP-059-FR并非形成于海面。

SCP-059-FR内部的环境较为恶劣，环境要素包括但不仅限于：
- 暴雪 (100 cm/h)
- 暴风 (> 80 km/h)
- 低温 (< -30°C)

实体SCP-059-FR-1则以三到八个为一组出现于SCP-059-FR中。而试图以人工方式重现SCP-059-FR风暴的尝试均以失败告终。

SCP-059-FR-1是一个长约1.50米的爬行动物剪影，其并非物质并具有掠食性。除躯干外，SCP-059-FR-1个体没有任何后肢，而前肢末端有三个长约30厘米的爪。。并通过一种仍然未知的机制漂浮，以类似蛇类的方式移动。

SCP-059-FR-1实体似乎随机移动而不会相互影响；但一旦发现一个单独行动的超过一米高的大型生物，实体将进入追捕模式。

SCP-059-FR-1将会跟踪其锁定的猎物，距离持续接近，通常持续5到10分钟，直到后者注意到实体的存在。

受害者通常表现出恐惧的迹象。如试图逃跑，实体将开始追捕。目击报告指出，受害者通常会于风暴中迷路，可能是因为SCP-059-FR-1运动轨迹缺乏可见性。

当猎物耗尽体力时，SCP-059-FR-1的实体会开始用爪子攻击其腹部，造成严重撕裂伤，然后移开让另一个实体实施攻击（通常位于受害者的背后）。受SCP-059-FR-1的爪的影响的区域不论穿着衣服与否也会冻结。在实体攻击期间，实体经常发出爬行动物的叫声与冰块碎裂声。

由于天气条件与伤口的原因，受害者通常会耗尽体力而死。SCP-059-FR-1似乎并不会食用受害者，但即使在死亡后仍继续攻击后者的尸体。

如欲阻止SCP-059-FR-1的攻击，则必须进入建筑物，车辆等等一切密闭空间中，或者与他人结伴而行。.

SCP-059-FR-1的实体对枪械和刀具免疫，但尤为容易被高温和钝器所伤。当实体“死亡”时，实体分裂成云状粉末。

附录： 记录传输

SCP-059-FR显现
D-5248身着冬季服装，前置摄像头与录音设备。

<开始注册>

Dr. Adrian : D-5248，请上前去。

D-5248 : 好的博士，然后呢？

Dr. Adrian : 直接走下去就好，等我叫你停你就停

D-5248前行两百米。无异常。

D-5248 : 博士，确定要我继续吗？好鸡儿冷啊，开始有点不正常了……

Dr. Adrian : 观察周边，有看见什么异样吗？

D-5248 : 没有。

Dr. Adrian : 好，请继续。

D-5248进一步向前三百米后突然停止。

Dr. Adrian : D-5248，为什么停下了？

D-5248 : 你看见了吗？

Dr. Adrian : 什么东西？

D-5248 : 看见有什么东西动了！

Dr. Adrian : 冷静，它离你多远？

D-5248 : 我……大约50米！

Dr. Adrian : 好，转身，冷静，慢慢回到这里。

D-5248 : 谢谢博士。

D-5248返回两百米然后再次停止。SCP-059-FR-1距离该D级人员约3米。通过暴风雪很难察觉异常现象。热学探测表明SCP-059-FR-1所处位置的温度极低（-70°C）。

D-5248 : 我去他就在那，他出现了！

Dr. Adrian : 你能描述一下吗？

D-5248 : 我看不太清楚……是一条蛇，很大一条蛇！

Dr. Adrian : 问一下……

D-5248转头。SCP-059-FR-1距离前者几英寸远。这一次，实体清晰可见。其前腿漂浮于身后，似乎在看着D-5248。

D-5248 : （大喊）

Dr. Adrian : D-5248，你能尝试触摸异常吗？

D-5248 : 他太可怕了。妈的，现在有两个……

第二个SCP-059-FR-1实体显现于屏幕上。D-5248伸出手并试图触碰，然后急剧缩回。

D-5248 : （大喊）

Dr. Adrian : 怎么回事？

D-5248 : 我的手……好冷……我的手套……它失效了……

// SCP-059-FR-1突然改变方向，向D-5248方向俯冲攻击。D-5248开始尖叫。前置摄像头掉落雪地中。//

Dr. Adrian : D-5248，回答，我没看见你。

D-5248 : （大喊）

<结束播放>

2日后发现D-5248的尸体，死因为出血，冻伤与体温过低

附录2 : 实验记录

实验1
实验对象 : 无生命诱饵
实验结果 : SCP-059-FR-1无视诱饵

实验2
实验对象 : 普通的犬只。
实验结果 : 实验对象显示出恐惧的迹象并被杀死。

实验3
实验对象 : D-2914，并在冬季服装上配有锁子甲。
实验结果 : 实验对象死亡。锁子甲没有损坏，划痕径直从皮肤通过。尔后数个实验对象被发现处于同样的状态，防御措施似乎被无视了。

实验4
实验对象 : D-3715配备加热衣物。
实验结果 : SCP-059-FR-1围绕实验对象周旋而不攻击，然后在风暴结束时消失。

附录3 : SCP-059-FR的第一个实体于██/██/████，██：██出现在[删除]城市，导致██名平民死亡。这个城市通常都处于危险区域之外，导致该地基金会工作人员准备不足。
