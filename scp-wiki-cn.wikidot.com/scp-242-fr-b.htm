SCP-242-FR-B
SCP-242-FR » SCP-242-FR-B

SCP-242-FR-B正接受审讯。

项目编号： SCP-242-FR-B

威胁等级： 黄 ●

项目等级： Euclid

特殊收容措施： SCP-242-FR-B目前被收容在Site-Beth的东翼的一个标准的人形生物收容间中，完全防火并可承受高温。收容间中的家具也必须做出相应的调整。虽然SCP-242-FR-B不需要进食，但其仍然希望每天获得两份日常标准餐食，此请求已获得批准。

作为被生擒的同行组织SAPHIR的罕见成员1, SCP-242-FR-B可能掌握有关于基金会的重要信息，应当予以异常收容对待。审讯必须每星期进行三到四次。为方便审讯，审讯地点将在一座防火冷库里进行，并由两名特工监督。项目的收容间必须持续监视。值班人员被提醒SCP-242-FR-B认为自身是一个活的正常人类，因此使用标准枪支足以在发生状况时起到威慑作用。

SCP-242-FR-B是基金会敌对同行组织的一部分。尽管考虑到其高度合作态度，但它并不能被认为是可信任的，并且必须始终受到监视。还有迹象表明SCP-242-FR-B清楚其是如何被感知的，不论其信仰，其可能试图将这种感知散布出去。

描述： SCP-242-FR-B是一个白化，可活动并具有意识的人类骨骼，原名为Nathan Santoni。 SCP-242-FR-B身着一件白色大衣，上面缝有SAPHIR 的标志。SCP-242-FR-B不断放出温度介于1800和2100摄氏度之间的蓝色光线，光纤具有异常高的光通量（介于46 000 lm和51 000 lm之间）。项目身上的火焰经基金会实验证明是无法被抑制的，并且尽管事实上所有的肉体已经被消耗，火焰仍然继续燃烧。与此同时，SCP-242-FR-B的外套却没有受到火焰的影响，这种现象的原因目前尚不清楚。

SCP-242-FR-B能够移动，视觉听觉正常，能够进行口头沟通，尽管所有器官损失殆尽，其依然能够执行一般人类能够做的大部分行为。项目的肉体已完全消失，但其心理似乎非常稳定，很大程度上是由于他被诊断出的急性Filbuson综合征。事实上，SCP-242-FR-B完全了解其物理状态，感知的方式以及其存在对其周围环境的影响，但他似乎声称这一切是理所当然的。项目坚称自己是一个正常的人，自从他的意外发生后就不会改变（见“回收日志”），并且始终与普通人类一样具有相同的需求和弱点。

事实证明SCP-242-FR-B在基金会的控制下非常愤怒并不善，但在审讯中始终保持协作。尽管如此，项目被证实依然有所保留。

回收日志： 2月30日/██，在阿维尼翁周围半径30公里范围内的所有人口都被观察到停止活动，停下了手中所有工作并开始收集物品，然后前往位于城市东郊的一个着火的仓库。这种感染的来源很快被确定为“珊瑚”级认知威胁，项目被编号为SCP-242-FR。由专门处理认知危害，模因和末世危险组成的三支机动特遣队被派往现场以压制SCP-242-FR并对平民进行组织，但一些成员本身不幸受到SCP-242-FR。与此同时，来自全球超自然联盟的六支队伍与Project Malleus的地平线倡议队伍与现场的机动特遣队进行了合作，控制了场面。

SCP-242-FR的认知影响在Nathan Santoni主动进入并且没有认知影响的情况下与SCP-242-FR接触时结束。这导致SCP-242-FR近乎无效化，现在被分类为SCP-242-FR-A；与项目本源接触过的分类为了SCP-242-FR-B; 以及第二个实体的外观，具有原始SCP-242-FR的认知能力的SCP-242-FR-C。三个项目似乎本质上是相互关联的，不应将它们收容在同一地点。

附录01：事件发生前存储SCP-242-FR的仓库的安全视频摘录。

录像带1。 日期：2005年10月8日15:09。 时长：23分钟。
// SCP-242-FR位于仓库的中心。十几个人盘腿坐在火堆周围。尽管仓库周围缺乏科学设施，但所有人都穿着白色外套，一些还穿戴手套或实验室眼镜。期间，所有人都在闭眼睡眠。//

录像带4。日期：2006年4月19日21:57。时间：15分钟。
有8人盘腿坐在SCP-242-FR周围。第十分钟结束时，右边的第三人迅速离开。其他七个人保持沉默。五十三秒后，同一人重新出现，手持很多书籍，笔记本，三个显微镜和一个电脑键盘，然后其开始逐个扔进火焰中。许多人睁开眼睛，看到有人在说话，但没有人站起来。一名高级人员走进来用手枪向受影响的人员射击。

录像带9。日期：2006年6月13日18:04。时长：49分钟。
在第28分钟内，仓库除SCP-242-FR之外是空的。第二十九分钟，SAPHIR数名成员进入右侧，其中两人手持一个大型双柄罐。最终聚集者共计十人，在防噪音头盔的帮助下遮住耳朵。之后，其中一人打开罐子，而另一人仔细地拿出被认为是SCP-████-IT的东西。项目被扔入火焰中，跳起片刻，然后在消失之前开始融化。之前SCP-████-IT被认为是不可破坏的。

附录 02
附录02：██/██/██ SCP-242-FR-B审问记录

Dr. Gémini : 你好，242.

SCP-242-FR-B : 你也好。

Dr. Gémini : 今天，如果您愿意，我希望我们能讨论一下阿维尼翁事件。告诉我，你想变回你本来的样子吗？

SCP-242-FR-B : 不，谢谢，这我们已经谈论过这一千次了。我正常得很，在一堆永远烧着的炉子里成为一个行走的骨架打破了什么？十几种基本的物理规律？去他的吧，我很好。

Dr. Gémini : 我明白了。你，很痛苦？

SCP-242-FR-B : 永远。奇怪的声音萦绕在我耳畔，比以往更强烈。无论我走到哪里，在做什么，即便我在睡觉。

Dr. Gémini : 这会不会让你抓狂？

SCP-242-FR-B : 听，当我们选择忽略一个声音时，我们可以忽略它。无论是耳语还是嚎叫都无所谓。至于痛苦，你可以看见那些把手放在火中的傻瓜，他们专注于忽视这种痛苦，以及如果没有任何事情发生，那么将它除去还必要吗？ 比如当你不知道有火灾这种东西的时候。

Dr. Gémini : 有趣。很高兴看到你在所有情况下都表现良好。那么，你能告诉我那天晚上在阿维尼翁发生的事情吗？

SCP-242-FR-B : 我不知道从哪里开始。但无论如何，我不会告诉你任何你不知道的事情。

Dr. Gémini : 我们有所有尸体的名单，你不会谴责他们的。我主要是想看看……你的观点，可以这么说。因为我们不得不掩盖此事情。

SCP-242-FR-B : 哦，那是奇点，你知道…火是存在的，因为我们相信它是存在的。但你知道吗？我们已经使用了它，因为它可以有效地摧毁其他奇点。而要训练我们产生忽略声音的错觉，头脑中的命令则与自然相悖，这是有害的。只有少数受害者能从中得到极大的心理抚慰。我猜奇点需要关注，而不是因为我们缺乏信念而消失 ——我敢肯定，它决定扩大其行动范围，它会爆炸。

Dr. Gémini : 字面意义上？

SCP-242-FR-B : 从字面上看，是的。所见即所信。我怀疑人们为了达到这个这个地步必须看见它。没有什么比火焰更好的了。显然，人们已经上当了。这是我们尽力想要避免的。

Dr. Gémini : 嗯。你的看法是什么？

SCP-242-FR-B : 我们被困在一个燃烧的仓库里，被成千上万像昆虫一样被吸引过来的人们包围着，争相牺牲着自己和他们最亲近的人，而灭火器根本没用。我们做不了什么，我们尽我们所能做了一切，最终阻止他们进来。 然后，我们站在窗户边，外面非常热闹，我们开了枪。那些以牺牲他人为优先考虑的人是射杀的第一位。那些想烧毁珍贵的物品的人则是第二位。

Dr. Gémini : 这就完了？

SCP-242-FR-B : 你知道得比我多，对吧？他们花了不到10分钟就完全包围了仓库。我们只是一群人文主义者，根本没接受过一丁点射击训练

Dr. Gémini : 那就是你…发生在你身上的事情。你知道你在做什么吗？

SCP-242-FR-B : 不知道。一半的盒子在火中损毁或者被人群践踏殆尽，但无论如何我都会继续。我含糊其词地想，如果这个柴捆堆要求信徒牺牲他们最喜欢的信仰，那么如果一个没有任何信仰的无神论者动摇了，至少也会使他恼火。事实证明我是对的，不仅仅是“至少”。

Dr. Gémini : 你的介入已经使你变成了今天这幅样子，并进一步将最初的SCP分成两部分。

SCP-242-FR-B : 不，“奇点”将我融入了它当中，并以较弱的表现延长了它的主要认知危害的发生。 这就是SAPHIR所说的事情。地平线倡议说，一位天使从一开始就隐藏在那里，我用追逐无神论的方式将他赶走。全球超自然联盟所持有的那部分竟然是破碎的，那简直像是在看一个水坑中的倒影你能相信吗？往水坑中荡入一块岩石，你会遇到任何麻烦。你无法摧毁它，但你会造成混乱。 我是水坑中间的岩石。现在我在我内心的深处，我全身心地投入了一切，这是值得的。

附录03-允许查看。

9/9/██11:01，当SCP-242-FR-B无了解SCP-343的存在时自发地提出与SCP-343会面。
请求被拒绝。

9/9/██11:01，SCP-343自发地要求与SCP-242-FR-B见面。
已接受请求。

SCP-242-FR-B尽快赶往Site-17。SCP-343邀请SCP-242-FR-B与他一起喝茶，并要求没有人陪伴或监视收容间。当场的特工只能接受该请求。

这是SCP-343进入房间内的安全摄像头的记录:

时间：18:07. // SCP-343和SCP-242-FR-B面对面坐在一张桌子上，并配有茶具。尽管SCP-242-FR-B的属性不正常，但其周边环境似乎没有着火或受热。//

SCP-242-FR-B : [喝一口茶]这茶不错。

SCP-343 : 谢谢，Nathan。这是我最好的作品之一。

项目保持沉默了43秒。

SCP-343 : 我很抱歉看到我的一个子嗣被引到了这个地方。[喝一口茶]与我为敌。家庭关系越来越糟总归是有点难过的。但是，你，永远不会在SAPHIR看到我。

SCP-242-FR-B : [把糖倒进茶]哦，是的，你知道……我们经常去你家。你的房子。[刺痛]那里经常爆炸。

SCP-343 : 我知道。[喝一口]我有人在看着。

SCP-242-FR-B : 我不这么认为。

SCP-343 : 不，当然不是。

SCP-242-FR-B : 你没有那个资格向我展示，是吗？

SCP-343 : [笑]确实。

SCP-242-FR-B : 你沦落为假装称为一个神奇的男人，并且住着一个免费的公寓的地步了吗？

SCP-343 : 我很喜欢这个表情…… [喝一口] …“神奇的男人”。她对我的定义非常好。

SCP-242-FR-B : 我毫不怀疑，让大屠杀发生而不需要做任何事情就像打一次牌一样有趣。

// SCP-242-FR-B举起杯子喝了一口。但杯子因热而破裂。SCP-242-FR-B假装喝完了，但杯子的手柄仍在他手中。//

SCP-242-FR-B : 你看，没有真正地破嘛。

SCP-343 : 我只看到我的信仰。

28秒静默。SCP-343拿出第二个杯子，灌满茶并在其中放入一块饼干，递给SCP-242-FR-B

SCP-343 : 讽刺的是，你批评了面对犹太人大屠杀时我缺乏行动。但在你的一次小型干预活动中，你却也不想在街角碰到我。

// SCP-242-FR-B拿起茶，慢慢地将其拉向右边，并且将内容物倒在地上。此过程修复了SCP-343。//

SCP-242-FR-B : 噢

SCP-343 : 我发现一个绝对不希望的事故。我猜……是像大爆炸那种？

SCP-242-FR-B : 你可以再次给我倒茶吗？

SCP-343 : 乐意之至。

SCP-242-FR-B : 好。一杯如此热的茶，即使你不能喝它。

SCP-343 : 我知道你会这么说。

SCP-242-FR-B : 一切说说都很容易。所以继续吧。经典人物，如果你是无所不能的，你就没有造不出的东西……

SCP-343 : 如果我无所不能，那么我什么也不能喝。确实是一个经典，即使在我看来，它……

SCP-242-FR-B : 别转移话题。

SCP-343 : 这是命令吗？
