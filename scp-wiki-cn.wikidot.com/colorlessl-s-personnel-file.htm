进入SCP-CN-408收容室以安装控制系统的“Colorless”.L，由breaddddd绘制

姓名：“Colorless”.L

性别：女

安保许可等级：2级，参与特定工作时可授予临时专项3级权限

当前职务：Site-CN-91实习工程师

专长：应用奇术学，电子奇术工程

简介：“Colorless”.L，毕业于████大学计算机科学系，于攻读博士学位期间因发表文献中涉及部分奇术计算原理而受到关注，并于毕业后被招募加入基金会。后续检测证明其为奇术适应个体，因其可调动的EVE粒子量过于微小而未在此前被发现或登记。

Colorless同时表现出较强的记忆力、精神阻抗能力与对██式记忆删除措施的高顺应性，至今为止的██次记忆删除均未导致任何并发症或后遗症（包括任何程度的原记忆遗留或复发），且其所受的精神创伤亦能随相应记忆的删除而完全复原。因该性质与Colorless表现的快速学习能力，Colorless被允许暂时记忆高保密等级信息，以弥补相应任务的人手不足问题1。Colorless已签署配套协议，于此类任务完成后无条件同意接受涉密时段的指向性记忆删除。该协议原件应保存于其所在站点的人事主管处。

Colorless通常需要一定的时间了解其他员工的喜好并学习与对方融洽相处的方式，这导致她在进入新环境时往往显得较为内向，并经常因斟酌自身用词是否得当而苦恼。

Colorless有广泛尝试各类桌游或电子游戏的爱好，但并未在任意类型的游戏上展现过超出平均水平的技术。同时有独自制作小型手工艺品的爱好，但本人拒绝展示任何完成品。

+ 需要4级权限
- 隐藏

“Colorless”.L为一异常人型个体在加入基金会时登记的代号。当基金会应其要求删除该个体于加入基金会前的生活记忆时，个体显现出异常性质，导致其基因序列与生理表现型均出现大幅变化。测序表明其重组后的基因序列不与已知任何其他人类个体有三代之内的亲缘关系。

[WARNING：资料待补充]

如该个体显现出察觉自身异常性质，或出现异常性质被激活的倾向，应立即对其执行时段性数据删除，并更换其工作环境。

脚注
1. 包括但不限于奇术阵列、信息危害载体及[数据删除]的运行状态判别与现场维护，或对外勤调查提供现场信息协助

于音乐沙龙演出前的POI-014-1324，由dofafa绘制

姓名：Consolas Lake

性别：男

所属组织：GOI-014“Are We Cool Yet?”

关注度：低

状态：未确认

简介：根据被截获的资料，确认POI-014-1324为项目计划书 2024-239: “加冕仪式”与项目计划书 2014-2390: “Mr.Sue”的提出者。根据以上文件，POI-014-1324被推定为奇术适应个体，且对其他关注组织及现实稳定/调整技术有初步认识。

待确认的资料：由于以下资料互相冲突，暂无法确认POI-014-1324目前的状况。

  • 于GOI-014的大型异常艺术集会中进行行为艺术演出，并于过程中意外死于枪击
  • 于一小型音乐沙龙中与POI-014-008“The Performer”进行合作，并于演出结束后死于割喉。据悉，该事件可能与███计划的实施有关。
  • 于香城圣帕特里克音乐厅进行巡回演出，并于一次攻击性表演中死于超自然技术袭击

我知道我的热情总是容易耗尽，但至少它现在仍灼灼燃烧。

♫ YELLOW
坠落之日她哭泣不止，因她见了所有人都被痛苦惩罚。

关于本人：

  • ColorlessL，又称C.L./CL/无色，目前为大二计算机系学生。但是代码能力超乎寻常地弱。
  • 出没于创作指导群，有精力时出没于草稿区，爱好挖出各种自己也知道自己没办法填好的坑。
  • 会主动给喜欢的文章进行校对捉虫之类的小工作。对能让我提起兴趣的草稿会尽力提建议。
  • 在描述主观感受的时候会使用各种*非常奇怪*的词汇。曾经把某名作者描述为一种食物并促使自己出去买了一份。
  • 和下层叙事Colorless一样喜欢游戏，不过我菜得绝望。
  • 和下层叙事Colorless不同，是个肥宅，并且远没有她勤快。
  • TRPG爱好者，各种规则似乎都挺好玩。虽然只开过CoC6th/7th和DnD5e。
  • 正在了解古典音乐。

关于对基金会作品的爱好：

  • 喜欢以角色为中心的文章，但是谜一般地对抒情风格的SCP文档不非常感冒。
  • 喜欢奇术设定，尤其喜欢GOC将奇术与音调和宝石联系的命名，十分浪漫。对音击术也有好奇但因为本人手残不会打音游而始终不太了解。
  • dado炒鸡可爱是的。
  • 喜欢核心概念和编程相关的东西，因为亲切。
  • 相信“所有的点子都是可写的，只是写好的难度有所差别”。
  • 有一个写遍GOI格式的妄想，但是写的最多的还是项目计划书。
  • 有一个写001提案的妄想，还没成。
  • 最喜欢的文档是SCP-4231，故事是平行世界和蒲公英酒。中分的话曾经是SCP-CN-899和睡过头啦。现在是SCP-CN-790和某篇笔记。
  • 这里有许多很可爱也很有才华的人，我喜欢你们如同喜欢凌晨的鸟鸣。

关于“Colorless”.L：

  • “Colorless”.L的设定某种意义上是为了让她能在几乎任何背景下可以出场而写作的。毕竟我也不知道我下一篇会写出来什么东西。
  • 研究员C.L.的异常人形个体讲座：低休谟个体的特征及辨识方式：Colorless的串场工作记录，普通到毫无特点的谈话。毫无自觉地说漏嘴了一些内容。
  • SCP-CN-408：其实也没什么关系，但是这或许可以是一次相遇？如果想写故事的话可能会从这里开始吧。

关于Consolas Lake：

+ 大量胡言乱语警告。
- 你已经被警告过了。抱歉。
  • 基于项目计划书 2024-239：“加冕仪式”与项目计划书 2014-2390：“Mr.Sue”，在凡人终死竞赛中作为参赛角色登场。
  • 初始的灵感是近乎疯狂的质问和爆发，询问一切的意义，然后得到笑声和“没有意义”的回答。他走得太远了，他思考的太偏而得到的回答太少了，从人群中把自己割离出来时他彻底地迷失了。他被自己的认知和“清醒”所折磨，太多的自由让他无所适从地痛苦着。
  • 一场并不Cool的演出：他把自己的生死交给陌生的观众们，让他们作出选择：给他足以活下去的惊喜，或者将他连同他的思想一同抹去。这并不Cool。
  • 噫場音樂沙龍.mp4：他迎来另一次生命。低休谟场对他而言是服从与交付自身的象征，演奏家如此限制他并引导他，让他和自己的现实重新组合：倾尽一切地感受那些曾经被他视为荒谬并弃去的事物，接纳那些他不懂得欣赏但却确实存在于世的事物，和他身为凡人的每一种本能重新相遇。感受，体会，表达，理解，这绚丽怪奇的世界中行走着愚昧却赤诚的生命们，现在他开始懂得为此陶醉。感谢Reverberate_a与Danza的写作帮我找到了这种感觉。
  • 三千世界，只在眼下 红莲业火，燔烬须弥：他的另一种可能性。彻底地飘离现实，挥洒攻击性和直感，在毁灭的浪潮中漂流着起舞。感谢Simon Arran的写作，我尚有很多需要学习的内容。

项目计划书 2024-239：“加冕仪式”

发布日期：04 Jun 2019 15:31 评分：+31(+31/-0) 讨论：9

+ 他的世界无边无际，但是却被无形的墙紧紧包围——他自己的思维。
- 然后他可以在新生的喜悦中回到这个平凡的世界，戴着充满廉价感的王冠。

留在站点上的第一篇文章，第一篇AWCY的GoI格式，同时也是Consolas Lake的起点。

写作这篇文章是一种很奇妙的体验。那些呓语、那些回忆、那些在梦醒时分成为木偶的人影、在层层叠套的连环梦境中随着不断膨胀的恐惧一次次惊醒，全部都在那一天流淌出来。

我此后将它记作“一次灵感爆发”，目前为止尚没有过第二次。这很快乐，但我总在期望我能用其他方式超越它。
2019/9：嗯，超越了，大概吧。


项目计划书 2014-2390：“Mr.Sue”

发布日期：15 Jun 2019 04:31 评分：+21(+25/-4) 讨论：12

+ Cool？我只看到了破坏和叛逆和随便什么东西的老调重弹。
- 认为自己是高雅而与众不同的？每个普通人都这么想，你也一模一样，毫无新意。

第二篇AWCY的GoI格式，关于一组困扰我许久的问题：

何为艺术？何为创意？什么是有价值的而什么是被包装的价值？我应当为被商业包装所欺骗的自己后悔吗，即使那些包装的确曾经给我带来快乐？我应当去追寻更高的存在吗，即使其中绝大部分是优越感的需求？

把这么多暴戾的疑问以我能做到的最为攻击性的方式抛出，并将那点“自身非同凡人”的骄傲也舍弃之后，就是这篇文章了。

虽然文中所指的对象是AWCY，但其实写作的时候多多少少想的是在基金会的创作。当然，我仍抱有期待。


C.L.的异常人形个体讲座：低休谟个体的特征及辨识方式

发布日期：21 Jun 2019 17:28 评分：+15(+18/-3) 讨论：1

+ 低休谟个体，又称被现实扭曲者，是自身休谟指数低于环境的异常人形个体，一般为100~105/20~60。
- 你的任务是发现他们。

一篇毫无根据的讲座，原来是另一篇故事的基础设定，但是现在反倒是故事自删了讲座留了下来。

如果把休谟指数比喻成现实的“柔软”程度，那么一种足够低休谟的载体会在基金会的行动中有什么用处？如果那种载体是血液呢？

作为讲座这篇的语言还很不成熟，正在纠结是否要重写重投。


一场并不Cool的演出

发布日期：27 Jun 2019 14:58 评分：+18(+18/-0) 讨论：6

+ “我们滥用一切手段冲击观众的官能，直到审美疲劳把我叫醒，告诉我这他妈糟透了。”
- 他的颅腔中迸出五彩斑斓的颜料，洒落在层层叠叠的廉价纸板上，流淌着试图拼出他未完的演讲词。

凡人终死竞赛第一轮参赛文章。第一次正面——好吧，其实也不是那么正面地描写Consolas Lake。

这是“Mr.Sue”的表演现场，也是对Mr.Sue的再解释：向这些除了伤害与残酷之外一无所有的“艺术”开战，然后像他所最厌恶的平凡的无名小卒那样死去。

我发誓我写这篇的时候完全没有想到在这场竞赛的之后轮次会发生什么。


SCP-CN-408 - 信号蝶

发布日期：06 Jul 2019 14:09 评分：+16(+19/-3) 讨论：6

+ 这里没有摘要。
- 只有蝴蝶。

能在三位数编号空位所剩无几的情况下拿到CN408这个编号，写作一篇基于SCP-408的文档并保持存活，对我而言真的无比幸运。

想法毫无疑问地来自SCP-408，然后SCP-CN-673告诉我在已有点子上再发挥的可能，SCP-CN-1670启发了一个不错的衔接片段，然后就有了这篇文档和这个小小的故事。

末节协奏曲

发布日期：03 Aug 2019 12:58 评分：+23(+23/-0) 讨论：9

+ 说到底，什么级别的议程需要一个月那么久的时间呢？
- 信纸悠闲地将自己折叠成一只飞鸟，包裹着那颗搏动渐弱的心脏飞向远方。

与凡人终死竞赛第四轮同步发表。对我而言，这篇能够按时完成已是奇迹。

起初的想法在赛程公布的时候就已经有了：因为最后一轮的角色并没有被其他选手击杀过，所以要以场外身份用一篇使其死亡的故事完成这种“补偿”。本来这个任务没有那么难的，但是因为目标角色之一是苍绯.aic……这不就必须写成大规模事件了吗。

然后就是这次的主线，协奏曲计划，拨动现实的弦线，献祭死亡以导向新生。其实这方面的处理是比较粗糙的，仅仅是勉强使逻辑闭合而已。可能又是未来的一个天坑吧。

关于角色。我的确十分喜欢以角色为中心写作，导致我写这个故事时绝大部分的时间都用在阅读各位角色之前的故事上……直接结果之一是最后留给真正动笔的时间所剩无几，并且这篇的语言风格也在各个段落被割裂了。篇幅分配和描写深度肯定是有差别的，毕竟我没有能力真正地同样深入所有角色，有一些原本在计划里的角色也因为我自己的认识不够而在终稿中一笔带过或直接删除了。

总之，这篇文章基本上是一篇面向全程关注凡人终死竞赛的作者和读者们的故事。感谢大家对这次竞赛的关注和呈现的作品。

致那有名或无名的，平凡或绚烂的，全部的幻想造物角色/生命。
