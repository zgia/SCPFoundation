SCP-136-FR

项目编号： SCP-136-FR

威胁等级： 黄 ●

项目等级： Safe

特殊收容措施： 物品通常存放于一个长20cm的木盒内，特殊情况下可存放在由4个旋转组合锁和1个机械锁密封的保险箱内。项目目前收容于Site-███。

由于实验变量过大，因此项目访问受限。而经项目负责人探明剂量的任何物品则立刻获得访问许可。

描述： SCP-136-FR是一个16厘米长的由钢与玻璃制成的注射器。尽管其设计相对较为老旧，但实际上使用年龄只有几年。

该仪器在20██年在法国北部的███████发现。一名名为██████████████████24岁的男子在用这个注射器杀害了██人后被警方逮捕。据报道，由于经常摄入强效药物，目击者描述嫌犯心理极其不稳定，与此同时，受害者通常也过度摄入过大剂量的药物。但是，嫌犯所描述剂量似乎远低于所述物质的致命极限剂量。

嫌犯目前拘留于警局，注射器与存放箱被送回Site-███。

在现场，K████D██████博士根据嫌犯的口供进行了一系列的初步实验。

因此，在20至30岁之间的所有男性受试者中，对D-4635，D-3895和D-1477进行3次不同物质的注射。 给予的剂量（分别为██毫克，██毫克，██毫克）远低于人类的平均致死极限值。此外D██████博士还确保受试者对注射物质没有过敏反应。

一个小时后，三名受试者已经死亡。在尸检中，毒理学检验显示了高达致死量10倍的剂量，导致受试者死亡。

D██████博士随即提出假设，即SCP-136-FR能够以未知方式指数级增加注射剂量，即使剂量非常低，也会达到致死剂量。

随后D██████用不同物质进行了其他实验（见附录136-FR-01）

其结论表示，即使对于人类大剂量的无害物质，假设也是成立的。

注射器的指数级增加效应似乎只能用于注射，在活体外吸收和排出的量在体积或质量上不会有变化。

J████教授负责物理实验，以确定注射物自我增加的机制。

除了在注射器的金属结构内发现有1微米数量级的印刷电路，研究进展极为缓慢。

在██个月的研究与设想之后，J████教授放弃对SCP-136-FR的研究。

███████博士提出使用SCP-136-FR作为处决D级人员的方法。此项目Site-██的管理层中讨论有一段时间，但是次提议最终放弃，目前使用的手段已足够便利与无痛，136-FR仍然是一个需要被收容的异常对象。

附录 136-FR-01 :

实验 A - ██/██/20██
实验对象 : D-2569, 男性, 62 kg
程序: 注射███mg的维生素C
结果 : 实验对象死亡
分析 : 血液中浓度接近11,900毫克/千克，为致死剂量的1.5倍。

实验 B - ██/██/20██
实验对象 : D-2103, 男性, 83 kg
程序: 10毫升水
结果 : 几秒之后，实验对象声称身体疼痛，尤其是头部疼痛。30分钟后，实验对象开始呕吐。90分钟后，实验对象死亡。
分析 : 尸检结果显示死于低血钠症。

附录 136-FR-02 :
SCP-136-FR的来源问题由██████博士提出，调查已经开始。两名特工被送往监狱审讯██████████████████。其中一名特工擅长面部微表情心理学，以便在面试过程中获取最真实的信息。
审讯在监狱的审讯室内进行。审讯记录如下:

受审人： ████████████ ██████, 男性，24岁。
审讯人： 特工██████ (审讯人) 与 特工██████████ (心理学观测)
前言： 对犯人进行审讯以获得更多██/██/20█时发现SCP-136-FR的信息。
审讯人 : 你好，██████先生。我们在这里和我的同事需要问你一些问题（拿出SCP-136-FR的照片，交给受审人），需要你如实回答。有其他问题吗？
受审人 : (盯住图片数秒) 没有问题。
审讯人 : 好。你认识这张照片中的东西吗？
受审人 : (点头，但保持沉默)
审讯人 : 好。那么告诉我们你是从什么渠道获得这样东西的？
受审人 : (低声) 她把他们全杀了。
审讯人 : 你们能放了我吗？
受审人 : 她把他们都杀了。我知道他们，因为…（似乎是在回忆）很长一段时间。他们是我的朋友，甚至是我的家人。
审讯人 : 我明白了，但是请告诉我…
受审人 : (眼中有流泪倾向) 他们不应该死去。他们知道他们在做什么。除了我以外，他们都在用这个。我从不喜欢分享我的注射器。

审讯人 : 但他们是怎么得到这个注射器的？ 我指的是……那些杀了他们的人。

受审人 : (似乎注意到了特工的存在，直起头来) 她在那里。
审讯人 : 什么那里?
受审人 : 是。我们出去买东西，当回来的时候，她就在那里。在一个不错的木箱里（微笑）。就是这样。
审讯人 : 上面没有铭文？没有说明书之类的吗？
受审人 : 不，不不不不不不不(猛烈摇头)
审讯人 : 很好。感谢抽空谈话，██████先生。
受审人 : (再次流泪，并低声) 求求你了……

审问后，对注射器及进行指纹比对搜索，但无结果。调查目前仍在进行中。
