SCP-089-DE

在抓捕SCP-032-DE时其携带的SCP-089-DE照片

项目编号： SCP-089-DE

项目等级： Keter

特殊收容措施： SCP-089-DE目前无法被限制。因此需监视所有媒体与“机器人”相关的消息。一旦确定消息走漏，需派出机动部队MTF-DE2-𝕴（“灯链”）调查。一旦确认SCP-089-DE与/或被其影响的机器，自动授权使用重型武器和电磁脉冲炸弹对项目进行限制。

一旦成功捕获SCP-089-DE和/或者受影响的机器，必须严格将其同其他任何机器隔离，直到完全平息后再使用机械异常专用运输单位进行运输。

一旦抓获SCP-089-DE，应立即运输至站点-DE17为SCP-089-DE准备的标准人形生物收容间。

由于SCP-089-DE转化过的机器存在巨大差异性，故无统一配置应对，已抓获的项目需使用单独的SCP分级。

描述： SCP-089-DE为一个人形的机器人，其身高约170厘米，其名字音近于“Hod”。SCP-089-DE的面容，脖子，手臂，手掌都类似正常人类女性。由于SCP-089-DE总是穿着一件连有浅粉色围裙的宽松黑色连衣裙、一双黑色的及膝长筒靴和一顶白色卷曲的假发，其身体的其余部分不可见。 然而，分析其服装受损的视频录像显示，其下半身、上臂和大腿与其身体可见部分类似。 另外值得注意的是，SCP-089-DE在脸上戴有一张黑蝴蝶面具。

截至目前为止，无法收容SCP-089-DE，但基金会与其产生多次接触。这些记录表明SCP-089-DE具有高度智慧，且似乎无需能源供应。SCP-089-DE可以视觉，听觉和触觉的方式感知周围的环境，并且具有女性的声音，并以被相关人员描述为“机器人般的嗡嗡声”的语气说话。 SCP-089-DE身体素质暂且未知，但是在正常情况下，SCP-089-DE能够用一只手提起装备齐全的MTF士兵，并将其投掷4米。 然而，其内部构造似乎非常敏感，因为手枪射击甚至某些情况下的拳击都能对SCP-089-DE造成可见伤害。 然而，由于其重量较轻，而便于其快速反应，从而使其能逃脱大多数攻击。 据观测，SCP-089-DE全速可达到高达60公里/小时的速度。 电磁脉冲武器似乎对SCP-089-DE没有影响。

SCP-089-DE只要和目标物体产生物理接触15秒即可使任何机器产生异常性质。不管是接触到机器的一部分还是隔着其衣服都不影响该效应。受SCP-089-DE影响的设备不能再被使用并开始自主移动。 在97.7％的事例观察中，受影响的机器也改变了其的形状，并且转化为类似人形的形状或动物的形状。 在这个过程中，质量没有明显的损失或增加，材料也没有改变。 如果设备处于锚定位置，其将会挣脱锚固，除非锚固将其连接到另一机器或底架，此时这些装置都会被包括在转换过程中。

SCP-089-DE转化后的载重汽车。

经SCP-089-DE转化过的机器具有人工智慧，可通过视觉、听觉和触觉感知其所处环境，并能够进行语言交流。类似SCP-089-DE本身，这些被转化设备似乎不需求任何能量，但机器部件仍然会出现常规磨损，且需要更换。由于被转化的机器执行了SCP-089-DE下达的命令，因而其似乎是一个指挥者，但有观测记录显示不同的机器对同一指令的差异执行，进而显示了其一定程度的自主权。各机器根据其原本的设计，对不同的武器和环境影响具有不同的抗性和弱点。此外还需注意的是，在任何情况下只有使用电磁脉冲武器才能暂时将该类机器“击败”，除非其对电磁脉冲具有抗性或类似其他情况。

受SCP-089-DE特性影响机器摘录
关闭

被转换机械： 摩托车，本田CB500F型

变化： 外观转化为长度约两米以上的猫科动物。其“腿部”与车轮连接，因而仅允许滚动移动。

被转化机械： 汽车，帕萨特B8型（3G型）

变化： 该项目转化成一种能让人联想到河马（Hippopotamus amphibius）的外形。虽然外表肥胖，但项目可达到极高加速度。

被转化机械： 焊接机器人██████ ██████ ███

变化： 机械臂与共六台自动机床构成的焊接室的一部分为所有被转化的机械。项目形状类似于一只步行虫科（Carabidae）动物，长约3米。该项目能够从其腿部末端发出电击。

被转化机械： 豹2式坦克

变化： 该项目转化为高约5米的类人外形。原履带形成额外的腿部装甲。火炮被移至后背，但是有需要时可以通过收缩头部而转向前方。双侧机关枪位于项目前臂上。

被转化机械： 斗轮式挖掘机，SRs███型

变化： [数据删除]

SCP-089-DE转换的机器的另一个特点就是能够向其他设备提供与其本身具有的异常属性一致的异常属性。通常是在其目标中安装其机器部件来实现的，但也观察到其他方法（例如参见SCP-███-DE）。 然而，以这种方式转化的设备，并未观察到形状的变化，并且似乎不能传递其异常效果。

SCP-089-DE及其转化设备的能力显然是相关组织“玛瑙盲者共同体”的一部分。其并不试图直接伤害正常人类，但其针对基金会及其他被其认为是敌人的团体的部分行为与搏斗已经导致人员间接受伤或死亡。 由于SCP-089-DE总是与至少三台经转换战斗机械一同移动，且其中至少有一台免疫电磁脉冲武器，因而SCP-089-DE至今尚未被基金会捕获过。 暂时未知SCP-089-DE是否可以在战斗后修复其遭受的伤害。

SCP-089-DE在事件DE / B33期间首次出现。在研究小组恢复25份25号部门经手的“雅典娜项目”文件前，基金会一直仅能通过观测资料收集有关该SCP的信息。 这些文件描述了与SCP-089-DE性质类似的异常个体，可假设二者为同一实体。

从25号部门恢复的SCP-089-DE文档
关闭

项目建议书056/42

为解决监视德意志民主共和国共产主义公民而产生的不断增加的数据量，现在此建议设计一台数据处理机器。理论上一台计算机即可胜任，但预计以当前技术水平需约80年后才能发展到该水平，故而有利用超自然现象而达到该目的的想法。Eisen硕士已为该目的起草一个大致利用向经处理载电结晶硅中充入人类记忆以产生思维的计划。明细内详。

柏林，1967/05/19



国家安全部


25号部门

批准雅典娜项目

兹批准雅典娜项目。授权Eisen硕士使用██████Eisen女士的记忆。

签名：Konstantin Mader长官

德累斯顿，1967/09/09



国家安全部


25号部门

雅典娜项目研究记录

我终于开始使雅典娜走上正轨。感官装置工作毫无瑕疵，其能及时反馈其视野内变化与杂音。但目前其智力仍类似婴儿，但我确信两周后即可令其达到成人水准。

签名：Eisen硕士

柏林，1967/09/13



国家安全部


25号部门

452状态项目报告

Eisen硕士拒绝了考量其妻子██████ Eisen昨天因肺炎去世而授予的额外假期。因而兹命令████████████同志对Eisen硕士进行秘密监视。其所有心理和情绪状态变化均需向我汇报。

签名：Konstantin Mader长官

德累斯顿，1967/9/21



国家安全部


25号部门

雅典娜项目展示

雅典娜现已达到理想水平。语言流畅，思维集中。现已确定无需外部供应能源，因而发电机关闭。雅典娜显现出操纵机械元件的能力，因而可以为其安装手臂。具体展示于1967/10/03。

签名：Eisen硕士

柏林，1967/09/22



国家安全部


25号部门

雅典娜项目研究记录

展示日期由我与秘书负责。

签名：Konstantin Mader长官

德累斯顿，1967/09/27



国家安全部


25号部门

██████ ██████监视报告

Eisen硕士为雅典娜装配了类似人类头部与上身的框架，随后其在框架上被覆了硅胶套，并将雅典娜的抓握用具塞入硅胶手套中。这些均为肉色色调，并类似于██████ Eisen女士的手部与上身。在对Eisen硕士的询问中，其表示其为了展示才如此做，且因雅典娜仰仗其妻子的记忆才成功，而故意选择了这张脸。怀疑其对雅典娜有情感关系。

德累斯顿，1967/10/04



国家安全部


25号部门

雅典娜项目新指示

既然展示圆满结束，请停止对此讨论并开始实验序列。需开始每周向25号部门提交报告。

签名：Konstantin Mader长官

德累斯顿，1967/10/10



国家安全部


25号部门

雅典娜项目研究记录

实验失败。雅典娜确实可处理指定数学问题，但无法独立得出解决途径。

签名：Eisen硕士

德累斯顿，1967/10/17



国家安全部


25号部门

雅典娜项目研究记录

演算能力未增强，需进一步调试。

签名：Eisen硕士

Dresden, den 24. 10. 1967



国家安全部


25号部门

雅典娜项目研究记录

记录到小幅改善，但雅典娜仍然缺乏组合技能。需进一步调试。

签名：Eisen硕士

德累斯顿1067/10/31



国家安全部


25号部门
雅典娜项目研究记录

雅典娜可能已告失败。虽然雅典娜有一个可接受的智力水平，但似乎基本不能整合专业知识。推荐中止项目并重启。

签名：Eisen硕士

德累斯顿，1967/11/03



国家安全部


25号部门

██████ ██████监视报告

Eisen硕士似乎并未停止实验。相反其与雅典娜进行了长达数小时的交谈并称其为██████。此外该项目似乎还具有之前未提及的转换其他机器并使其独立的能力。需触摸至少十五秒。Eisen硕士将这些创造物称为他的“孩子”。确认情感关系。

柏林，1967/11/07



国家安全部


25号部门

雅典娜项目新指令

由于缺乏明显进展，现将Eisen硕士调离雅典娜项目管理。Krone硕士将接管该项目并修复迄今的任何错误。

签名：Konstantin Mader长官

德累斯顿，1967/11/11



国家安全部


25号部门
雅典娜项目研究记录

雅典娜有一些不必要的功能，例如触觉，Eisen硕士很可能疏忽了。作为简单的专项数据处理用机器，雅典娜亦无必要拥有现有情绪。将在下周进行删除。

签名：Krone 硕士

德累斯顿，1967/11/14



国家安全部


25号部门
雅典娜项目事件报告

Eisen硕士试图于今日8:27借助撬棍进入雅典娜项目。Eisen硕士惊讶于Krone硕士此时正在进行删除雅典娜情感系统，并对其后脑部位造成撕裂伤。Eisen硕士试图将雅典娜核心元件结晶硅从机体取出，但被恢复平衡的Krone硕士推开。于是两位硕士开始扭打起来，最终Krone硕士下定决心并打破Eisen博士的头部，然后雅典娜突然开始放声尖叫，并开始将其机身与仍然还处于连接状态的发电机一并转换，从而得到人形躯干以便于其移动。此外其也能袭击Krone硕士[文件缺失]最终其眼区的硅胶面具严重受损。在转化了建筑内电梯后，雅典娜趁乱逃走。在德累斯顿██████-█████████街的设施经火完全损毁，无法使用。Krone硕士目前被控犯故意杀人罪。但是我们的工作人员的证词很快以正当防卫理由导致其无罪释放。雅典娜去向现暂不可知。

柏林，1967/11/15



国家安全部


25号部门
