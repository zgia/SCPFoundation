SCP 维基上的用户工具

这是什么：

本页面组织了一些扩展extensions、工具tools、脚本scripts，可以改善并有很大期望能地提高你在网站上的体验。一旦某个工具被展示在这里，那么它就应该经测试后可在所有的浏览器里运行。如果不，则向Tech Staff报告任何存在的bug，然后由他们对其进行修复。

这不是什么：

这些玩意儿既不想去玩弄你的浏览器、你的电脑、或者你的生活，也不大大滴谋图窃取你在沙滩上的那些除了穿着小小天竺鼠和水果的衣物外，你身上便啥也不剩的图片。讲真，除了在SCP维基上改善用户体验外，它们不会做别的事情。



扩展

扩展对于浏览器来说，是个完全成熟的附加组件，会自动更新，可在浏览器的扩展中心和/或应用程序商店里安装。

用户自定义脚本Userscripts用户自定义脚本，是能运行在你页面上的单个 JavaScript 文件，基本上就是一个更小的扩展。而这些文件，必须通过脚本管理器（比如油猴）去手动安装，也都不会自动更新。

如何在浏览器中启用并运行用户自定义脚本

浏览器并非默认知道应如何运行脚本——若要去教它们该怎么办，那么就需安装扩展来管理。于是我们推荐能使用在绝大多数的浏览器上的 Tampermonkey。虽然其他脚本管理器也可以，但不能保证这些脚本都能运行。

Firefox：
  1. 去这里。
  2. 点击底部标记 '+ Add to Firefox ' 以安装 Tampermonkey。
Chrome：
  1. 去这里。
  2. 点击底部标记 ' Add to Chrome' 以安装 Tampermonkey。
Safari：
  1. 去这里。
  2. 点击 'Install now' 以安装 Tampermonkey。
Opera：
  1. 去这里。
  2. 点击底部标记 'Add to Opera' 以安装 Tampermonkey。
Edge：
  1. 去这里。
  2. 点击底部标记 'Get' 以安装 Tampermonkey.

这将会为你安装所需的基本工具，以让你得以允许此处链接中的脚本。

怎样安装用户自定义脚本

任何能用 Tampermonkey 的浏览器：

  1. 选取你想要的脚本，并点击标记有"点此下载/安装"的链接。
  2. 你应该重定向到 Tampermonkey。点击 "安装/Install"。
    • 如果你不重定向，那么文件只是单单地被下载，然后不会被 Tampermonkey 正确地安装。
    • 如果脚本因任何情况而安装失败，那么就该让技术团队成员去知道。
  3. 刷新页面，然后你的脚本就应该会起作用了。
  4. 接着去浏览！
  5. 如果在任何情况下，你被重定向到一个显示着“一个用户脚本试图访问跨域资源”的页面时，就去点击左下方的“始终允许”。你有20秒的时间来做这件事——如果你用光了时间，别担心，浏览器会认为你选择“no”，然后在有需要的时候再询问你一次。

跳转框The Jumpbox

链接：点此下载/安装

截屏：

描述：这会把一个小的文本框和按钮，插入到屏幕右上角的帐户信息边。Jumpbox 允许你立即跳转到任何你想看的文章，只输入数字就能把你弹送到相应的项目里去。输入英文字母和数字的组合，或者只输入英文，就可以获取那个页面。所以，输入“chat guide”会跳转到《聊天指南》。注意，Wikidot 会负责处理文本间的空格，并将其转写为破折号2。

ENTER键和确认按钮都可以提交请求。

注意：该脚本不检查页面是否存在，也不提供拼写纠正功能。

[+]  详细的使用说明
[-] 详细的使用说明

在标题前加上[修饰符]|，以获取特定类型的文章。当你想看 A Fat Ghost 写的故事“827”而不是 SCP-827 时，输入 t|827 将会给你你确切想看的文章。若仅仅是数字，或者 SCP-827 则会给你SCP项目。

  • j：为搞笑条目（包括特殊调整到 SCP-J ）
  • x：为 -EX 已解明文章。
  • a：为 -ARC 被归档的文章。
  • d：为 -D 被废除的文章。
  • h：用于中心页 hub （换言之，输入h|unfounded就能给你未建立设定中心Unfounded Canon hub的页面）。
  • t：为故事。

若你使用其他的修饰符，则会被脚本忽略，并像你只是在竖杠后请求指定的页面一样被执行。例如，输入o|chat guide将忽略未知的修饰符，而给你《聊天指南》页。
